С помощью выпадающего списка пользователь может выбрать один из нескольких вариантов.

Используйте `Select`, если вариантов много — пять или больше. Если вариантов меньше,
подойдёт [RadioGroup](##LIBS.LIB.STAND/lib:uikit/stand:components-radiogroup-stable)
или [ChoiceGroup](##LIBS.LIB.STAND/lib:uikit/stand:components-choicegroup-stable).
