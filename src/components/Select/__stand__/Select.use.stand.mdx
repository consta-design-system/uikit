Используйте `Select`, чтобы дать пользователю возможность выбрать элемент из заранее определенного списка. Компонент часто применяется в формах, настройках, фильтрах, где значения предопределены.

<MdxImage src={start_example} />

# Контекст использования

## Формы ввода данных

Компонент `Select` может использоваться в формах ввода данных для выбора страны, языка, часового пояса, профессии и других параметров.

**Пример**. Выбор организации из преднастроенного списка в регистрационной форме.

<MdxImage src={form} />

## Фильтры и сортировка

Компонент подойдет для выбора критериев фильтрации или сортировки при поиске товаров, услуг или информации.

**Пример**. Фильтрация посещений по переходам из различных ресурсов.

<MdxImage src={filter} />

## Настройки и параметры

Компонент `Select` может использоваться для выбора различных параметров в настройках приложения.

**Пример**. Выбор настроек сервера из списка преднастроенных значений.

<MdxImage src={server} />

# Лучшие практики

## Подсказка

Вы можете добавить [подсказку](##LIBS.LIB.STAND.TAB/lib:uikit/stand:components-select-stable/tab:dev/hash:подсказка) для выбора элемента. Так, например, можно использовать слово "Выберите".

## Значение по умолчанию

Установите значение по умолчанию, если можно предположить, что один из элементов списка будет использоваться чаще других. Это поможет пользователю быстрее завершить заполнение формы.

## Группировка

Если в списке много элементов, разделите их на логические [группы](##LIBS.LIB.STAND.TAB/lib:uikit/stand:components-select-stable/tab:dev/hash:группы-вариантов). Это упростит навигацию и поиск нужного варианта.
