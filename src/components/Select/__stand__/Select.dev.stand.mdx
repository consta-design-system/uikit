## Обзор

Компонент `Select` — это выпадающий список, который позволяет пользователю выбрать один вариант из предложенных. Поддерживает настройку внешнего вида, содержимого, поведения и кастомизацию элементов.

## Импорт

```tsx
import { Select } from '@consta/uikit/Select';
```

```tsx
type SelectItemDefault = {
  label: string;
  id: string | number;
  groupId?: string | number;
  disabled?: boolean;
};

type SelectGroupDefault = {
  label: string;
  id: string | number;
};

type SelectPropOnChange<ITEM = SelectItemDefault> = (
  value: ITEM | null,
  props: {
    e: React.SyntheticEvent;
  },
) => void;

type SelectRenderItemProps<ITEM> = {
  item: ITEM;
  active: boolean;
  hovered: boolean;
  onClick: (e: React.SyntheticEvent) => void;
  onMouseEnter: (e: React.SyntheticEvent) => void;
  ref: React.Ref<HTMLDivElement>;
};

type SelectPropRenderValue<ITEM> = (props: {
  item: ITEM;
}) => React.ReactNode | null;

type SelectPropGetItemLabel<ITEM> = (item: ITEM) => string;
type SelectPropGetItemKey<ITEM> = (item: ITEM) => string | number;
type SelectPropGetItemGroupKey<ITEM> = (
  item: ITEM,
) => string | number | undefined;
type SelectPropGetItemDisabled<ITEM> = (item: ITEM) => boolean | undefined;
type SelectPropGetGroupKey<GROUP> = (group: GROUP) => string | number;
type SelectPropGetGroupLabel<GROUP> = (group: GROUP) => string;
```

## Содержимое

### Варианты

Свойство `items` определяет массив элементов выпадающего списка. По умолчанию тип `items` соответствует интерфейсу `SelectItemDefault`:

- `label` — название элемента (пишите с заглавной буквы, без точки в конце).
- `id` — уникальный ключ элемента.
- `groupId` — идентификатор группы, если элементы разбиты на группы.
- `disabled` — флаг неактивного элемента.

<MdxTabs>

<SelectExampleItems />

```tsx
const items = [
  { label: 'Первый', id: 1 },
  { label: 'Второй', id: 2 },
  { label: 'Третий', id: 3 },
];
```

</MdxTabs>

### Выбранное значение

Свойство `value` задаёт выбранное значение из массива `items`. Для обработки выбора используется `onChange`, который возвращает выбранный элемент и событие. Выбор срабатывает по клику или нажатию `Enter`.

<MdxTabs>

<SelectExampleItems />

```tsx
type Item = {
  label: string;
  id: number;
};

const items: Item[] = [
  { label: 'Первый', id: 1 },
  { label: 'Второй', id: 2 },
  { label: 'Третий', id: 3 },
];

function SelectExampleItems() {
  const [value, setValue] = useState<Item | null>();
  return <Select items={items} value={value} onChange={setValue} />;
}
```

</MdxTabs>

### Управление состоянием выпадающего списка

Для управления состоянием списка используйте:

- `dropdownOpen` — флаг открытия/закрытия.
- `onDropdownOpen` — обработчик открытия/закрытия.
- `ignoreOutsideClicksRefs` — список элементов, клики по которым игнорируются.

<MdxTabs>

<SelectExampleDropdownOpen />

```tsx
type Item = {
  label: string;
  id: number | string;
};

const items: Item[] = [
  { label: 'Первый', id: 1 },
  { label: 'Второй', id: 2 },
  { label: 'Третий', id: 3 },
];

const Icon = withAnimateSwitcherHOC({
  startIcon: IconArrowDown,
  startDirection: 0,
  endDirection: 180,
});

const SelectExampleDropdownOpen = () => {
  const [value, setValue] = useState<Item | null>();
  const [open, setOpen] = useFlag();
  const buttonRef = useRef<HTMLButtonElement>(null);
  const inputRef = useRef<HTMLInputElement>(null);
  const onDropdownOpen = useCallback((open: boolean) => {
    setOpen.set(open);
    if (open) {
      inputRef.current?.focus();
    }
  }, []);

  return (
    <AnimateIconSwitcherProvider active={open}>
      <FieldGroup>
        <Button
          ref={buttonRef}
          label={open ? 'Закрыть' : 'Открыть'}
          onClick={setOpen.toggle}
          iconLeft={Icon}
          onlyIcon
        />
        <Select
          inputRef={inputRef}
          placeholder="Выберите вариант"
          items={items}
          value={value}
          onChange={setValue}
          dropdownOpen={open}
          onDropdownOpen={onDropdownOpen}
          ignoreClicksInsideRefs={[buttonRef]}
        />
      </FieldGroup>
    </AnimateIconSwitcherProvider>
  );
};
```

</MdxTabs>

### Индикация загрузки

Свойство `isLoading` включает индикатор загрузки. Используется для асинхронной загрузки данных.

<MdxTabs>

<SelectExampleIsLoading />

```tsx
type Item = {
  label: string;
  id: number;
};

const useMockLoadData = (
  searchValue = '',
  timeout = 2000,
): [Item[], boolean, () => void] => {
  const [isLoading, setIsLoading] = useFlag();
  const [isStart, setIsStart] = useFlag();
  const [length, setLength] = useState<number>(0);
  const isLoadingOffDebounce = useDebounce(setIsLoading.off, 2000);

  const items = useMemo(() => {
    if (searchValue) {
      return new Array(10000)
        .fill(null)
        .map((_, i) => ({
          label: `Опция ${i + 1}`,
          id: i,
        }))
        .filter(
          (item) =>
            item.label
              .toLocaleLowerCase()
              .indexOf(searchValue.toLocaleLowerCase()) !== -1,
        );
    }
    return new Array(length).fill(null).map((_, i) => ({
      label: `Опция ${i + 1}`,
      id: i,
    }));
  }, [length, searchValue]);

  useEffect(() => {
    setIsLoading.on();
    isLoadingOffDebounce();
  }, [searchValue]);

  useEffect(() => {
    if (isStart) {
      setIsLoading.on();
    }

    const timeoutNumber = setTimeout(() => {
      if (isStart) {
        setLength((state) => state + 100);
        setIsLoading.off();
        setIsStart.off();
      }
    }, timeout);

    return () => {
      clearTimeout(timeoutNumber);
    };
  }, [isStart]);

  useEffect(() => {
    return () => {
      setIsLoading.off();
      setIsStart.off();
    };
  }, []);

  return [searchValue && isLoading ? [] : items, isLoading, setIsStart.on];
};

const SelectExampleIsLoading = () => {
  const [value, setValue] = useState<Item | null>();
  const [data, isLoading, load] = useMockLoadData();
  const onDropdownOpen = useCallback((open: boolean) => {
    open && load();
  }, []);

  return (
    <Select
      placeholder="Кликните по полю ввода, чтобы данные загрузились"
      items={data}
      value={value}
      onChange={setValue}
      onDropdownOpen={onDropdownOpen}
      isLoading={isLoading}
      virtualScroll
      onScrollToBottom={load}
    />
  );
};
```

</MdxTabs>

  label: string;
  id: number;

const items: Item[] = [

  { label: 'Второй', id: 2 },
  { label: 'Третий', id: 3 },
];

function SelectExampleRenderItem() {

    <Select
