# BasicSelect

BasicSelect — компонент, реализующий элемент управления меню опций (нативного `<select>`).

Компоненту необходимо передать тип опции: `<BasicSelect<T> {..props}/>`.

## Cвойства компонента

| Свойство       | Тип или варианты значения                              | По умолчанию | Описание                                                                                      |
| -------------- | ------------------------------------------------------ | ------------ | --------------------------------------------------------------------------------------------- |
| disabled?      | `boolean`                                              | -            | Активность компонента                                                                         |
| size?          | `s , m , l`                                            | `m`          | Размер компонента                                                                             |
| view?          | `default , clear`                                      | `default`    | Внешний вид компонента                                                                        |
| ariaLabel?     | `string`                                               | -            | `aria` атрибут для метки                                                                      |
| placeholder?   | `string`                                               | -            | Placeholder — текст, который отобразится в качестве подсказки в пустом поле                   |
| className?     | `string`                                               | -            | Дополнительный класс                                                                          |
| value?         | `T , null`                                             | -            | Значение селекта, по умолчанию `null`.                                                        |
| id             | `string`                                               | -            | Идентификатор селекта                                                                         |
| options        | `T[]`                                                  | -            | Опции                                                                                         |
| onChange?      | `(value: string | null) => void`                       | -            | Обработчик события изменения значения компонента                                              |
| getOptionLabel | `(arg: T) => string`                                   | -            | Функция для получения текста опции, который будет отображаться в поле для выбранного значения |
| onBlur?        | `(event?: React.FocusEvent<HTMLInputElement>) => void` | -            | Обработчик события `blur`                                                                     |
| onFocus?       | `(event?: React.FocusEvent<HTMLInputElement>) => void` | -            | Обработчик события `focus`                                                                    |

По умолчанию компонент всегда занимает всё доступное пространство в блоке.

## Пример использования

```ts
import React from 'react';
import { BasicSelect } from '@consta/uikit/BasicSelect';

type Option = {
  label: string;
  value: string;
};

const items = [
  { label: 'Москва', value: 'moscow' },
  { label: 'Санкт-Петербург', value: 'spb' },
  { label: 'Томск', value: 'tomsk' },
];

const Example = () => {
  const getItemLabel = (option: SelectOption): string => option.label;

  return (
    <>
      <div>
        <BasicSelect id="city" options={items} getOptionLabel={getItemLabel} />
      </div>
    </>
  );
};
```

## С заданным значением

```ts
import React from 'react';
import { BasicSelect } from '@consta/uikit/BasicSelect';

type Option = {
  label: string;
  value: string;
};

const items = [
  { label: 'Москва', value: 'moscow' },
  { label: 'Санкт-Петербург', value: 'spb' },
  { label: 'Томск', value: 'tomsk' },
];

const Example = () => {
  const getItemLabel = (option: SelectOption): string => option.label;
  const getItemKey = (option: SelectOption): string => option.value;
  const getItemValue = (option: SelectOption): string => option.value;

  return (
    <>
      <div>
        <BasicSelect<Option>
          id="city"
          value={{ label: 'Москва', value: 'moscow' }}
          options={items}
          getOptionLabel={getItemLabel}
        />
      </div>
    </>
  );
};
```
