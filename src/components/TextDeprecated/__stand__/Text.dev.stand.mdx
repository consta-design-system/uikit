## Текст

### Цвет

`view` отвечает за тип и цвет текста.

```ts
<Text view="primary">Просто текст</Text> /* основной текст */
<Text view="secondary">Просто текст</Text> /* дополнительный */
<Text view="brand">Просто текст</Text> /* цвет бренда */
<Text view="ghost">Просто текст</Text> /* примечания */
<Text view="link">Просто текст</Text> /* ссылка */
<Text view="link-minor">Просто текст</Text> /* дополнительная ссылка */
<Text view="system">Просто текст</Text> /* системный текст */
<Text view="normal">Просто текст</Text> /* нейтральный текст */
<Text view="success">Просто текст</Text> /* успешное действие */
<Text view="warning">Просто текст</Text> /* предупреждение */
<Text view="alert">Просто текст</Text> /* ошибка */
<Text view="caution">Просто текст</Text> /* осторожность */
<Text view="critical">Просто текст</Text> /* критическая ошибка */
```

### Размер

`size` отвечает за размер текста.

- По умолчанию в параграфах используется размер `m`.
- Даже для самых неважных подсказок используйте размер `xs` или больше.
- `2xs` — только для текста в [верхнем регистре](#регистр).

```ts
<Text size="2xs">Просто текст</Text>
<Text size="xs">Просто текст</Text>
<Text size="s">Просто текст</Text>
<Text size="m">Просто текст</Text>
<Text size="l">Просто текст</Text>
<Text size="xl">Просто текст</Text>
<Text size="2xl">Просто текст</Text>
<Text size="3xl">Просто текст</Text>
<Text size="4xl">Просто текст</Text>
<Text size="5xl">Просто текст</Text>
<Text size="6xl">Просто текст</Text>
```

<TextExampleSize />

### Гарнитура

`font` отвечает за гарнитуру шрифта.

Вы можете выбрать гарнитуру, заданную в системе:

- `primary` меняет шрифт на основной шрифт дизайн-системы,
- `mono` — на шрифт, заданный как моноширный.

Это нужно, чтобы расставить дополнительные акценты: например, выделить что-то важное
прямо в тексте.

```ts
<Text font="primary">Просто текст</Text>
<Text font="mono">Просто текст</Text>
```

<TextExampleFont />

### Насыщенность

`weight` отвечает за насыщенность (жирный, полужирный, обычный, светлый и другие варианты).

> Убедитесь, что в шрифте есть насыщенность, которую вы используете. Браузер имитирует
> насыщенность, которой нет в наборе, но неизвестно, как это будет выглядеть.

```ts
<Text weight="black">Просто текст</Text>
<Text weight="bold">Просто текст</Text>
<Text weight="semibold">Просто текст</Text>
<Text weight="regular">Просто текст</Text>
<Text weight="light">Просто текст</Text>
<Text weight="thin">Просто текст</Text>
```

<TextExampleWeight />

### Выравнивание

`align` отвечает за выравнивание текста в контейнере.

```ts
<Text align="left">Просто текст</Text>
<Text align="center">Просто текст</Text>
<Text align="right">Просто текст</Text>
```

<TextExampleAlign />

### Курсор

`cursor` отвечает за внешний вид курсора при наведении на текст.
По умолчанию это текстовый курсор, если нужно поменять его на указатель,
добавьте `cursor="pointer"`.

```ts
<Text>Это текст с обычным текстовым курсором</Text>
<Text cursor="pointer">Это текст с курсором-указателем</Text>
```

<TextExampleCursor />

### Слишком длинный текст

Текст в одну строчку, который может не поместиться в родительский контейнер, можно обрезать с помощью `truncate`. В конце добавляется многоточие.

```ts
<Text truncate>
  Очень длинный текст, который должен располагаться на одной строке и не влезает
  в родительский контейнер.
</Text>
```

## Контейнер с текстом

### Тег для контейнера

По умолчанию `Text` представлен в HTML-документе в виде тега `div`. Вы можете его
переопределить с помощью свойства `as`.

### Расположение

`display` определяет, как будет вести себя контейнер с текстом по отношению к другим элементам
на странице (как строчный или блочный элемент). Варианты: `block`, `inline-block`, `inline`.

## Блоки с текстом

### Абзац и заголовок

Пример классической конструкции: абзац с заголовком.

```ts
<Text
  as="div"
  align="left"
  font="sans"
  lineHeight="l"
  size="2xl"
  spacing="m"
  transform="uppercase"
  view="primary"
  weight="bold"
>
  Заголовок несет суть
</Text>
<Text as="p" align="left" font="serif" lineHeight="s" size="s" view="primary">
  Абзац — это часть текста между отступами с новой строки. Абзац можно назвать микротекстом.
  В общей теме могут выделяться и микротемы. При этом каждый абзац раскрывает свою
  микротему. Что же ещё можно сказать про абзацы? Мы видим, что обычно в абзацах несколько
  предложений. В таком случае каждый абзац служит для выделения своей микротемы.
</Text>

```

### Форматирование текста в абзаце

Вы можете выделять отдельные слова, фразы или части фраз с помощью форматирования.
Например, чтобы обратить внимание на важную информацию, выделить термин, элемент
интерфейса, прямую речь.

```ts
<Text as="div" align="left" font="sans" lineHeight="l" size="m" view="primary">
  <Text as="span" fontStyle="italic">
    Примечание.&nbsp;
  </Text>
  База скважин сформирована на основе отчетов сотрудников. Любые&nbsp;
  <Text weight="black" as="span">
    срочные изменения
  </Text>{' '}
  должны вноситься по регламенту в сроки, указанные Центром Разработок.
</Text>
```

## Пример использования

Использование компонента:

```ts
// src/App.ts
import React from 'react';
import { Text } from '@consta/uikit/Text';

export function App() {
  return (
    <Text
      as = 'p'
      align = 'center'
      cursor = 'default'
      decoration = 'none'
      display = 'block'
      font = 'mono'
      lineHeight = '2xs'
      size = 'm'
      spacing = 'xs'
      view = 'primary'
      weight = 'regular'
      width = 'default'
      className = 'custom-class-name'
    >
      Текст
    </>
  );
}
```
