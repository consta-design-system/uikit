## Обзор

Компонент `Pagination` предназначен для навигации по страницам в интерфейсе. Он позволяет пользователю переключаться между страницами, отображать определённое количество страниц, задавать их внешний вид и поведение, а также поддерживает адаптивность для разных устройств.

## Импорт

```tsx
import { Pagination } from '@consta/uikit/Pagination';
```

## Свойства

| Свойство                                         | Тип                                              | По умолчанию       | Описание                                                    |
| ------------------------------------------------ | ------------------------------------------------ | ------------------ | ----------------------------------------------------------- |
| [`items?`](#страницы)                            | `number`                                         | —                  | Количество страниц                                          |
| [`value?`](#страницы)                            | `number`                                         | —                  | Номер текущей страницы                                      |
| [`visibleCount?`](#количество-элементов)         | `number`                                         | —                  | Количество отображаемых элементов                           |
| [`showFirstPage?`](#первая-и-последняя-страница) | `boolean`                                        | —                  | Флаг, отображать ли первый элемент                          |
| [`showLastPage?`](#первая-и-последняя-страница)  | `boolean`                                        | —                  | Флаг, отображать ли последний элемент                       |
| [`form?`](#форма)                                | `'default', 'round', 'brick'`                    | `'default'`        | Форма элементов компонента                                  |
| [`size?`](#размер)                               | `'xs', 's', 'm', 'l'`                            | `'m'`              | Размер элементов компонента                                 |
| [`type?`](#тип)                                  | `'default', 'input'`                             | `'default'`        | Тип компонента (кнопки или поле ввода)                      |
| [`arrows?`](#стрелки)                            | `[PaginationPropArrow?, PaginationPropArrow?]`   | —                  | Стрелки перемещения по страницам                            |
| [`outerMostArrows?`](#конец-или-начало)          | `[PaginationPropArrow?, PaginationPropArrow?]`   | —                  | Стрелки перемещения на первую и последнюю страницу          |
| [`hotKeys?`](#горячие-клавиши)                   | `[PaginationPropHotKey?, PaginationPropHotKey?]` | —                  | Горячие клавиши для переключения страниц                    |
| [`onChange?`](#страницы)                         | `PaginationPropOnChange`                         | —                  | Функция для смены страницы                                  |
| `getTotalLabel?`                                 | `(totalPages: number) => string`                 | `из ${totalPages}` | Функция для получения номера страницы из количества страниц |

| `getItemAs?`                                     | `(item: PaginationItem) => AsTags`               | `button`           | Указание HTML-тега элемента                                 |
| `getItemAttributes?`                             | `(item: PaginationItem) => AsAttributes`         | —                  | Указание HTML-атрибутов элемента                            |
| `className?`                                     | `string`                                         | —                  | Дополнительные CSS-классы                                   |

## Содержимое

### Страницы

Свойства `items` и `value` определяют количество страниц и текущую страницу соответственно. Событие изменения страницы задаётся через `onChange`.

<MdxTabs>

<PaginationExample />

```tsx
const [page, setPage] = useState(1);
return <Pagination value={page} onChange={setPage} items={3} />;
```

</MdxTabs>

### Количество элементов

Свойство `visibleCount` позволяет ограничить количество отображаемых страниц, чтобы не загромождать интерфейс.

<MdxTabs>

<PaginationExampleVisibleCount />

```tsx
const [page, setPage] = useState(1);
return (
  <>
    <Pagination items={15} value={page} onChange={setPage} visibileCount={5} />
    <Pagination items={15} value={page} onChange={setPage} visibileCount={9} />
  </>
);
```

</MdxTabs>

### Первая и последняя страница

Свойства `showFirstPage` и `showLastPage` включают отображение кнопок для перехода к первой и последней страницам.

<MdxTabs>

<PaginationExampleFirstLastItem />

```tsx
const [page, setPage] = useState(7);
return (
  <Pagination
    items={15}
    value={page}
    onChange={setPage}
    showFirstPage
    showLastPage
    visibleCount={7}
  />
);
```

</MdxTabs>

## Внешний вид

### Форма

Свойство `form` задаёт форму элементов компонента. Возможные значения: `'default'`, `'round'`, `'brick'`. По умолчанию — `'default'`.

<PaginationExampleForm />

### Размер

Свойство `size` определяет размер элементов. Возможные значения: `'xs'`, `'s'`, `'m'`, `'l'`. По умолчанию — `'m'`.

<PaginationExampleSize />

## Навигация

### Тип

Свойство `type` определяет, как отображается компонент: как набор кнопок с номерами страниц (`'default'`) или как поле ввода номера страницы (`'input'`). По умолчанию — `'default'`.

<PaginationExampleType />

### Стрелки

Свойство `arrows` задаёт кнопки для перехода по страницам (назад/вперёд). Принимает массив двух значений: `true` (иконка по умолчанию), `false` (скрыть), объект `{ label?: string, disabled?: boolean, icon?: IconComponent }` или компонент иконки.

<MdxTabs>

<PaginationExampleArrows />

```tsx
const [page, setPage] = useState(1);
return (
  <>
    <Pagination
      items={5}
      value={page}
      onChange={setPage}
      arrows={[true, true]}
    />
    <Pagination
      items={5}
      value={page}
      onChange={setPage}
      arrows={[{ label: 'Сюда' }, { label: 'Туда' }]}
    />
  </>
);
```

</MdxTabs>

### Конец или начало

Свойство `outerMostArrows` управляет отображением кнопок для перехода к первой и последней страницам. Принимает массив двух значений: `true` (иконка по умолчанию), `false` (скрыть) или компонент иконки.

<MdxTabs>

<PaginationExampleOuterMostArrows />

```tsx
const [page, setPage] = useState(1);
return (
  <Pagination
    items={5}
    value={page}
    onChange={setPage}
    outerMostArrows={[true, true]}
  />
);
```

</MdxTabs>

### Горячие клавиши

Свойство `hotKeys` задаёт комбинации клавиш для переключения страниц. Принимает массив двух объектов с полями `label` (подсказка) и `keys` (клавиши, см. [список](https://developer.mozilla.org/en-US/docs/Web/API/KeyboardEvent/key/Key_Values)).

<MdxTabs>

<PaginationExampleHotKeys />

```tsx
const [page, setPage] = useState(1);
return (
  <Pagination
    items={5}
    value={page}
    onChange={setPage}
    arrows={[{ label: 'Назад' }, { label: 'Вперёд' }]}
    hotKeys={[
      { label: '← Shift', keys: ['Shift', 'ArrowLeft'] },
      { label: 'Shift →', keys: ['Shift', 'ArrowRight'] },
    ]}
  />
);
```

</MdxTabs>

## Адаптивность

Для адаптации к разным размерам экрана можно:

- Уменьшать количество отображаемых кнопок через `visibleCount`.
- Использовать `type="input"` для компактного ввода номера страницы.
- Заменить компонент `Pagination` кнопкой «Показать ещё» для подгрузки страниц.

<MdxTabs>

<PaginationExampleAdaptivity />

```tsx
const [page, setPage] = useState(1);
return (
  <>
    <Pagination
      items={15}
      value={page}
      showFirstPage
      showLastPage
      onChange={setPage}
      arrows={[true, true]}
      style={{ width: 1000 }}
    />
    <Pagination
      items={15}
      value={page}
      showFirstPage
      showLastPage
      onChange={setPage}
      arrows={[true, true]}
      style={{ width: 600 }}
    />
    <Pagination
      items={15}
      value={page}
      showFirstPage
      showLastPage
      onChange={setPage}
      arrows={[true, true]}
      style={{ width: 300 }}
    />
    <Button label="Показать еще" view="ghost" width="full" />
  </>
);
```

</MdxTabs>
