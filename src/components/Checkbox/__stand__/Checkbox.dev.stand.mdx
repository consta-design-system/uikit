## Обзор

Чекбокс — это элемент интерфейса, который позволяет пользователю выбирать один или несколько вариантов из предложенных. Используется для отметки состояния (выбрано/не выбрано) и может быть настроен по размеру, внешнему виду и поведению.

## Импорт

```tsx
import { Checkbox } from '@consta/uikit/Checkbox';
```

## Содержимое

### Название

Свойство `label` задает текст, отображаемый рядом с чекбоксом. Надпись должна начинаться с заглавной буквы и не заканчиваться точкой.

Свойство `name` определяет имя элемента `<input>` в DOM, используется для идентификации в формах.

## Внешний вид

### Размер

Свойство `size` определяет размер чекбокса. Доступные значения: `'xs'`, `'s'`, `'m'`, `'l'`. По умолчанию используется `'m'`.

<MdxTabs>

<CheckboxExampleSize />

```tsx
<>
  <Checkbox size="xs" label="Размер xs" />
  <Checkbox size="s" label="Размер s" />
  <Checkbox size="m" label="Размер m" />
  <Checkbox size="l" label="Размер l" />
</>
```

</MdxTabs>

### Стиль

Свойство `view` задает внешний вид чекбокса. Варианты:

- `'primary'` — акцентный стиль (по умолчанию).
- `'ghost'` — второстепенный стиль, менее выделяющийся.

<MdxTabs>

<CheckboxExampleView />

```tsx
<>
  <Checkbox view="primary" checked label="Акцентный чекбокс" />
  <Checkbox view="ghost" checked label="Второстепенный чекбокс" />
</>
```

</MdxTabs>

### Выравнивание

Свойство `align` управляет вертикальным выравниванием текста относительно чекбокса. Варианты:

- `'center'` — текст выровнен по центру (по умолчанию).
- `'top'` — текст выровнен по верхнему краю.

  <Checkbox
    align="top"
    label="Длинный текст для чекбокса с выравниванием по верху, здесь несколько строк"

## Поведение

### Состояния

Свойства `checked`, `intermediate`, и `disabled` управляют состоянием чекбокса:

- `checked` — если `true`, чекбокс отмечен галочкой.
- `intermediate` — если `true`, чекбокс находится в неопределенном состоянии (например, когда часть вложенных чекбоксов выбрана).
- `disabled` — если `true`, чекбокс заблокирован и не реагирует на взаимодействие.
- `onChange` — обработчик события при смене состояния чекбокса.
- `onFocus` и `onBlur` — обработчики событий фокуса и потери фокуса.
- `tabIndex` — задает порядок перехода по элементам интерфейса с помощью клавиши **Tab**.
