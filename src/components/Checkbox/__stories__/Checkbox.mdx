# Checkbox

Чекбокс — элемент с двумя состояниями: **выбран** и **не выбран**.

С помощью чекбокса пользователь может сделать выбор — например, отметить
элементы в списке.

## Название

Название — это надпись рядом с чекбоксом. Пишется с заглавной буквы, без точки.

### Согласуйте с заголвком

Это просто лучше читается. Сравните:

Где деньги?
- [] Тумбочка
- [] Банк
- [] Матрас

Где деньги?
- [] В тумбочке
- [] В банке
- [] Под матрасом, в маленьком потайном кармашке

Если не нравится, что пункты начинаются с предлогов, поменяйте заголовок.

Выберите место для хранения денег
- [] Тумбочка
- [] Банк
- [] Матрас

### Не используйте отрицания

Чтобы не путать пользователя. Когда выбран чекбокс, рядом с которым написано **Не хочу**,
приходится мысленно решать равенство: не не хочу = хочу.

- [x] Hе хочу больше этих ваших дурацких булок
- [] Съесть ещё этих мягких французских булок

### Не обобщайте

Если хотите дать пользователю возможность отметить все элементы в списке
или, наоборот, удалить все отметки, сделайте это другим способом.

Каких булок ещё съесть
- []Этих
- []Мягких
- []Сладких
- []Французских
- []Никаких

Каких булок ещё съесть
- []Этих
- []Мягких
- []Сладких
- []Французских

## Свойства

| Свойство                 | Тип                         | По умолчанию | Описание                                     |
| ------------------------ | --------------------------- | ------------ | -------------------------------------------- |
| `label`                  | `string`                    | -            | Надпись рядом с чекбоксом                                 |
| `size`                   | `'m', 'l'`                  | `'m'`        | Размер                                       |
| `checked`                | `boolean`                    | -            | Отмечает, выбран чекбокс или нет (ставит галочку)                            |
| `disabled`               | `boolean`                    | -            | Блокирует чекбокс                            |
| `intermediate`           | `boolean`                    | -            | Делает чекбокс неопределенным                             |
| `name`                   | `string`                    | -            | Имя компонента в DOM |
| `onFocus`                  | `React.FocusEventHandler<HTMLInputElement>`| -            | Расширение файла  |
| `onBlur`                   | `React.FocusEventHandler<HTMLInputElement>` | -            | Индикация загрузки |
| `autoFocus` | `boolean`                   | -            | Показать/скрыть индикатор прогресса загрузки |
| `readOnly`         | `boolean`                    | -            | Указывает прогресс индикации загрузки   |      
| `required`       | `boolean`                    | -            | Указывает прогресс индикации загрузки        |
| `step`         | `number | string`                    | -            | Указывает прогресс индикации загрузки        |
| `tabIndex`         | `number `                    | -            | Порядковый номер при переходе между элементами интерфейса по клавише **tab**        |
| `children`                | `string`                    | -            | Дополнительный CSS-класс                     |
| `className`                | `string`                    | -            | Дополнительный CSS-класс                     |
| `inputRef`                | `React.Ref<HTMLInputElement>` | -            | Ссылка на корневой DOM-элемент этого компонента |




## Пример

```ts
// src/App.ts
import React from 'react';
import { Checkbox } from '@gpn-design/uikit/Checkbox';

const App = () => {
  return <Checkbox label="Поставьте галочку" />;
};
```
