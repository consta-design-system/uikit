# Card

Карточка — небольшой компонент (в общем смысле) с большими возможностями. Все из-за многообразия вариаций
наполняющего контента: шапка+таблица/график, просто текст, промо карточка с иллюстрациями и т.д. Мы не будем
стандартизировать наполнение — проекты разные, всем не угодишь. Вместо этого мы сделали карточку в виде
“коробки”, которая подчиняется правилам тематизации нашей дизайн-системы, а контент дизайнер помещает сам
(главное, чтобы контент тоже подчинялся правилам тематизации).

## Пример использования

<CardExample />

```tsx
<Card verticalSpace={'2xl'} horizontalSpace={'2xl'} state={'alert'}>
  <Text>Something message</Text>
</Card>
```

## Список свойств

| Свойство          | Тип или варианты значения                               | По умолчанию | Описание                                                                     |
| ----------------- | ------------------------------------------------------- | ------------ | ---------------------------------------------------------------------------- |
| `verticalSpace`   | `'xs', 's', 'm', 'l', 'xl', '2xl', '3xl', '4xl', '5xl'` | `xs`         | Вертикальные отступы                                                         |
| `horizontalSpace` | `'xs', 's', 'm', 'l', 'xl', '2xl', '3xl', '4xl', '5xl'` | `xs`         | Горизонтальные отступы                                                       |
| `state`           | `string`                                                | -            | Состояние карточки, в зависимоти от состояния отрисовываются разные границы  |
| `form`            | `'round', 'square'`                                     | 'round'      | Форма                                                                        |
| `hasShadow`       | `boolean`                                               | 'true'       | Тень                                                                         |
| `tabIndex`        | `number`                                                | —            | Порядковый номер при переходе между элементами интерфейса по клавише **tab** |
| `as`              | `React.ElementType`                                     | `button`     | HTML-тег для кнопки                                                          |
| `className`       | `string`                                                | —            | Дополнительные CSS-классы                                                    |
