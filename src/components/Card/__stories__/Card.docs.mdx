# Card

Карточка — небольшой компонент с большими возможностями. В него можно поместить что угодно:
таблицу или график, промо-карточку с иллюстрациями или обычный текст.

Мы не стали стандартизировать наполнение — проекты разные, всем не угодишь.

Вместо этого сделали карточку в виде «коробки», которая подчиняется правилам
тематизации нашей дизайн-системы. В неё можно поместить любой контент —
главное, чтобы контент тоже подчинялся правилам тематизации.

Данный компонент использует в себе миксин `MixCard`, ознакомиться с его документацией и
примерами [можно здесь](?path=/docs/mixins-mixcard--playground)

- [Список свойств](#список-свойств)
- [Пример использования](#пример-использования)

## Пример использования

<CardExample />

```tsx
<Card verticalSpace={'2xl'} horizontalSpace={'2xl'} status={'alert'}>
  <Text>Здесь может быть ваш текст</Text>
</Card>
```

## Список свойств

| Свойство          | Тип или варианты значения                               | По умолчанию | Описание                                                                     |
| ----------------- | ------------------------------------------------------- | ------------ | ---------------------------------------------------------------------------- |
| `verticalSpace`   | `'xs', 's', 'm', 'l', 'xl', '2xl', '3xl', '4xl', '5xl'` | `xs`         | Вертикальные отступы                                                         |
| `horizontalSpace` | `'xs', 's', 'm', 'l', 'xl', '2xl', '3xl', '4xl', '5xl'` | `xs`         | Горизонтальные отступы                                                       |
| `status`          | `'alert', 'success', 'warning'`                         | -            | Состояние карточки, в зависимости от состояния отрисовываются разные границы |
| `form`            | `'round', 'square'`                                     | 'round'      | Форма                                                                        |
| `shadow`          | `boolean`                                               | 'true'       | Тень                                                                         |
| `tabIndex`        | `number`                                                | —            | Порядковый номер при переходе между элементами интерфейса по клавише **tab** |
| `as`              | `React.ElementType`                                     | `button`     | HTML-тег для кнопки                                                          |
| `className`       | `string`                                                | —            | Дополнительные CSS-классы                                                    |
