## Отступы

Отступы внутри карточки можно менять с помощью двух свойств:

- за отступы сверху и снизу отвечает `verticalSpace`,
- за отступы по бокам — `horizontalSpace`.

Варианты:
`'xs'`, `'s'`, `'m'`, `'l'`, `'xl'`, `'2xl'`, `'3xl'`, `'4xl'`, `'5xl'`.

По умолчанию — `'xs'`.

```tsx
<Card verticalSpace="xs" horizontalSpace="xs">
  <Text>Отступы 'xs'</Text>
</Card>
```

## Статус

Свойство `status` окрашивает рамку карточки в цвета соответствующего статуса. Варианты:
`'alert'`, `'success'`, `'warning'`.

```tsx
<Card verticalSpace="2xl" horizontalSpace="2xl" status="alert">
  <Text>Опасная карточка</Text>
</Card>
<Card verticalSpace="2xl" horizontalSpace="2xl" status="warning">
  <Text>Тревожная карточка</Text>
</Card>
<Card verticalSpace="2xl" horizontalSpace="2xl" status="success">
  <Text>Успешная карточка</Text>
</Card>
```

## Форма

За форму карточки отвечает свойство `form`. Варианты:
`'round'`, `'square'`, по умолчанию `'round'`.

```tsx
<Card verticalSpace="2xl" horizontalSpace="2xl" form="round">
  <Text>Круглая карточка</Text>
</Card>
<Card verticalSpace="2xl" horizontalSpace="2xl" form="square">
  <Text>Квадратная карточка</Text>
</Card>
```

## Тень

По умолчанию у карточки есть тень, её можно убирать и показывать с помощью свойства `shadow`.

```tsx
<Card verticalSpace="2xl" horizontalSpace="2xl" status="success" shadow={false} border>
  <Text>Карточка с границей</Text>
</Card>
<Card verticalSpace="2xl" horizontalSpace="2xl" status="success" shadow={false}>
  <Text>Карточка без границы</Text>
</Card>
```

## Граница

По умолчанию у карточки отключены границы, за их отображение отвечает свойство `border`.

```tsx
<Card verticalSpace="2xl" horizontalSpace="2xl" status="success">
  <Text>Карточка с тенью</Text>
</Card>
<Card verticalSpace="2xl" horizontalSpace="2xl" status="success" shadow={false}>
  <Text>Карточка без тени</Text>
</Card>
```

## Адаптивность

Поведение карточек при изменении ширины экрана зависит от правил адаптивности, которые вы установили для своего проекта, от брекпойнтов, сетки и так далее.

По умолчанию карточки растягиваются и сжимаются в зависимости от ширины экрана, а высота карточек зависит от высоты их содержимого. Но на самом деле карточка — просто контейнер, и может вести себя так, как вы захотите.

Например, сетка из карточек может перестраиваться так:

## Список свойств

| Свойство                       | Тип или варианты значения                               | По умолчанию | Описание                                                                     |
| ------------------------------ | ------------------------------------------------------- | ------------ | ---------------------------------------------------------------------------- |
| [`verticalSpace?`](#отступы)   | `'xs', 's', 'm', 'l', 'xl', '2xl', '3xl', '4xl', '5xl'` | `'xs'`       | Вертикальные отступы                                                         |
| [`horizontalSpace?`](#отступы) | `'xs', 's', 'm', 'l', 'xl', '2xl', '3xl', '4xl', '5xl'` | `'xs'`       | Горизонтальные отступы                                                       |
| [`status?`](#статус)           | `'alert', 'success', 'warning'`                         | -            | Статус, меняет цвет границ карточки                                          |
| [`form?`](#форма)              | `'round'`, `'square'`                                   | `'round'`    | Форма                                                                        |
| [`shadow?`](#тень)             | `boolean`                                               | `'true'`     | Тень                                                                         |
| [`border?`](#граница)          | `boolean`                                               | `'false'`    | Граница                                                                      |
| `tabIndex`                     | `number`                                                | -            | Порядковый номер при переходе между элементами интерфейса по клавише **tab** |
| `as`                           | `React.ElementType`                                     | `div`        | HTML-тег для кнопки                                                          |
| `className`                    | `string`                                                | -            | Дополнительные CSS-классы                                                    |
