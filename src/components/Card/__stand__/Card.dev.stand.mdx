## Обзор

Компонент `Card` — это контейнер для отображения контента с настраиваемыми отступами, формой, тенью, границей и статусом. Подходит для создания карточек с различным содержимым, например, текстом, изображениями или другими компонентами.

## Импорт

```tsx
import { Card } from '@consta/uikit/Card';
```

## Свойства

| Свойство                                | Тип или варианты значения                                               | По умолчанию | Описание                                                          |
| --------------------------------------- | ----------------------------------------------------------------------- | ------------ | ----------------------------------------------------------------- |
| [`verticalSpace?`](#отступы)            | `'xs' \| 's' \| 'm' \| 'l' \| 'xl' \| '2xl' \| '3xl' \| '4xl' \| '5xl'` | `'xs'`       | Вертикальные отступы                                              |
| [`horizontalSpace?`](#отступы)          | `'xs' \| 's' \| 'm' \| 'l' \| 'xl' \| '2xl' \| '3xl' \| '4xl' \| '5xl'` | `'xs'`       | Горизонтальные отступы                                            |
| [`status?`](#статус)                    | `'alert' \| 'success' \| 'warning'`                                     | -            | Статус, меняет цвет границ карточки                               |
| [`form?`](#форма)                       | `'round' \| 'square'`                                                   | `'round'`    | Форма карточки                                                    |
| [`shadow?`](#тень)                      | `boolean`                                                               | `true`       | Тень                                                              |
| [`border?`](#граница)                   | `boolean`                                                               | `false`      | Граница                                                           |
| [`tabIndex?`](#навигация-по-клавиатуре) | `number`                                                                | -            | Порядковый номер при переходе между элементами по клавише **Tab** |
| `className?`                            | `string`                                                                | -            | Дополнительные CSS-классы                                         |
| `as?`                                   | `React.ElementType`                                                     | `div`        | HTML-тег для компонента                                           |

## Внешний вид

### Отступы

Свойства `verticalSpace` и `horizontalSpace` задают внутренние отступы карточки. Возможные значения: `'xs'`, `'s'`, `'m'`, `'l'`, `'xl'`, `'2xl'`, `'3xl'`, `'4xl'`, `'5xl'`. По умолчанию — `'xs'`.

- `verticalSpace` отвечает за отступы сверху и снизу.
- `horizontalSpace` — за отступы по бокам.

<MdxTabs>

<CardExampleSpace />

```tsx
<Card verticalSpace="xs" horizontalSpace="xs">
  <Text>Отступы 'xs'</Text>
</Card>
```

</MdxTabs>

### Статус

Свойство `status` окрашивает рамку карточки в цвета, соответствующие статусу. Возможные значения: `'alert'`, `'success'`, `'warning'`. Без указания свойства статус не отображается.

<MdxTabs>

<CardExampleState />

```tsx
<Card verticalSpace="2xl" horizontalSpace="2xl" status="alert">
  <Text>Опасная карточка</Text>
</Card>
<Card verticalSpace="2xl" horizontalSpace="2xl" status="warning">
  <Text>Тревожная карточка</Text>
</Card>
<Card verticalSpace="2xl" horizontalSpace="2xl" status="success">
  <Text>Успешная карточка</Text>
</Card>
```

</MdxTabs>

### Форма

Свойство `form` определяет форму углов карточки. Возможные значения: `'round'` (скругленные углы, по умолчанию) и `'square'` (прямые углы).

<MdxTabs>

<CardExampleForm />

```tsx
<Card verticalSpace="2xl" horizontalSpace="2xl" form="round">
  <Text>Круглая карточка</Text>
</Card>
<Card verticalSpace="2xl" horizontalSpace="2xl" form="square">
  <Text>Квадратная карточка</Text>
</Card>
```

</MdxTabs>

### Тень

Свойство `shadow` управляет отображением тени. По умолчанию тень включена (`true`). Установите `shadow={false}`, чтобы отключить тень.

<MdxTabs>

<CardExampleShadow />

```tsx
<Card verticalSpace="2xl" horizontalSpace="2xl" status="success">
  <Text>Карточка с тенью</Text>
</Card>
<Card
  verticalSpace="2xl"
  horizontalSpace="2xl"
  status="success"
  shadow={false}
>
  <Text>Карточка без тени</Text>
</Card>
```

</MdxTabs>

### Граница

Свойство `border` включает или отключает отображение границы. По умолчанию граница отключена (`false`). Установите `border={true}`, чтобы включить границу.

<MdxTabs>

<CardExampleBorder />

```tsx
<Card
  verticalSpace="2xl"
  horizontalSpace="2xl"
  status="success"
  shadow={false}
  border
>
  <Text>Карточка с границей</Text>
</Card>
<Card
  verticalSpace="2xl"
  horizontalSpace="2xl"
  status="success"
  shadow={false}
>
  <Text>Карточка без границы</Text>
</Card>
```

</MdxTabs>

## Поведение

### Адаптивность

Компонент `Card` адаптируется к ширине экрана в зависимости от заданных в проекте правил (брейкпоинтов, сетки и т.д.). По умолчанию карточка растягивается или сжимается по ширине, а высота зависит от содержимого. Вы можете настроить поведение карточки под свои нужды, так как это универсальный контейнер.

Пример адаптивной сетки из карточек:

<CardExampleAdaptive name="923" />

### Навигация по клавиатуре

Свойство `tabIndex` задаёт порядковый номер для перехода между элементами интерфейса с помощью клавиши **Tab**. Используйте, если карточка должна быть интерактивной.
