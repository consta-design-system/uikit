## FieldControlLayout

компонент "Коробка" для размещения в ней контролов поля ввода

<MdxTabs>

```tsx
import { IconMail } from '@consta/icons/IconMail';
import React from 'react';
import {
  FieldClearButton,
  FieldControlLayout,
  getFieldIconSize,
} from '@consta/uikit/FieldComponents';

export const FieldControlLayoutExampleSimple = () => {
  const size = 'm';

  return (
    <FieldControlLayout
      size={size}
      leftSide={<IconMail size={getFieldIconSize(size)} />}
      rightSide={['@company.ru', <FieldClearButton size={size} />]}
    >
      <FieldInput />
    </FieldControlLayout>
  );
};
```

<FieldControlLayoutExampleSimple />

</MdxTabs>

### Свойства FieldControlLayout

```tsx
type FieldPropSize = 'm' | 'xs' | 's' | 'l';
type FieldPropForm =
  | 'default'
  | 'defaultClear'
  | 'defaultBrick'
  | 'brick'
  | 'brickDefault'
  | 'brickClear'
  | 'brickRound'
  | 'round'
  | 'roundClear'
  | 'roundBrick'
  | 'clearRound'
  | 'clearDefault'
  | 'clearBrick'
  | 'clear';

type FieldPropStatus = 'alert' | 'success' | 'warning';
type FieldPropView = 'default' | 'clear';
```

| Свойство          | Тип                                                   | По умолчанию | Описание                                      |
| ----------------- | ----------------------------------------------------- | ------------ | --------------------------------------------- |
| `view?`           | `FieldPropView`                                       | `default`    | Внешний вид компонента                        |
| `disabled?`       | `boolean`                                             | `false`      | Визуальное состояние не доступного компонента |
| `focused?`        | `boolean`                                             | `false`      | Визуальное состояние компонента в фокусе      |
| `status?`         | `FieldPropStatus`                                     | -            | Тип                                           |
| `size?`           | `FieldPropSize`                                       | `m`          | Размер                                        |
| `form?`           | `FieldPropForm`                                       | `default`    | Форма                                         |
| `leftSide?`       | `React.ReactNode` <code>&#124;</code> `JSX.Element[]` | -            | Элементы слева                                |
| `rightSide?`      | `React.ReactNode` <code>&#124;</code> `JSX.Element[]` | -            | Элементы справа                               |
| `children?`       | `React.ReactNode`                                     | -            | Элемент по центру                             |
| `leftSlotsRefs?`  | `React.Ref<HTMLDivElement>[]`                         | -            | Ссылки на слоты элементов в `leftSide`        |
| `rightSlotsRefs?` | `React.Ref<HTMLDivElement>[]`                         | -            | Ссылки на слоты элементов в `rightSide`       |

## FieldButton

  FieldButton,

  getFieldIconSize,

  return (
    <FieldControlLayout
      size={size}

};

<FieldButtonExampleSimple />

# FieldToggleVisiblePasswordButton

Кнопка переключения видимости пароля ("глазик").

<MdxTabs>

```tsx
import React from 'react';
import {
  FieldControlLayout,
  FieldToggleVisiblePasswordButton,
} from '@consta/uikit/FieldComponents';

import { useFlag } from '@consta/uikit/useFlag';

export const FieldToggleVisiblePasswordButtonExampleSimple = () => {
  const [passwordVisible, setPasswordVisible] = useFlag();

  return (
    <FieldControlLayout
      rightSide={
        <FieldToggleVisiblePasswordButton
          active={passwordVisible}
          onClick={setPasswordVisible.toggle}
        />
      }
    />
  );
};
```

<FieldToggleVisiblePasswordButtonExampleSimple />

</MdxTabs>

### Свойства FieldWrapper

```tsx
type FieldPropSize = 'm' | 'xs' | 's' | 'l';
type Counter = string | number | [string | number, string | number];
```

| Свойство         | Тип                                  | По умолчанию | Описание                                        |
| ---------------- | ------------------------------------ | ------------ | ----------------------------------------------- |
| `children?`      | `React.ReactNode`                    | -            | Контрол                                         |
| `size?`          | `FieldPropSize`                      | `'m'`        | Размер                                          |
| `label?`         | `string`                             | -            | Заголовок                                       |
| `labelIcon?`     | `IconComponent`                      | -            | Иконка у заголовка                              |
| `labelIconRef?`  | `React.Ref<HTMLSpanElement>`         | -            | Ссылка на иконку                                |
| `labelPosition?` | `'top'` <code>&#124;</code> `'left'` | `'top'`      | Позиция заголовка                               |
| `labelHtmlFor?`  | `string`                             | -            | Id поля в которому необходимо привязать `label` |
| `caption?`       | `string`                             | -            | Подсказка                                       |
| `required?`      | `boolean`                            | -            | Признак обязательности                          |
| `status?`        | `FieldPropStatus`                    | -            | Статус                                          |
| `side?`          | `React.ReactNode`                    | -            | Дополнительный контент около контрола           |
| `counter?`       | `Counter`                            | -            | Счетчик количества символов                     |
