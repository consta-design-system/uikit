## Обзор

Компонент `Text` используется для отображения текста с различными стилями и настройками. Он позволяет управлять цветом, размером, гарнитурой, насыщенностью, наклоном, межбуквенным интервалом, высотой строки, подчёркиванием, регистром, выравниванием, курсором и поведением текста в контейнере. Подходит для создания заголовков, абзацев, подписей и других текстовых элементов.

## Импорт

```tsx
import { Text } from '@consta/uikit/Text';
```

## Содержимое

### Дочерние элементы

Свойство `children` определяет содержимое компонента `Text`. Это может быть текст, другие React-компоненты или элементы.

```tsx
<Text>Просто текст</Text>
<Text><span>Текст с вложенным элементом</span></Text>
```

## Внешний вид

### Цвет

Свойство `view` задаёт тип и цвет текста в зависимости от его назначения.

```tsx
<Text view="primary">Просто текст</Text> {/* Основной текст */}
<Text view="secondary">Просто текст</Text> {/* Дополнительный текст */}
<Text view="brand">Просто текст</Text> {/* Цвет бренда */}
<Text view="ghost">Просто текст</Text> {/* Примечания */}
<Text view="link">Просто текст</Text> {/* Ссылка */}
<Text view="link-minor">Просто текст</Text> {/* Дополнительная ссылка */}
<Text view="system">Просто текст</Text> {/* Системный текст */}
<Text view="normal">Просто текст</Text> {/* Нейтральный текст */}
<Text view="success">Просто текст</Text> {/* Успешное действие */}
<Text view="warning">Просто текст</Text> {/* Предупреждение */}
<Text view="alert">Просто текст</Text> {/* Ошибка */}
<Text view="caution">Просто текст</Text> {/* Осторожность */}
<Text view="critical">Просто текст</Text> {/* Критическая ошибка */}
```

### Размер

Свойство `size` определяет размер текста. По умолчанию для параграфов используется размер `m`. Для мелких подсказок рекомендуется использовать `xs` или больше, а `2xs` — только для текста в верхнем регистре.

```tsx
<Text size="2xs">Просто текст</Text>
<Text size="xs">Просто текст</Text>
<Text size="s">Просто текст</Text>
<Text size="m">Просто текст</Text>
<Text size="l">Просто текст</Text>
<Text size="xl">Просто текст</Text>
<Text size="2xl">Просто текст</Text>
<Text size="3xl">Просто текст</Text>
<Text size="4xl">Просто текст</Text>
<Text size="5xl">Просто текст</Text>
<Text size="6xl">Просто текст</Text>
```

### Гарнитура

Свойство `font` задаёт гарнитуру шрифта. Используется для акцентов в тексте.

```tsx
<Text font="primary">Просто текст</Text> {/* Основной шрифт дизайн-системы */}
```

<TextExampleFont />

### Насыщенность

Свойство `weight` определяет насыщенность текста (жирность).

<MdxInformer status="success">

Убедитесь, что выбранная насыщенность поддерживается шрифтом. Браузер может имитировать неподдерживаемые начертания, что может привести к непредсказуемому отображению.

</MdxInformer>

```tsx

<Text weight="bold">Просто текст</Text>
```

### Выравнивание

Свойство `align` определяет выравнивание текста в контейнере.

```tsx
<Text align="left">Просто текст</Text>
```

<TextExampleAlign />

### Курсор

Свойство `cursor` задаёт тип курсора при наведении на текст. По умолчанию используется текстовый курсор, но можно установить `pointer`.

```tsx

<Text cursor="pointer">Это текст с курсором-указателем</Text>
```

<TextExampleCursor />

### Слишком длинный текст

Свойство `truncate` обрезает текст, не помещающийся в контейнер, добавляя многоточие.

```tsx
<Text truncate>
  Очень длинный текст, который должен располагаться на одной строке и не влезает
  в родительский контейнер.
</Text>
```

## Контейнер с текстом

### Тег для контейнера

Свойство `as` позволяет задать HTML-тег, в котором будет отображаться компонент `Text`. По умолчанию используется `div`.

```tsx
<Text as="span">Текст в спане</Text>
<Text as="p">Текст в параграфе</Text>
```

### Расположение

Свойство `display` определяет поведение контейнера с текстом относительно других элементов на странице.

```tsx
<Text display="block">Блочный элемент</Text>
<Text display="inline">Строчный элемент</Text>
<Text display="inlineBlock">Строчно-блочный элемент</Text>
```

### Абзац и заголовок

Пример использования компонента для создания заголовка и абзаца.

<MdxTabs>

<TextExampleHeadingAndParagraph />

```tsx
<Text
  as="div"
  align="left"
  font="sans"
  lineHeight="l"
  size="2xl"
  spacing="m"
  transform="uppercase"
  view="primary"
  weight="bold"
>
  Заголовок несет суть
</Text>
<Text as="p" align="left" font="serif" lineHeight="s" size="s" view="primary">
  Абзац — это часть текста между отступами с новой строки. Абзац можно назвать микротекстом.
  В общей теме могут выделяться и микротемы. При этом каждый абзац раскрывает свою
  микротему. Что же ещё можно сказать про абзацы? Мы видим, что обычно в абзацах несколько
  предложений. В таком случае каждый абзац служит для выделения своей микротемы.
</Text>
```
