## Обзор

Компонент `Banner` предназначен для отображения срочных и важных оповещений, которые требуют внимания пользователя. Он размещается в верхней или нижней части страницы и может включать текст, иконки и интерактивные элементы, такие как кнопки.

В отличие от `Informer` (для кратких статичных сообщений) и `Snackbar` (для временных уведомлений), `Banner` более заметный, кастомизируемый и подходит для длительного отображения критически важной информации, например, предупреждений о сбоях или системных статусов.

## Импорт

```tsx
import { Banner } from '@consta/uikit/Banner';
```

## Содержимое

Для наполнения баннера используйте свойства `leftSide` и `rightSide`. Эти свойства позволяют добавлять текст, иконки, кнопки или другие элементы. Если в `leftSide` или `rightSide` передан массив элементов, отступы между ними задаются автоматически с помощью свойства `itemsGap`. Если первый элемент массива `leftSide` — строка, она отображается жирным шрифтом, а остальные строки оборачиваются в компонент `Text`.

<MdxTabs>

<BannerExampleContent />

```tsx
<Banner
  leftSide={['Заголовок', 'описание']}
  rightSide={[<Button label="Кнопка" />]}
/>
```

</MdxTabs>

## Внешний вид

### Фон

Свойства `view` и `status` определяют стиль и цветовую схему баннера. Параметр `view` задает тип фона:

- `filled` — заполненный фон с цветом, зависящим от `status`.
- `ghost` — полупрозрачный фон.
- `transparent` — прозрачный фон, подходит для кастомизации с помощью `style`.

Свойство `status` определяет цветовую схему: `normal`, `warning`, `alert`, `success`, `system`.

<MdxTabs>

<BannerExampleView />

```tsx
<Banner leftSide="Заголовок" view="filled" status="normal" />
```

</MdxTabs>

Для создания баннера с пользовательским фоном используйте `view="transparent"` и задайте фон через свойство `style`.

<MdxTabs>

<BannerExampleViewCustomBg />

```tsx
const { themeClassNames } = useTheme();

<Banner
  leftSide="Заголовок"
  view="transparent"
  style={{
    background: `url(${BannerCustomBg}) right top no-repeat, rgb(1, 45, 155)`,
  }}
  className={themeClassNames.color.accent}
/>;
```

</MdxTabs>

### Размер

Свойство `size` задает размер баннера: `xs`, `s`, `m`, `l`. Размер влияет на высоту и отступы компонента.

<MdxTabs>

<BannerExampleSize />

```tsx
<Banner leftSide="Заголовок" size="s" />
```

</MdxTabs>

### Форма

Свойство `form` определяет форму углов баннера:

- `default` — стандартные углы.
- `round` — скругленные углы.
- `brick` — прямые углы.

<MdxTabs>

<BannerExampleForm />

```tsx
<Banner leftSide="Заголовок" form="brick" />
```

</MdxTabs>
