# ChoiceGroup

```
type Item = unknown;

type OnChange = (props: { e: React.ChangeEvent<HTMLInputElement>; value: ITEM | null }) => void;

type OnChangeMultiple = (props: { e: React.ChangeEvent<HTMLInputElement>; value: ITEM[] | null }) => void;

```

| Свойство   | Тип                                              | По умолчанию | Описание                                                                                                                                                                                                                              |
| ---------- | ------------------------------------------------ | ------------ | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| items      | `Item[]`                                         | -            | Элементы                                                                                                                                                                                                                              |
| multiple   | `boolean`                                        | -            | Множественный выбор                                                                                                                                                                                                                   |
| value      | multiple ? `Item[]` : `Item` or `null`           | -            | Значение                                                                                                                                                                                                                              |
| onChange   | multiple ? `onChangeMultiple` : `OnChange`       | -            | Событие сработает при выборе элемента                                                                                                                                                                                                 |
| name       | `string`                                         | -            | Имя поля                                                                                                                                                                                                                              |
| getLabel   | `(item: Item) => string, number`                 | -            | Функция, результат которого будет использоваться в качестве текса на элементе, или если `onlyIcon = true` будет использоваться в качестве всплывающей подсказки на элементе. Также используется в качестве уникального ключа элемента |
| getIcon?   | `(item: Item) => React.FC<IconProps>, undefined` | -            | Функция, результат которого будет использоваться в качестве иконки на элементе                                                                                                                                                        |
| onlyIcon?  | `boolean`                                        | -            | Показывать только иконку                                                                                                                                                                                                              |
| size?      | `'xs', 's', 'm', 'l'`                            | `'m'`        | Размер                                                                                                                                                                                                                                |
| form?      | `'default', 'brick', 'round'`                    | `'default'`  | Форма                                                                                                                                                                                                                                 |
| view?      | `'primary', 'ghost', 'secondary'`                | `'primary'`  | Вид                                                                                                                                                                                                                                   |
| className? | `string`                                         | -            | Дополнительный класс                                                                                                                                                                                                                  |
| ref?       | `React.Ref<HTMLDivElement>`                      | -            | Ссылка на корневой DOM элемент компонента                                                                                                                                                                                             |
