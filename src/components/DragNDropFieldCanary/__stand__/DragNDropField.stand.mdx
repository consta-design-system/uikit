import { DragNDropField } from '@consta/uikit/DragNDropFieldCanary';

Компонент похож на [FileField](##LIBS.LIB.STAND/lib:uikit/stand:components-filefield-stable), но имеет больше возможностей:

- это самостоятельный элемент, который можно использовать отдельно от других компонентов,
- в него можно перемещать файлы,
- есть возможность ограничить типы и размер файлов, а также обрабатывать ошибки.

Кроме того, `DragNDropField` предоставляет информативные сообщения об ошибках и статусе загрузки файлов с помощью вспомогательного компонента `DragNDropFieldInformer`.
