## Обзор

Компонент `Radio` используется для создания радиокнопок, которые позволяют пользователю выбрать один вариант из нескольких. Подходит для форм, где нужно сделать однозначный выбор. Компонент поддерживает настройку внешнего вида, состояний и поведения.

## Импорт

```tsx
import { Radio } from '@consta/uikit/Radio';
```

## Содержимое

### Подпись

Свойство `label` задаёт текст, отображаемый рядом с радиокнопкой. Рекомендуется писать текст с заглавной буквы и без точки в конце. Это свойство помогает пользователю понять, за что отвечает радиокнопка.

```tsx
<Radio label="Это радиокнопка" checked />
```

## Внешний вид

### Размер

Свойство `size` определяет размер радиокнопки. Доступные значения: `'xs'`, `'s'`, `'m'`, `'l'`. По умолчанию используется `'m'`. Выбор размера зависит от контекста использования и дизайна интерфейса.

<MdxTabs>

<RadioExampleSize />

```tsx
<Radio size="xs" label="Размер xs" />
<Radio size="s" label="Размер s" />
<Radio size="m" label="Размер m" />
<Radio size="l" label="Размер l" />
```

</MdxTabs>

### Стиль

Свойство `view` задаёт стиль радиокнопки. Доступные варианты:

- `'primary'` — акцентный стиль, выделяет радиокнопку (используется по умолчанию).
- `'ghost'` — второстепенный стиль, менее заметный.

<MdxTabs>

<RadioExampleView />

```tsx
<Radio view="primary" checked label="Акцентная радиокнопка" />
<Radio view="ghost" checked label="Второстепенная радиокнопка" />
```

</MdxTabs>

### Выравнивание

Свойство `align` управляет вертикальным выравниванием подписи относительно радиокнопки. Доступные варианты:

- `'center'` — подпись выровнена по центру (по умолчанию).
- `'top'` — подпись выровнена по верхнему краю, полезно для многострочного текста.

<MdxTabs>

<RadioExampleAlign />

```tsx
<Radio
  align="center"
  label="Длинный текст для радиокнопки с выравниванием по центру, тут несколько строк"
/>
<Radio
  align="top"
  label="Длинный текст для радиокнопки с выравниванием по верху, тут несколько строк"
/>
```

</MdxTabs>

## Поведение

### Состояния

Свойства `checked`, `disabled`, и `required` определяют состояние радиокнопки:

- `checked` — радиокнопка выбрана.
- `disabled` — радиокнопка неактивна и не реагирует на действия пользователя.
- `required` — указывает, что выбор радиокнопки обязателен.

### Фокус и взаимодействие

Свойства, связанные с взаимодействием:

- `autoFocus` — автоматически устанавливает фокус на радиокнопку при загрузке.
- `onChange` — функция, вызываемая при изменении состояния радиокнопки.
- `onFocus` — функция, вызываемая при получении фокуса.
- `onBlur` — функция, вызываемая при потере фокуса.
- `name` — имя для `<input>` в DOM, используется для группировки радиокнопок.
- `tabIndex` — задаёт порядок перехода по элементам интерфейса с помощью клавиши **Tab**.
- `inputId` — уникальный идентификатор для `<input>`.

Эти свойства полезны для управления поведением радиокнопки в интерактивных формах.

## Пример использования

Базовый пример использования компонента `Radio`:

```tsx
import React from 'react';
import { Radio } from '@consta/uikit/Radio';

const App = () => {
  return <Radio label="Это радиокнопка" />;
};
```
