## Название

За название радиокнопки отвечает свойство `label`. Лучше его писать с заглавной буквы, точка не нужна.

```tsx
<Radio label="Это радиокнопка" checked />
```

<RadioExampleOne />

## Состояния

Чтобы сделать радиокнопку выбранной, добавьте `checked`.

Чтобы не дать пользователю сделать выбор, добавьте `disabled`.

```tsx
<Radio label="Radio" />
<Radio label="Checked" checked />
<Radio label="Disabled" disabled />
<Radio label="Checked Disabled" checked disabled />
```

<RadioExampleStatus />

## Размер

За размер отвечает свойство `size`. Варианты: `'l'`, `'m'`, `'s'`, `'xs'`, по умолчанию `'m'`.

```tsx
<Radio size="m" label="Размер l" />
<Radio size="m" label="Размер m" />
<Radio size="m" label="Размер s" />
<Radio size="l" label="Размер xs" />
```

<RadioExampleSize />

## Внешний вид

За внешний вид радиокнопки отвечает свойство `view`. Варианты:

- `primary` — акцентная радиокнопка (по умолчанию),
- `ghost` — второстепенная радиокнопка.

```tsx
<Radio
  view="primary"
  checked
  label="Акцентная радиокнопка"
/>
<Radio
  view="ghost"
  checked
  label="Второстепенная радиокнопка"
/>
```

<RadioExampleView />

## Выравнивание

За выравнивание названия по вертикали по отношению к радиокнопке отвечает свойство `align`. Варианты:

- `center` — радиокнопка выровнена по центру контейнера с текстом,
- `top` — радиокнопка выровнена по верхнему краю контейнера с текстом.

```tsx
<Radio
  align="center"
  label="Длинный текст для радиокнопки с выравниванием по центру, тут несколько строк"
/>
<Radio
  align="top"
  label="Длинный текст для радиокнопки с выравниванием по верху, тут несколько строк"
/>
```

## Свойства

| Свойство                  | Тип                                                                              | По умолчанию | Описание                                                                     |
| ------------------------- | -------------------------------------------------------------------------------- | ------------ | ---------------------------------------------------------------------------- |
| [`label?`](#название)     | `string`                                                                         | -            | Текст рядом с радиокнопкой                                                   |
| [`size?`](#размер)        | `'l', 'm', 's', 'xs'`                                                            | `'m'`        | Размер радиокнопки                                                           |
| [`view?`](#внешний-вид)   | `'primary', 'ghost'`                                                             | `'primary'`  | Внешний вид                                                                  |
| [`align?`](#выравнивание) | `'center', 'top'`                                                                | `'center'`   | Выравнивание                                                                 |
| [`checked`](#состояния)   | `boolean, undefined`                                                             | -            | Определяет выбор радиокнопки                                                 |
| [`disabled?`](#состояния) | `boolean`                                                                        | -            | Делает радиокнопку неактивной                                                |
| `required?`               | `boolean`                                                                        | -            | Такую радиокнопку обязательно нужно выбрать                                  |
| `autoFocus?`              | `boolean`                                                                        | -            | Переносит фокус на эту кнопку                                                |
| `tabIndex?`               | `number`                                                                         | -            | Порядковый номер при переходе между элементами интерфейса по клавише **tab** |
| `name?`                   | `string`                                                                         | -            | Имя для `<input>` в DOM                                                      |
| `onChange?`               | `(object: {e: React.ChangeEvent<HTMLInputElement>; checked: boolean;}) => void;` | -            | Действие при выборе радиокнопки                                              |
| `onFocus?`                | `React.FocusEventHandler<HTMLInputElement>`                                      | -            | Действие по фокусу                                                           |
| `onBlur?`                 | `React.FocusEventHandler<HTMLInputElement>`                                      | -            | Действие при потере фокуса                                                   |
| `inputId?`                | `string`                                                                         | -            | Идентификатор для инпута                                                     |
| `inputRef?`               | `React.FocusEventHandler<HTMLInputElement>`                                      | -            | Ссылка на DOM-элемент `<input>`                                              |
| `ref?`                    | `React.FocusEventHandler<HTMLLabelElement>`                                      | -            | Ссылка на корневой DOM-элемент этого компонента                              |
| `className?`              | `string`                                                                         | -            | Дополнительные CSS-классы                                                    |
