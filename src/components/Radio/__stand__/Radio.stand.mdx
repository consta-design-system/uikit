Радиокнопки нужны, чтобы пользователь мог сделать выбор — только один из нескольких
вариантов. Этот компонет — один элемент из списка, одна радиокнопка.

> Если вам нужен список из нескольких радиокнопок, используйте [RadioGroup](?path=/docs/components-radiogroup--playground). Чтобы использовать `Radio`, придётся самостоятельно написать обработку выбора из группы.
