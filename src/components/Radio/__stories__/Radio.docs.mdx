# Radio

Радиокнопки нужны, чтобы пользователь мог сделать выбор — только один из нескольких
вариантов. Этот компонет — один элемент из списка, одна радиокнопка.

> Если вам нужен список из нескольких радиокнопок, используйте [RadioGroup](?path=/docs/components-radiogroup--playground). Чтобы использовать `Radio`, придётся самостоятельно написать обработку выбора из группы.

## Название

За название радиокнопки отвечает свойсвто `label`. Лучше его писать с заглавной буквы, точка не нужна.

<RadioExampleOne />

```tsx
<Radio label="Это радиокнопка" checked />
```

## Состояния

Чтобы сделать радиокнопку выбранной, добавьте `checked`.

Чтобы не дать пользователю сделать выбор, добавьте `disabled`.

<RadioExampleStatus />

```tsx
<Radio label="Radio" />
<Radio label="Checked" checked />
<Radio label="Disabled" disabled />
<Radio label="Checked Disabled" checked disabled />
```

## Размер

За размер отвечает свойство `size`. Варианты: `'m'`, `'l'`, по умолчанию `'m'`.

<RadioExampleSize />

```tsx
<Radio size="m" label="Размер m" />
<Radio size="l" label="Размер l" />
```

## Внешний вид

За внешний вид радиокнопки отвечает свойство `view`. Варианты:

- `primary` — акцентная радиокнопка (по умолчанию),
- `ghost` — второстепенная радиокнопка.

<RadioExampleView />

## Выравнивание

За выравнивание названия по вертикали по отношению к радиокнопке отвечает свойство `align`. Варианты:

- `center` — радиокнопка выровнена по центру контейнера с текстом,
- `top` — радиокнопка выровнена по верхнему краю контейнера с текстом.

<RadioExampleAlign />

```tsx
<Radio
  align="center"
  label="Длинный текст для радиокнопки с выравниванием по центру, тут несколько строк"
/>
<Radio
  align="top"
  label="Длинный текст для радиокнопки с выравниванием по верху, тут несколько строк"
/>
```
