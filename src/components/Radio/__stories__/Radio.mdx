# Radio

Радиокнопка нужна, чтобы пользователь мог сделать выбор — только один из нескольких
вариантов.

## Название

Пишется с заглавной буквы, точка не нужна.

## Как использовать

### Только один на выбор

Список должен быть таким, чтобы пользователь мог выбрать только один вариант.
Это может быть способ доставки заказа, форма пирога, модель телефона.

Если вы понимаете, что в вашем списке можно выбрать несколько вариантов,
радиокнопки не подойдут. Используйте [Checkbox](?path=/docs/components-checkbox--playground).

<RadioExampleOne />

### Больше одного варианта

Радиокнопки подходят, если у вас в списке несколько вариантов.
Если вариант только один, возможно, пользователю не нужно делать выбор,
а значит, и радиокнопка не нужна. Если нужно выбрать один из двух вариантов,
используйте [переключатель](?path=/docs/components-switch--playground).

<RadioExampleMore />

### Хотя бы один выбранный

Радиокнопки обычно предлагают сделать выбор, без которого нельзя обойтись. Например,
если пользователь не укажет, как доставить товар, вы не сможете выбрать за него.
Поэтому обязательно отметьте один пункт выбранным.

<RadioExampleChecked />

```tsx
<p>Как вас постричь</p>
<Radio label="Под бобра" checked />
<Radio label="В кружок" />
<Radio label="А ля рюс" />
<Radio label="С бакендардами" />
<Radio label="Только кончики" />
```

## Состояния

`checked` отмечает радиокнопку выбранной.

`disabled` не дает пользователю сделать выбор.

<RadioExampleStatus />

```tsx
<Radio label="Radio" />
<Radio label="Checked" checked />
<Radio label="Disabled" disabled />
<Radio label="Checked Disabled" checked disabled />
```

## Размер

За размер отвечает свойство `size`.

<RadioExampleSize />

```tsx
<Radio size="m" label="Размер M" />
<Radio size="l" label="Размер L" />
```

## Выравнивание

За выравнивание отвечает свойство `align`.

<RadioExampleAlign />

```tsx
<Radio
  align="center"
  label="Длинный текст радиокнопки с выравниванием по центру, записанный в несколько строк"
/>
<Radio
  align="top"
  label="Длинный текст радиокнопки с выравниванием по верху, записанный в несколько строк"
/>
```
