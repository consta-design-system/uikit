# Grid

Модульная сетка — один из ключевых инструментов компановки содержимого на странице.
Ее можно встраивать в любые блоки для управления потоком контента или построить на ней свой лэйаут.

## Колонки

Для указания колонок используйте свойство `cols`

```tsx
<Grid cols="2">
  <GridItem>1</GridItem>
  <GridItem>2</GridItem>
  <GridItem>3</GridItem>
  <GridItem>4</GridItem>
</Grid>
```

## Отступы

Для указания отступов используйте свойство `gap`, `colGap`, `rowGap`

```tsx
<Grid gap="xl" cols="2">
  <GridItem>1</GridItem>
  <GridItem>2</GridItem>
  <GridItem>3</GridItem>
  <GridItem>4</GridItem>
</Grid>
```

## Breakpoints

Контрольные точки(breakpoints), добовляет возможность задавать свойства в зависимости ширины экрана.
предсмотренные контрольные точки:

- `xs`: >= 360px
- `s`: >= 640px;
- `m`: >= 1024px;
- `l`: >= 1280px;
- `xl`: >= 1440px;
- `2xl`: >= 1920px;

Ниже в примере Grid будет вести себя так:

- `>= 0px`: cols="1" gap="xl"
- `>= 360px`: cols="1" gap="2xl"
- `>= 1024pxs`: cols="2" gap="2xl"

```tsx
export function GridExampleBreakpoints() {
  return (
    <Grid
      cols="1"
      gap="xl"
      breakpoints={{
        xs: {
          gap: '2xl',
        },
        m: {
          cols: 2,
          gap: 'xl',
        },
      }}
    >
      <GridItem>1</GridItem>
      <GridItem>2</GridItem>
      <GridItem>3</GridItem>
      <GridItem>4</GridItem>
    </Grid>
  );
}
```

## Свойства Grid

```tsx
type Gap = 0 | '0' | '2xs' | 'xs' | 's' | 'm' | 'l' | 'xl' | '2xl' | '3xl' | '4xl' | '5xl' | '6xl';

type XAlign = 'left' | 'center' | 'right';

type YAlign = 'top' | 'center' | 'bottom';

type Breakpoint = {
  cols?: Cols;
  gap?: Gap;
  colGap?: Gap;
  rowGap?: Gap;
  xAlign?: XAlign;
  yAlign?: YAlign;
};

type Breakpoints = {
  'xs'?: Breakpoint;
  's'?: Breakpoint;
  'm'?: Breakpoint;
  'l'?: Breakpoint;
  'xl'?: Breakpoint;
  '2xl'?: Breakpoint;
};
```

| Свойство       | Тип                           | По умолчанию | Описание                                                                                            |
| -------------- | ----------------------------- | ------------ | --------------------------------------------------------------------------------------------------- |
| `cols?`        | `number | string;`            | —            | Количество колонок (1..12)                                                                          |
| `gap?`         | `Gap`                         | —            | Размер отступов между колонок и рядов                                                               |
| `colGap?`      | `Gap`                         | —            | Размер отступов между колонок                                                                       |
| `rowGap?`      | `Gap`                         | —            | Размер отступов между рядов                                                                         |
| `xAlign?`      | `'left' | 'center' | 'right'` | —            | Выравнивание по горизонтали                                                                         |
| `yAlign?`      | `'top' | 'center' | 'bottom'` | —            | Выравнивание по вертикали                                                                           |
| `breakpoints?` | `Breakpoints`                 | —            | Контрольные точки(breakpoints), добовляет возможность задавать свойства в зависимости ширины экрана |
| `className?`   | `string`                      | —            | Дополнительный CSS-класс                                                                            |
| `as?`          | `React.ElementType`           | 'div'        | HTML-тег, в виде которого этот компонент будет представлен в DOM                                    |
| `ref?`         | `React.Ref<HTMLDivElement>`   | -            | Ссылка на DOM-элемент компонента                                                                    |

  <GridItem>6</GridItem>
</Grid>

## Order

Для перемещения элемента используйте свойство `order`

- `order = -1` - перемещение в начало
- `order = 1` - перемещение в конец
- `order = 0` - оставить на месте

```tsx
<Grid cols="3" gap="xl">
  <GridItem>1</GridItem>
  <GridItem>2</GridItem>
  <GridItem order="1">3</GridItem>
  <GridItem>4</GridItem>
  <GridItem order="-1">5</GridItem>
  <GridItem>6</GridItem>
</Grid>
```

## Breakpoints

Контрольные точки(breakpoints), добовляет возможность задавать свойства в зависимости ширины экрана.
предсмотренные контрольные точки:

- `xs`: >= 360px
- `s`: >= 640px;
- `m`: >= 1024px;
- `l`: >= 1280px;
- `xl`: >= 1440px;
- `2xl`: >= 1920px;

```tsx
<Grid cols="3" gap="xl">
  <GridItem
    row="1"
    breakpoints={{
      m: {
        row: 3,
      },
    }}
  >
    1
  </GridItem>
  <GridItem>2</GridItem>
  <GridItem
    order="-1"
    row="3"
    breakpoints={{
      m: {
        order: 0,
        row: 1,
      },
    }}
  >
    3
  </GridItem>
  <GridItem>4</GridItem>
  <GridItem>5</GridItem>
  <GridItem>6</GridItem>
  <GridItem>7</GridItem>
</Grid>
```

```tsx
type Breakpoint = {

  's'?: Breakpoint;
  'm'?: Breakpoint;
  'l'?: Breakpoint;
  'xl'?: Breakpoint;
  '2xl'?: Breakpoint;

```

| Свойство     | Тип                         | По умолчанию | Описание                                                         |
| ------------ | --------------------------- | ------------ | ---------------------------------------------------------------- |
| `col?`       | `number`, `string`          | —            | Количество колонок которое занимает элемент (1..12)              |
| `colStart?`  | `number`, `string`          | —            | Сдвиг элемента в стобец (1..12)                                  |

| `className?` | `string`                    | —            | Дополнительный CSS-класс                                         |

| `ref?`       | `React.Ref<HTMLDivElement>` | -            | Ссылка на DOM-элемент компонента                                 |
