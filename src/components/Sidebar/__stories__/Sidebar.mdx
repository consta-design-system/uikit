# Sidebar

Всплывающее окно, «прилипающее» к краю экрана. Показывается поверх контента,
содержимое страницы можно закрыть полупрозрачной подложкой.

Внутри может быть что угодно: текст, кнопки, изображения или другие элементы.

## Содержимое

Всё, что вы хотите показать внутри сайдбара, поместите в контейнер `<Sidebar></Sidebar>`.


## Расположение

За расположение сайдбара отвечает свойство `position`. Варианты: `'right'`, `'bottom'`, `'left'`, `'top'`.
По умолчанию — `'right'`.

Если укажете `'right'` или `'left'`, сайдбар растянется на всю высоту экрана.

Если укажете `'top'` или `'bottom'` — на всю его ширину.

## Размер

Вы можете указать ширину сайдбара в свойстве `width`, а высоту — в свойстве `height`.
Доступно только одно значение — `'auto'`.


## Подложка

За подложку вокруг открытого сайдбара отвечает свойство `hasOverlay`. По умолчанию
окно открывается с подложкой: она полупрозрачная и закрывает всю страницу.

Чтобы убрать подложку, добавьте `hasOverlay=false`.

## Список свойств

| Свойство               | Тип или варианты значения             | По умолчанию              | |
| ---------------------- | --------------------------------------| ------------------------- | ------------------------- |
| [`position?`](#расположение)              | `'right', 'bottom', 'left', 'top'`    | `right`                   | Расположение сайдбара |
| [`width?`](#размер)                 | `'auto'`                              | `auto`                    | Ширина сайдбара |
| [`height?`](#размер)                | `'auto'`                              | `auto`                    | Высота сайдбара |
| `isOpen?`                | `boolean`                             | -                         | Показывает и убирает сайдбар |
| `onClose?`               | `() => void`                          | -                         | Действие при закрытии окна |
| `onOpen?`                | `() => void`                          | -                         | Действие при открытии окна |
| [`hasOverlay?`](#подложка)            | `boolean`                             | -                         | Добавляет подложку вокруг сайдбара, поверх основной страницы |
| [`onOverlayClick?`](#подложка)        | `(e: React.MouseEvent) => void`       | -                         | Действие по клику на подложку |
| `className? `            | `string`                              | -                         | Дополнительный CSS-класс для компонента |
| `children?`              | `React.ReactNode`                     | -                         | Дочерние элементы |
| `container?`             | `HTMLDivElement, undefined`           | `window.document.body`    | Родительский контейнер |

## Пример использования

```ts
<Sidebar
  className={cnSidebarStories('Sidebar')}
  isOpen={isSidebarOpen}
  onClose={() => console.log('Коллбэк на закрытие')}
  onOpen={() => console.log('Коллбэк на открытие')}
  hasOverlay={hasOverlay}
  onOverlayClick={() => setIsSidebarOpen(false)}
  width={width}
  height={height}
  position={position}
>
  <Sidebar.Content className={cnSidebarStories('Content')}>
    <Text
      as="p"
      size="l"
      view="primary"
      weight="semibold"
      className={cnSidebarStories('Title')}
    >
      Заголовок сайдбара
    </Text>
    <Text as="p" size="m" view="secondary" className={cnSidebarStories('Body')}>
      Sed porta sollicitudin purus sagittis elementum. Pellentesque habitant morbi tristique
      senectus et netus et malesuada fames ac turpis egestas. Pellentesque tincidunt, augue et
      placerat cursus, neque sem imperdiet mi, ut fringilla nunc nibh non enim. Vivamus id
      metus convallis, commodo magna vel, scelerisque nisi.
    </Text>
  </Sidebar.Content>
  <Sidebar.Actions className={cnSidebarStories('Actions')}>
    <Button
      size="m"
      view="clear"
      label="Ок"
      width="default"
      onClick={() => setIsSidebarOpen(false)}
    />
    <Button
      size="m"
      view="ghost"
      label="Закрыть"
      width="default"
      onClick={() => setIsSidebarOpen(false)}
    />
  </Sidebar.Actions>
</Sidebar>
```
