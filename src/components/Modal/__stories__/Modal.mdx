# Modal

Всплывающее окно. Контейнер, в котором можно показать любое содержимое поверх основного экрана.
Там может быть просто текст или что угодно: таблица, форма, изображение.

## Содержимое

Всё, что вы показать в модальном окне, поместите в контейнер `<Modal></Modal>`.

<ModalExampleText />


## Размер

Вы можете указать ширину модального окна в свойстве `width`.

## Расположение

За расположение модального окна отвечает свойство `position`. Варианты: `'center'`, `'top'`,
по умолчанию — `'center'`.

## Подложка

За подложку вокруг открытого модального окна отвечает свойство `hasOverlay`. По умолчанию
окно открывается с подложкой: она полупрозрачная и закрывает всю страницу.

Чтобы убрать подложку, добавьте `hasOverlay=false`.

## Свойства

| Свойство               | Тип или варианты значения      | По умолчанию              | Описание |
| ---------------------- | ------------------------------ | ------------------------- | ------------------------- |
| [`width?`](#размер)                 | `'auto'`                       | `auto`                    |  Ширина модального окна |
| [`position?`](#расположение)              | `'center'`, `'top'`            | `center`                  | Расположение модального окна|
| [`hasOverlay?`](#подложка)             | `boolean`                      | `true`                    | Добавляет подложку вокруг окна — поверх содержимого страницы |
| [`onOverlayClick?`](#подложка)         | `(e: React.MouseEvent) => void`| -                         | Действие по нажатию на подложку —  область за пределами окна |
| `isOpen?`                | `boolean`                      | -                         | Открывает и закрывает окно|
| `onClose?`               | `() => void`                   | -                         | Действие при закрытии окна|
| `onOpen?`                | `() => void`                   | -                         | Действие при открытии окна|
| `children?`              | `React.ReactNode`              | -                         | Дочерние элементы |
| `container?`             | `HTMLDivElement, undefined`    | `window.document.body`    | Родителький контейнер для модального окна|
| `className?`             | `string`                       | -                         | Дополнительный CSS-класс компонента |

## Пример

```ts
<Modal className='myClassName' isOpen={isModalOpen} onOverlayClick={() => setIsModalOpen(false)}>
  <Text as='p' size='s' view='secondary' className={cnModalStories('title')}>
    Заголовок модалки
  </Text>
  <Text as='p' size='m' view='primary' className={cnModalStories('body')}>
    Описание в теле модалки. Здесь может находиться какая-то информация.
    Lorem ipsum dolor sit amet, consectetur adipiscing elit, sed do eiusmod tempor incididunt.
  </Text>
  <div className={cnModalStories('action')}>
    <Button
      size="m"
      view="primary"
      label="Закрыть"
      width='default'
      onClick={() => setIsModalOpen(false)}
    />
  </div>
</Modal>
```
