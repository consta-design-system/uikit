## Обзор

Компонент `Tag` используется для создания тегов, которые обозначают характеристики или категории объектов. Теги могут быть интерактивными (например, кнопки, ссылки, переключатели) или информационными, в зависимости от настроек. Поддерживаются различные размеры, иконки и режимы работы.

## Импорт

```tsx
import { Tag } from '@consta/uikit/Tag';
```

## Свойства

| ---------------------------- | ---------------------------------------------------------------------------------------------------------- | ------------ | ----------------------------------------------------------------------- |
| [`label?`](#текст)           | `string`                                                                                                   | -            | Текст тега                                                              |
| [`icon?`](#иконка)           | `IconComponent`                                                                                            | -            | Иконка                                                                  |
| [`group?`](#группы-тегов)    | `1 \| 2 \| 3 \| 4 \| 5 \| 6 \| 7 \| 8 \| 9 \| '1' \| '2' \| '3' \| '4' \| '5' \| '6' \| '7' \| '8' \| '9'` | -            | Метка группы                                                            |
| [`size?`](#размер)           | `'xs' \| 's' \| 'm' \| 'l'`                                                                                | `'m'`        | Размер тега                                                             |
| [`mode?`](#режим-работы)     | `'button' \| 'link' \| 'check' \| 'cancel' \| 'info'`                                                      | `'button'`   | Тип тега                                                                |
| [`checked?`](#режим-работы)  | `boolean`                                                                                                  | -            | Отмечает тег выбранным. Доступно только при `mode="check"`              |
| [`onChange?`](#режим-работы) | `({ e, checked }: { e?: React.MouseEvent; checked: boolean }) => void`                                     | -            | Действие при клике по тегу. Доступно только при `mode="check"`          |
| [`onClick?`](#режим-работы)  | `React.MouseEventHandler`                                                                                  | -            | Действие при клике по тегу. Доступно при `mode="button"`, `mode="link"` |
| [`onCancel?`](#режим-работы) | `React.MouseEventHandler`                                                                                  | -            | Действие при клике по крестику. Доступно только при `mode="cancel"`     |
| [`disabled?`](#режим-работы) | `boolean`                                                                                                  | `false`      | Блокирует тег                                                           |
| `className?`                 | `string`                                                                                                   | -            | Дополнительный CSS-класс                                                |
| `ref?`                       | `React.Ref<HTMLElement>`                                                                                   | -            | Ссылка на корневой DOM-элемент этого компонента                         |

## Содержимое

### Текст

Свойство `label` задаёт текст тега. Рекомендуется использовать существительные или прилагательные в единственном числе, чтобы обозначить характеристику объекта. Например, теги для галереи изображений могут выглядеть так:

```tsx
<Tag label="Рисунок" />
<Tag label="Портрет" />
<Tag label="Фото" />
<Tag label="Полосатый" />
<Tag label="В клеточку" />
```

<TagExampleText />

### Иконка

Свойство `icon` позволяет добавить иконку к тегу. Используйте иконки из [библиотеки иконок Consta](##LIBS.LIB/lib:icons).

```tsx
<Tag icon={IconMoon} label="Луна" />
<Tag icon={IconSun} label="Солнце" />
```

<TagExampleIcon />

### Группы тегов

Свойство `group` позволяет пометить тег числовой меткой (от 1 до 9). Это полезно для категоризации или визуального выделения тегов.

## Внешний вид

### Размер

Свойство `size` определяет размер тега. Доступные значения: `'xs'`, `'s'`, `'m'`, `'l'`. По умолчанию используется `'m'`.

```tsx
<Tag size="xs" label="Размер XS" />
<Tag size="s" label="Размер S" />
<Tag size="m" label="Размер M" />
<Tag size="l" label="Размер L" />
```

## Поведение

### Режим работы

Свойство `mode` задаёт тип тега и его поведение. Доступные режимы:

- `button` — тег ведёт себя как кнопка (по умолчанию). Используйте с обработчиком `onClick`.
- `link` — тег становится ссылкой. Используйте с `href` и `onClick`.
- `check` — тег поддерживает состояния "выбран" и "не выбран". Используйте с `checked` и `onChange`.
- `cancel` — тег можно удалить, нажав на крестик. Используйте с `onCancel`.
- `info` — тег информационный, без интерактивности.

Свойство `disabled` блокирует взаимодействие с тегом.

```tsx
<Tag mode="button" onClick={action('onClick')} label="Button" />
<Tag mode="link" label="Link" href="#" />
<Tag mode="check" onChange={action('onChange')} label="Check" checked={false} />
<Tag mode="cancel" onCancel={action('onCancel')} label="Cancel" />
```

## Пример использования

```tsx
// src/App.ts
import React from 'react';
import { Tag } from '@consta/uikit/Tag';

const handleClick = () => console.log('click');

const App = () => {
  return <Tag mode="button" label="Label" onClick={handleClick} />;
};
```
