## Режим работы

За тип тега отвечает свойство `mode`.

- `button` делает из тега кнопку, такой тег можно нажимать. Это основной тип тега, он включен по умолчанию.
- `link` превращает тег в ссылку.
- `check` поддерживает два состояния: выбранное и невыбранное. Используйте этот тип, если хотите, чтобы пользователь мог отмечать некоторые теги. Чтобы этот режим заработал, используйте `onChange`.
- `cancel` нужен, чтобы у пользователя была возможность удалить тег. Чтобы этот режим заработал, используйте `onCancel`.

```ts
<Tag mode="button" onClick={action('onClick')} label="Button" />
<Tag mode="link" label="Link" href="#" />
<Tag mode="check" onChange={action('onChange')} label="Check" checked={false} />
<Tag mode="cancel" onCancel={action('onCancel')} label="Cancel" />
```

## Текст

Тег — это характеристика объекта. Поэтому в тексте обычно используются
существительные и прилагательные в единственном числе.
Например, в галерее изображений могут быть такие теги:

```ts
<Tag label="Рисунок" />
<Tag label="Портрет" />
<Tag label="Фото" />
<Tag label="Полосатый" />
<Tag label="В клеточку" />
```

### Иконка

Вы можете добавить к тегу иконку — любую из [библиотеки иконок](##LIBS.LIB/lib:icons).

```ts
<Tag icon={IconMoon} label="Луна" />
<Tag icon={IconSun} label="Солнце" />
```

## Размер

За размер тега отвечает свойство `size`. По умолчанию — `m`.

```ts
<Tag size="xs" label="Размер XS" />
<Tag size="s" label="Размер S" />
<Tag size="m" label="Размер M" />
<Tag size="l" label="Размер L" />
```

## Свойства

| Свойство                     | Тип и значения                                                                | По умолчанию | Описание                                                                        |
| ---------------------------- | ----------------------------------------------------------------------------- | ------------ | ------------------------------------------------------------------------------- |
| [`label?`](#текст)           | `string`                                                                      | -            | Текст тега                                                                      |
| [`mode?`](#режим-работы)     | `'check', 'button', 'cancel','link'`                                          | `'button'`   | Тип тега                                                                        |
| [`checked?`](#режим-работы)  | `boolean, never`                                                              | -            | Отмечает тег выбранным. Доступно только при `mode` = `'check'`                  |
| [`size?`](#размер)           | `'xs' , 's' , 'm' , 'l'`                                                      | `'m'`        | Размер тега                                                                     |
| [`group?`](#группы-тегов)    | `1..9, '1'...'9'`                                                             | -            | Метка группы                                                                    |
| [`icon?`](#иконка)           | `IconComponent`                                                               | -            | Иконка                                                                          |
| `onClick?`                   | `React.MouseEventHandler, never`                                              | -            | Действие при клике по тегу. Доступно при `mode` = `'button'`, `'link'`          |
| [`onCancel?`](#режим-работы) | `React.MouseEventHandler, never`                                              | -            | Действие при клике по крестику на теге. Доступно только при `mode` = `'cancel'` |
| [`onChange?`](#режим-работы) | `({ e, checked }: { e?: React.MouseEvent; checked: boolean }) => void, never` | -            | Действие при клике по тегу. Доступно только при `mode` = `'check'`              |
| `className?`                 | `string`                                                                      | -            | Дополнительный CSS-класс                                                        |
| `ref?`                       | `React.Ref<HTMLElement>`                                                      | -            | Ссылка на корневой DOM-элемент этого компонента                                 |

## Пример

```ts
// src/App.ts
import React from 'react';
import { Tag } from '@consta/uikit/Tag';

const handleClick = () => console.log('click');

const App = () => {
  return <Tag mode="button" label="Label" onClick={handleClick} />;
};
```
