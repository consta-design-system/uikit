## Обзор

Компонент `Badge` используется для отображения статуса или дополнительной информации об объекте в виде компактного бейджика. Он может содержать текст, иконки или быть минималистичным (без текста). Подходит для обозначения состояний, таких как "черновик", "на проверке" или "готов", а также для привлечения внимания к определенным элементам интерфейса.

Типичные сценарии использования:

- Отображение статуса документа (например, "новый", "ошибка").
- Индикация состояния процесса (например, "в процессе", "завершено").
- Минималистичные метки для компактных интерфейсов.

## Импорт

```tsx
import { Badge } from '@consta/uikit/Badge';
```

## Свойства

```tsx
type BadgePropStatus = 'success' | 'alert' | 'warning' | 'normal' | 'system';
type BadgePropView = 'filled' | 'stroked';
type BadgePropSize = 's' | 'm' | 'l';
type BadgePropForm = 'default' | 'round';
```

| Свойство                           | Тип                                                         | По умолчанию | Описание                                   |
| ---------------------------------- | ----------------------------------------------------------- | ------------ | ------------------------------------------ |

| [`view?`](#типы-бейджиков)         | `'filled' \| 'stroked'`                                     | `'filled'`   | Тип бейджика (заполненный или с обводкой)  |
| [`status?`](#статус)               | `'success' \| 'alert' \| 'warning' \| 'normal' \| 'system'` | `'normal'`   | Статус бейджика                            |
| [`form?`](#форма)                  | `'default' \| 'round'`                                      | `'default'`  | Форма бейджика                             |
| [`minified?`](#бейджик-без-текста) | `boolean`                                                   | -            | Отображает бейджик без текста в виде точки |
| [`iconLeft?`](#бейджик-с-иконкой)  | `IconComponent`                                             | -            | Иконка слева от текста                     |
| [`iconRight?`](#бейджик-с-иконкой) | `IconComponent`                                             | -            | Иконка справа от текста                    |

## Статус

Свойство `status` определяет состояние бейджика и его цветовую индикацию. Возможные значения:

- `normal` — нейтральный статус (по умолчанию).
- `success` — успешное завершение.
- `alert` — ошибка или проблема.
- `warning` — предупреждение, требующее внимания.
- `system` — системное сообщение.

<MdxTabs>

<BadgeExampleStatus />

```tsx
import React from 'react';
import { Badge } from '@consta/uikit/Badge';

const BadgeExampleStatus = () => {
  return (
    <>
      <Badge status="system" label="Черновик" />
      <Badge status="normal" label="Новый" />
      <Badge status="warning" label="На проверке" />
      <Badge status="alert" label="Есть ошибки" />
      <Badge status="success" label="Готов" />
    </>
  );
};
```

</MdxTabs>

## Содержимое

### Бейджик с текстом

Свойство `label` задаёт текст, который отображается на бейджике. Используется для описания статуса или дополнительной информации.

<MdxTabs>

<BadgeExampleStatus />

```tsx
import React from 'react';
import { Badge } from '@consta/uikit/Badge';

const BadgeExampleStatus = () => {
  return (
    <>
      <Badge status="system" label="Черновик" />
      <Badge status="normal" label="Новый" />
      <Badge status="warning" label="На проверке" />
      <Badge status="alert" label="Есть ошибки" />
      <Badge status="success" label="Готовка" />
    </>
};
```

</MdxTabs>

### Бейджик без текста

Свойство `minified` позволяет отображать бейджик в виде компактной точки без текста, полагаясь на цвет для индикации статуса.

<MdxTabs>

<BadgeExampleMinified />

```tsx
import React from 'react';
import { Badge } from '@consta/uikit/Badge';

const BadgeExample = () => {
  return (
    <>
      <Badge minified="true" status="alert" label="Стойте" />
      <Badge minified="true" status="success" label="Идите" />
    </>
  );
};
```

</MdxTabs>

### Бейджик с иконкой

Свойства `iconLeft` и `iconRight` добавляют иконки слева или справа от текста. Используются для усиления визуального контекста. Иконки берутся из [библиотеки иконок Consta](##LIBS.LIB/lib:icons).

<MdxInformer status="warning">

С версии 4.17.2 свойство `icon` помечено как `deprecated`. Используйте `iconLeft` или `iconRight`.

</MdxInformer>

<MdxTabs>

<BadgeExampleIcon />

```tsx
import React from 'react';
import { Badge } from '@consta/uikit/Badge';
import { IconCheck } from '@consta/uikit/IconCheck';
import { IconClose } from '@consta/uikit/IconClose';

const BadgeExampleIcon = () => {
  return (
    <>
      <Badge iconLeft={IconCheck} status="success" label="Съедобно" />
      <Badge iconRight={IconClose} status="alert" label="Отрава" />
    </>
  );
};
```

</MdxTabs>

## Внешний вид

### Типы бейджиков

Свойство `view` определяет стиль бейджика:

- `filled` — заполненный фон, для акцента на статусе.
- `stroked` — с обводкой, менее заметный.

<MdxTabs>

<BadgeExampleView />

```tsx
import React from 'react';
import { Badge } from '@consta/uikit/Badge';

const BadgeExampleView = () => {
  return (
    <>
      <Badge view="filled" label="Filled badge" />
      <Badge view="stroked" label="Stroked badge" />
    </>
  );
};
```

</MdxTabs>

### Размер

Свойство `size` задаёт размер бейджика: `s`, `m` (по умолчанию), `l`.

<MdxTabs>

<BadgeExampleSize />

```tsx
import React from 'react';
import { Badge } from '@consta/uikit/Badge';

const BadgeExampleSize = () => {
  return (
    <>

      <Badge size="m" label="Badge M" />

};
```

</MdxTabs>

### Форма

Свойство `form` определяет форму бейджика: `default` (прямоугольная, по умолчанию) или `round` (скруглённая).

<MdxTabs>

<BadgeExampleForm />

```tsx
const BadgeExampleForm = () => {
  return (
    <>
      <Badge form="default" label="Default badge" />
      <Badge form="round" label="Round badge" />
    </>
  );
};
```

</MdxTabs>

## Пример использования

Пример демонстрирует базовое использование компонента `Badge` для отображения статуса.

```tsx
import React from 'react';
import { Badge } from '@consta/uikit/Badge';

const App = () => {
  return <Badge label="badge" />;
};
```
