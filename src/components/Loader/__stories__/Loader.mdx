# Loader

Прелоадер показывает, что информация загружается.


## Свойства

<!-- props:start -->

| Свойство         | Тип                        | По умолчанию  | Описание                                     |
| ---------------- | ---------------------------| ------------- | -------------------------------------------- |
| size             | `'s', 'm'`                 | `'s'`         | Размер прелоадера                            |
| className        | `string`                   | -             | Дополнительный CSS-класс                     |
| ref              | `React.Ref<HTMLDivElement>`| -             | Ссылка на корневой DOM-элемент этого компонента |

<!-- props:end -->

## Пример

```ts
// src/App.ts
import React from 'react';
import { Loader } from '@gpn-design/uikit/Loader';

const App = () => {
  return <Loader />;
};
```
