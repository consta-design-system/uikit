# Loader

```tsx
import { Loader } from '@consta/uikit/Loader';
```

Прелоадер показывает, что информация загружается.

- [Размер](#размер)
- [Импорт](#импорт)
- [Список свойств](#свойства)
- [Пример использования](#пример)

## Размер

За размер прелоадера отвечает свойство `size`.

```tsx
<Loader size="m" />
<Loader size="s" />
```

| Размер M               | Размер S               |
| ---------------------- | ---------------------- |
| <LoaderExampleSizeM /> | <LoaderExampleSizeS /> |

## Импорт

```tsx
import { Loader } from '@consta/uikit/Loader';
```

## Свойства

<!-- props:start -->

| Свойство           | Тип                         | По умолчанию | Описание                                        |
| ------------------ | --------------------------- | ------------ | ----------------------------------------------- |
| [`size?`](#размер) | `'s', 'm'`                  | `'m'`        | Размер прелоадера                               |
| `className?`       | `string`                    | -            | Дополнительный CSS-класс                        |
| `ref?`             | `React.Ref<HTMLDivElement>` | -            | Ссылка на корневой DOM-элемент этого компонента |

<!-- props:end -->

## Пример

```ts
// src/App.ts
import React from 'react';
import { Loader } from '@consta/uikit/Loader';

const App = () => {
  return <Loader />;
};
```
