## Тип

За размер компонента отвечает свойство `type`.

Может принимать значения:

- `'dots'` - в виде точек
- `'circle'` - в виде круга

<MdxTabs>

```tsx
<Loader type="dots" />
<Loader type="circle" />
```

</MdxTabs>

### Кастомный размер

Чтобы установить любой размер, используйте css-переменную `--loader-size-prop`

<MdxTabs>

```tsx
import './LoaderExampleCustomSize.css';

import React from 'react';
import { cn } from '##/utils/bem';
import { Loader } from '@consta/uikit/Loader';

const cnLoaderExampleCustomSize = cn('LoaderExampleCustomSize');

export const LoaderExampleCustomSize = () => (
  <Loader className={cnLoaderExampleCustomSize()} />
);
```

```css
.LoaderExampleCustomSize {
  --loader-size-prop: 70px;
}
```

<LoaderExampleCustomSize />

</MdxTabs>

### Кастомный цвет

Для перекрашивания компонента в любой цвет используйте css-переменную `--loader-color-prop`.

<MdxTabs>

```tsx
import './LoaderExampleCustomColor.css';

import React from 'react';
import { cn } from '##/utils/bem';
import { Loader } from '@consta/uikit/Loader';

const cnLoaderExampleCustomColor = cn('LoaderExampleCustomColor');

export const LoaderExampleCustomColor = () => (
  <Loader type="circle" className={cnLoaderExampleCustomColor()} />
);
```

```css
.LoaderExampleCustomColor {
  --loader-color-prop: var(--color-typo-primary);
}
```

<LoaderExampleCustomColor />

</MdxTabs>

## Свойства компонента

| Свойство           | Тип                                                        | По умолчанию | Описание                                        |
| ------------------ | ---------------------------------------------------------- | ------------ | ----------------------------------------------- |
| [`size?`](#размер) | `'xs'` <code>&#124;</code> `'s'` <code>&#124;</code> `'m'` | `'m'`        | Размер                                          |
| [`type?`](#тип)    | `'dots'` <code>&#124;</code> `'circle'`                    | `dots'`      | Тип                                             |
| [`view?`](#тип)    | `'primary'` <code>&#124;</code> `'clear'`                  | `primary'`   | Вид                                             |
| `className?`       | `string`                                                   | -            | Дополнительный CSS-класс                        |
| `ref?`             | `React.Ref<HTMLDivElement>`                                | -            | Ссылка на корневой DOM-элемент этого компонента |

## Сss-переменные

| Свойство              | Описание          |
| --------------------- | ----------------- |
| `--loader-color-prop` | Установка цвета   |
| `--loader-size-prop`  | Установка размера |
