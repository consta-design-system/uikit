## Обзор

`Collapse` — это компонент для компактного отображения информации с возможностью сворачивания и разворачивания. Он состоит из заголовка и содержимого, которое показывается или скрывается при клике на заголовок. Подходит для экономии пространства в интерфейсе, например, в списках, FAQ или настройках.

## Импорт

```tsx
import { Collapse } from '@consta/uikit/Collapse';
```

## Содержимое

### Заголовок

Свойство `label` задаёт текст или содержимое заголовка коллапса, видимого в закрытом состоянии. Заголовок может быть текстом, компонентом или их комбинацией. Рекомендуется писать заголовок с прописной буквы без точки в конце.

<MdxTabs>

<CollapseExampleLabel />

```tsx
<>
  <Collapse
    label={
      <>
        Заголовок
        <Badge label="Статус" size="s" />
      </>
    }
    isOpen={isOpen}
    onClick={() => setOpen(!isOpen)}
  >
    {content}
  </Collapse>
</>
```

</MdxTabs>

### Дополнительный элемент

Свойство `rightSide` позволяет добавить справа от заголовка дополнительный контент, например, иконку, бейджик или кнопку, для выделения коллапса или группировки.

<MdxTabs>

<CollapseExampleIconRightSideBadge />

```tsx
<>
  <Collapse
    label="Здесь справа что-то есть"
    isOpen={isOpen}
    onClick={() => setOpen(!isOpen)}
    rightSide={[
      <Badge label="Badge" status="success" />,
      <IconAlert size="s" view="warning" />,
    ]}
  >
    {content}
  </Collapse>
</>
```

</MdxTabs>

<MdxTabs>

<CollapseExampleIconRightSideButton />

```tsx
<>
  <Collapse
    label="А здесь вообще справа настройки"
    isOpen={isOpen}
    onClick={() => setOpen(!isOpen)}
    rightSide={
      <Button
        iconLeft={IconSettings}
        size="xs"
        view="ghost"
        onClick={(e) => {
          e.stopPropagation();
          alert('Button Click');
        }}
      />
    }
  >
    {content}
  </Collapse>
</>
```

</MdxTabs>

## Внешний вид

### Форма

Свойство `form` определяет форму компонента. Варианты:

- `brick` — без скругления (по умолчанию),
- `default` — со скруглёнными углами,
- `round` — с полностью круглыми углами.

<MdxTabs>

<CollapseExampleForm />

```tsx
<>
  <Collapse
    label="Brick"
    form="brick"
    hoverEffect
    horizontalSpace="m"
    isOpen={isOpen}
    onClick={() => setOpen(!isOpen)}
  >
    {content}
  </Collapse>
  <Collapse
    label="Default"
    form="default"
    hoverEffect
    horizontalSpace="m"
    isOpen={isOpenTwo}
    onClick={() => setOpenTwo(!isOpenTwo)}
  >
    {content}
  </Collapse>
  <Collapse
    label="Round"
    form="round"
    horizontalSpace="m"
    hoverEffect
    isOpen={isOpenThree}
    onClick={() => setIsOpenThree(!isOpenThree)}
  >
    {content}
  </Collapse>
</>
```

</MdxTabs>

### Размер

Свойство `size` задаёт размер заголовка, иконок и отступов. На содержимое коллапса не влияет. Варианты: `'l'`, `'m'`, `'s'`, `'xs'`, `'2xs'`. По умолчанию — `'m'`.

<MdxTabs>

<CollapseExampleSize />

```tsx
    size="l"
    isOpen={isOpen}
    onClick={() => setOpen(!isOpen)}
  >
    {content}
  </Collapse>
</>
```

</MdxTabs>

## Иконка

По умолчанию коллапс использует иконки `IconArrowDown` (закрыт) и `IconArrowUp` (открыт).

### Внешний вид иконки

Свойство `iconView` изменяет внешний вид иконки. Варианты: `'clear'` (по умолчанию), `'ghost'`.

<MdxTabs>

<CollapseExampleIconView />

</MdxTabs>

### Замена иконки

Свойства `icon` и `closeIcon` позволяют заменить стандартные иконки на любые из [библиотеки иконок Consta](##LIBS.LIB/lib:icons). `icon` задаёт иконку в закрытом состоянии, `closeIcon` — в открытом.

<MdxTabs>

<CollapseExampleIconCustomIcon />

```tsx
<>
  <Collapse
    label="Здесь меняется иконка"
    isOpen={isOpen}
    onClick={() => setOpen(!isOpen)}
    icon={IconAdd}
    closeIcon={IconRemove}
  >
    {content}
  </Collapse>
</>
```

</MdxTabs>

### Расположение иконки

Свойство `iconPosition` определяет расположение иконки относительно заголовка. Варианты: `'left'` (по умолчанию), `'right'`.

<MdxTabs>

<CollapseExampleIconPosition />

```tsx
<>
  <Collapse
    label="Иконка слева"
    isOpen={isOpen}
    onClick={() => setOpen(!isOpen)}
    iconPosition="left"
  >
    {content}
  </Collapse>
  <Collapse
    label="Иконка справа"
    isOpen={isOpenTwo}
    onClick={() => setOpenTwo(!isOpenTwo)}
    iconPosition="right"
  >
    {content}
  </Collapse>
</>
```

</MdxTabs>

### Направление иконки

Свойства `directionIcon` и `closeDirectionIcon` задают направление иконки в закрытом и открытом состоянии соответственно. Варианты: `'up'`, `'right'`, `'down'`, `'left'`, `'upRight'`, `'downRight'`, `'upLeft'`, `'downLeft'`. По умолчанию: `'up'` для `directionIcon`, `'down'` для `closeDirectionIcon`.

<MdxTabs>

<CollapseExampleIconCustomIconDirection />

```tsx
<>
  <Collapse
    label="Здесь у иконки меняется направление"
    isOpen={isOpen}
    onClick={() => setOpen(!isOpen)}
    icon={IconAdd}
    directionIcon="up"
    closeDirectionIcon="downRight"
  >
    {content}
  </Collapse>
</>
```

</MdxTabs>

## Отступы

Свойство `horizontalSpace` регулирует внутренние боковые отступы между текстом и границами компонента. Варианты: `'6xl'`, `'5xl'`, `'4xl'`, `'3xl'`, `'2xl'`, `'xl'`, `'l'`, `'m'`, `'s'`, `'xs'`, `'2xs'`, `'3xs'`. По умолчанию — 0.

<MdxTabs>

<CollapseExampleSpace />

```tsx
<>
  <Collapse
    label="Отступы 6xl"
    horizontalSpace="6xl"
    isOpen={isOpen}
    onClick={() => setOpen(!isOpen)}
    hoverEffect
  >
    {content}
  </Collapse>
</>
```

</MdxTabs>

## Пример использования

```tsx
import React from 'react';
import { Collapse } from '@consta/uikit/Collapse';

const App = () => {
  const [isOpen, setOpen] = React.useState<boolean>(false);

  return (
    <Collapse
      label="Заголовок коллапса"
      isOpen={isOpen}
      onClick={() => setOpen(!isOpen)}
    >
      Тело коллапса
    </Collapse>
  );
};
```
