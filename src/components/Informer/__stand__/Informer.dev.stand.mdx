## Обзор

Компонент `Informer` используется для отображения информационных сообщений, уведомлений или предупреждений. Он поддерживает настройку статуса, текста, внешнего вида, иконок и размеров, что делает его универсальным для различных сценариев.

## Импорт

```tsx
import { Informer } from '@consta/uikit/Informer';
```

| [`label?`](#текст-сообщения)   | `string`                                                                                                 | -            | Текст сообщения                      |
| [`status?`](#статус-сообщения) | `'success'` <code>&#124;</code> `'warning'` <code>&#124;</code> `'alert'` <code>&#124;</code> `'system'` | `'success'`  | Статус сообщения                     |
| [`size?`](#размер)             | `'m'` <code>&#124;</code> `'s'`                                                                          | `'m'`        | Размер                               |
| [`view?`](#стиль)              | `'filled'` <code>&#124;</code> `'bordered'` <code>&#124;</code> `'outline'`                              | `'filled'`   | Стиль информера (с заливкой или без) |

## Содержимое

### Текст сообщения

Для задания текста в компоненте `Informer` используйте свойства:

- `title` — задаёт текст заголовка сообщения.
- `label` — задаёт основной текст сообщения.
- `children` — позволяет передать дополнительное содержимое в виде `React.ReactNode`.

### Статус сообщения

Свойство `status` определяет тип сообщения и его визуальное оформление:

- `success` — для обозначения успешной операции.
- `warning` — для обозначения ошибки.
- `alert` — для обозначения ограничений, блокирующих операцию.
- `system` — для важной, но не влияющей на результат операции информации.

<MdxTabs>

<InformerExampleStatus />

```tsx
<Informer
  status="success"
  view="filled"
  label="Файл Отпуск_2025 успешно загружен"
/>
<Informer
  status="warning"
  view="filled"
  title="Заполните все поля"
  label="Поле Конец отпуска не заполнено"
/>
<Informer
  status="alert"
  view="filled"
  title="Не удалось загрузить файл"
  label="Максимальный размер файла 20 Гб"
/>
<Informer
  status="system"
  view="filled"
  title="Файл сохранен"
  label="Файл Отпуск_2025 сохранен на Рабочий стол"
/>
```

</MdxTabs>

## Внешний вид

### Размер

Свойство `size` задаёт размер информера:

- `m` — стандартный размер (по умолчанию).
- `s` — уменьшенный размер.

<MdxTabs>

<InformerExampleSize />

```tsx
<Informer
  status="system"
  view="filled"
  title="Размер m"
  label="Это обычное сообщение с обычным размером"
  size="m"
/>
<Informer
  status="system"
  view="filled"
  title="Размер s"
  label="Это сообщение поменьше"
  size="s"
/>
```

</MdxTabs>

### Стиль

Свойство `view` задаёт стиль отображения информера:

- `filled` — информер с заливкой.
- `outline` — информер без заливки, с обводкой.
- `bordered` — информер без заливки, с границей.

<MdxTabs>

<InformerExampleView />

```tsx
<Informer label="Сообщение отправлено" view="filled" status="success" />
<Informer label="Работа выполнена" view="outline" status="success" />
<Informer label="Всё заполнено" view="bordered" status="success" />
```

</MdxTabs>

### Иконка

Чтобы добавить иконку в информер, используйте свойство `icon`. Полный набор иконок Consta смотрите в [библиотеке иконок](##LIBS.LIB/lib:icons).

<MdxInformer status="warning">

Рекомендуем добавлять иконки в информер с заливкой (`view="filled"`).

</MdxInformer>
