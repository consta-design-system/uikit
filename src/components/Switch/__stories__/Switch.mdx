# Switch

Переключатель с двумя положениями: **включено** и **выключено**. Нужен, чтобы переключать
режимы — мгновенно, сразу после того, как пользователь меняет положение переключателя.

## Название

Пишется с заглавной, точка не нужна.

Сформулируйте название так, чтобы текст был понятен в обоих состояниях —
и во включенном, и в выключенном. Ориентируетесь не на процесс (что включается),
а на результат: что будет, когда переключатель включится.

К сожалению, это не всегда получается, и пользователи могут запутаться. Если можете,
используйте другой компонент, например,
[ChoiceGroup](?path=/docs/components-choicegroup--playground)
из двух элементов.

### С глаголом

Если в названии есть **глагол**, представьте, что это вопрос,
а переключатель отвечает на него только **да** и **нет**. Если получившийся
диалог правильно описывает то, что происходит в интерфейсе, значит, всё в порядке.

<SwitchExampleNameVerb />

> Глагол **включить** лучше не использовать (даже если очень хочется). Потому что
> с этим глаголом сложно понять, что будет, если нажать на активный переключатель.

<SwitchExampleNameEnable />

### Без глагола

Если в названии глагола нет, подставьте вместо переключателя значения **включен** и **выключен**
и попробуйте прочитать, что получилось. Если фраза правильно описывает то,
что происходит в интерфейсе, значит, всё в порядке.

<SwitchExampleNameNoVerb />

```tsx
  <Switch label="Тестовый режим" checked/>
  <Switch label="Темная тема" />
  <Switch label="Автоматическая трансформация" />
```

## Состояния

Чтобы включить переключатель, добавьте `checked`.

Чтобы сделать переключатель неактивным, добавьте `disabled`.

<SwitchExampleStatus />

```tsx
<Switch label="Switch"/>
<Switch label="Checked" checked/>
<Switch label="Disabled" disabled />
```

## Размер

За размер переключателя отвечает свойство `size`.

<SwitchExampleSize />

```tsx
<Switch size="s" label="Размер S" checked/>
<Switch size="m" label="Размер М" />
<Switch size="l" label="Размер L" />
```

## Выравнивание

За выравнивание отвечает свойство `align`.

<SwitchExampleAlign />

```tsx
<Switch
  align="center"
  label="Длинный текст переключателя с выравниванием по центру, записанный в несколько строк"
/>
<Switch
  align="top"
  label="Длинный текст переключателя с выравниванием по верху, записанный в несколько строк"
/>
```

## Свойства

| Свойство                  | Тип и значения                                                                  | По умолчанию | Описание                                                                     |
| ------------------------- | ------------------------------------------------------------------------------- | ------------ | ---------------------------------------------------------------------------- |
| [`label?`](#название)     | `string`                                                                        | -            | Текст рядом с переключателем                                                 |

| [`checked`](#состояния)   | `boolean`, `undefined`                                                          | -            | Ставит переключатель в положение **включен**                                 |
| [`disabled?`](#состояния) | `boolean`                                                                       | -            | Делает переключатель неактивным                                              |
| `required?`               | `boolean`                                                                       | -            | Такой переключатель обязательно нужно выбрать                                |
| `autoFocus?`              | `boolean`                                                                       | -            | Переносит фокус на этот переключатель                                        |
| `tabIndex?`               | `number`                                                                        | -            | Порядковый номер при переходе между элементами интерфейса по клавише **tab** |  |
| `name?`                   | `string`                                                                        | -            | Имя элемента в DOM                                                           |
| `onChange`                | `(object: {e: React.ChangeEvent<HTMLInputElement>;checked: boolean;}) => void;` | -            | Действие при смене состояния переключателя                                   |
| `onFocus?`                | `React.FocusEventHandler<HTMLInputElement>`                                     | -            | Действие по фокусу                                                           |
| `onBlur?`                 | `React.FocusEventHandler<HTMLInputElement>`                                     | -            | Действие при потере фокуса                                                   |
| `inputRef?`               | `React.Ref<HTMLInputElement>`                                                   | -            | Ссылка на DOM-элемент `<input>`                                              |
| `ref?`                    | `React.Ref<HTMLLabelElement>`                                                   | -            | Ссылка на корневой DOM-элемент этого компонента                              |

## Пример

```ts
// src/App.ts
import React from 'react';
import { Switch } from '@consta/uikit/Switch';

const App = () => {
  return <Switch label="Меня можно включать и выключать" />;
};
```
