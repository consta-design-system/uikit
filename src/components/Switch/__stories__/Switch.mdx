# Switch

Переключаетль с двумя положениями: **включено** и **выключено**. Нужен, чтобы переключать
состояния — мгновенно, сразу после того, как пользователь меняет положение переключателя.

- [Список свойств](#свойства)
- [Пример использования](#пример)

## Свойства


<!-- props:start -->

| Свойство         | Тип                                     | По умолчанию  | Описание                                     |
| ---------------- | --------------------------------------- | ------------- | -------------------------------------------- |
| `label `         |  `string`                               |   -           | Текст рядом с переключателем    |
| `size`           | `'s', 'm', 'l'`                         | `'m'`         | Размер переключателя                              |
| `checked`        | `boolean`                               | -             | Ставит переключатель в положение **включен**             |
| `disabled`       | `boolean`                               | -             | Делает переключатель неактивным                 |
| `required`       | `boolean`                               | -             | Такой переключатель обязательно нужно выбрать       |
| `readOnly`       | `boolean`                               | -             | Переключатель, который нельзя поменять  |
| `autoFocus`      | `boolean`                               | -             | Переносит фокус на этот переключатель |
| `tabIndex`       | `number`                                | -             | орядковый номер при переходе между элементами интерфейса по клавише **tab**  |
| `step `          | `number \ string`                       | -             |  |
| `name    `       | `string`                                | -             | Имя элемента в DOM |
| `onChange`       | `string`                               | -             | Действие при смене состояния переключателя  |
| `onFocus`        | `React.FocusEventHandler<HTMLInputElement>`| -             | Действие по фокусу   |
| `onBlur`         | `React.FocusEventHandler<HTMLInputElement>` | -             | Действие при потере фокуса   |
| `inputRef`       | `React.FocusEventHandler<HTMLInputElement>`| -          | Ссылка на корневой DOM-элемент этого компонента |

<!-- props:end -->

## Пример

```ts
// src/App.ts
import React from 'react';
import { Switch } from '@gpn-design/uikit/Switch';

const App = () => {
  return <Switch label="Меня можно включать и выключать" />;
};
```
