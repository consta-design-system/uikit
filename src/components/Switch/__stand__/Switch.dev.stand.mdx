## Обзор

Компонент `Switch` представляет собой переключатель, который позволяет пользователю выбирать между двумя состояниями: включено и выключено. Используется в формах и интерфейсах для управления настройками или параметрами. Поддерживает настройку размера, внешнего вида, выравнивания текста и поведения.

## Импорт

```tsx
import { Switch } from '@consta/uikit/Switch';
```

## Содержимое

### Текст

Свойство `label` задаёт текст, отображаемый рядом с переключателем. Используется для описания функции переключателя. Если текст не указан, переключатель отображается без подписи.

## Внешний вид

### Размер

Свойство `size` определяет размер переключателя. Доступные значения: `'xs'`, `'s'`, `'m'`, `'l'`. По умолчанию используется `'m'`.


</MdxTabs>

### Стиль

Свойство `view` задаёт внешний вид переключателя. Доступны два варианта:

- `'primary'` — стандартный стиль (по умолчанию).
- `'ghost'` — минималистичный стиль.

<MdxTabs>

<SwitchExampleView />

```tsx

<Switch view="ghost" checked />
```

</MdxTabs>

## Поведение

### Состояние

- `checked`: Устанавливает переключатель во включённое состояние, если задано значение `true`.
- `disabled`: Делает переключатель неактивным, отключая взаимодействие.
- `required`: Отмечает переключатель как обязательный для выбора.

### Выравнивание

Свойство `align` управляет выравниванием текста относительно переключателя. Доступные значения:

- `'center'` — текст выравнивается по центру (по умолчанию).
- `'top'` — текст выравнивается по верхнему краю.

Подходит для многострочных текстов.

```tsx
<Switch
  align="center"
  label="Длинный текст переключателя с выравниванием по центру, записанный в несколько строк"
/>
<Switch
  align="top"
  label="Длинный текст переключателя с выравниванием по верху, записанный в несколько строк"
/>
```

### Фокус и взаимодействие

- `autoFocus`: Автоматически устанавливает фокус на переключатель при загрузке страницы.
- `onChange`: колбэк-функция, вызываемая при смене состояния переключателя.
- `onFocus`: колбэк-функция, вызываемая при получении фокуса.
- `onBlur`: колбэк-функция, вызываемая при потере фокуса.

Эти свойства позволяют настраивать интерактивное поведение компонента.

## Пример использования

```tsx
const App = () => {
  return <Switch label="Меня можно включать и выключать" />;
};
```
