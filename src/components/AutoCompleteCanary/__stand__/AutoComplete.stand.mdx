```tsx
import { AutoComplete } from '@consta/uikit/AutoCompleteCanary';
```

Пользователь начинает писать текст в поле, компонент ищет подсказку, в которой
есть набранные символы и показывает в выпадающем списке. Можно выбрать
одну из подсказок или написать значение вручную.

Этот компонент выглядит как
[TextField](##LIBS.LIB.STAND/lib:uikit/stand:components-textfield-stable), найденные значения показываются в выпадающем списке, похожем на
[Select](##LIBS.LIB.STAND/lib:uikit/stand:components-select-stable).
