# ThemeToggler

Переключатель тем, с помощью которого можно выбрать одну из двух или более переданных тем.

## Свойства

<!-- props:start -->

| Свойство                     | Тип и значения                 | По умолчанию   | Описание                                                                                                                                                 |
| ---------------------------- | ------------------------------ | -------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------- |
| [`size?`](#размер)           | `'xs', 's', 'm', 'l'`          | `'m'`          | Размер переключателя                                                                                                                                     |
| [`className?`](#внешний-вид) | `string`                       | -              | Дополнительный CSS-класс для контейнера - в варианте с двумя темами или для кнопки, окрывающей контекстное меню - в варианте для большего количества тем |
| `direction?`                 | `Direction`                    | —              | см. [Popover](?path=/docs/components-popover--popover-positioned-by-coords-story) (используется только при передачи в компонент более двух тем)          |
| `possibleDirections?`        | `Direction[]`                  | —              | см. [Popover](?path=/docs/components-popover--popover-positioned-by-coords-story) (используется только при передачи в компонент более двух тем)          |
| `themes`                     | `T[]`                          | -              | Массив тем для выбора                                                                                                                                    |
| [`value`](#состояния)        | `ThemePreset`                  | -              | Текущая тема                                                                                                                                             |
| [`setValue`](#состояния)     | `(arg: ThemePreset) => void;`  | -              | Функция для изменения темы                                                                                                                               |
| `getItemKey?`                | `(arg: T) => string OR number` | `getItemLabel` | Функция для получения ключа, уникального для каждой переданной темы. Если опустить это свойство, то вместо этой функции будет использована getItemLabel  |
| `getItemLabel`               | `(arg: T) => string`           | -              | Функция для получения названия темы, который будет отображаться в контекстом меню (отображается только в варианте с тремя и более темами)                |
| `getItemIcon`                | `(arg: T) => FC<IconProps>`    | -              | Функция для получения иконки темы в формате компонента                                                                                                   |

<!-- props:end -->

## Пример

```ts
import React, { useState } from 'react';
import { ThemeToggler } from '@consta/uikit/ThemeToggler';
import { presetGpnDark, presetGpnDefault } from '@consta/uikit/Theme';
import { IconSun } from '@consta/uikit/IconSun';
import { IconMoon } from '@consta/uikit/IconMoon';

const themes = [
  {
    theme: presetGpnDefault,
    label: 'Default',
    icon: IconSun,
  },
  {
    theme: presetGpnDark,
    label: 'Dark',
    icon: IconMoon,
  },
];

const Examle = () => {
  const [value, setValue] = useState(themes[0].theme);
  const getThemeIconDefault = (theme) => theme.icon;

  return (
    <ThemeToggler
      size="m"
      themes={themes}
      value={value}
      setValue={setValue}
      getItemIcon={getThemeIconDefault}
    />
  );
};
```
