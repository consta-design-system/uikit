# ThemeToggler

Переключатель тем, с помощью которого можно выбрать одну из двух или трех переданных тем.

## Свойства

<!-- props:start -->

| Свойство                                | Тип и значения                | По умолчанию | Описание                                                                                                                                  |
| --------------------------------------- | ----------------------------- | ------------ | ----------------------------------------------------------------------------------------------------------------------------------------- |
| [`size?`](#размер)                      | `'xs', 's', 'm', 'l'`         | `'m'`        | Размер переключателя                                                                                                                      |
| [`className?`](#внешний-вид)            | `string`                      | -            | Дополнительный CSS-класс для контейнера - в варианте с двумя темами или для кнопки, окрывающей контекстное меню - в варианте для трех тем |
| [`contextMenuClassName?`](#внешний-вид) | `string`                      | -            | Дополнительный CSS-класс для контекстного меню (только в варианте с тремя темами)                                                         |
| `themes`                                | `T[]`                         | -            | Массив тем для выбора (пока можно передать только 2 или только 3 темы)                                                                    |
| [`value`](#состояния)                   | `ThemePreset`                 | -            | Текущая тема                                                                                                                              |
| [`setValue`](#состояния)                | `(arg: ThemePreset) => void;` | -            | Функция для изменения темы                                                                                                                |
| `getThemeLabel?`                        | `(arg: T) => string`          | -            | Функция для получения названия темы, который будет отображаться в контекстом меню (нужен только в варианте с тремя темами)                |
| `getThemeValue`                         | `(arg: T) => ThemePreset`     | -            | Функция для получения темы в формате ThemePreset                                                                                          |
| `getThemeIcon`                          | `(arg: T) => FC<IconProps>`   | -            | Функция для получения иконки темы в формате компонента                                                                                    |

<!-- props:end -->

## Пример

```ts
import React, { useState } from 'react';
import { ThemeToggler } from '@consta/uikit/ThemeToggler';
import { presetGpnDark, presetGpnDefault } from '@consta/uikit/Theme';
import { IconSun } from '@consta/uikit/IconSun';
import { IconMoon } from '@consta/uikit/IconMoon';

const themes = [
  {
    theme: presetGpnDefault,
    label: 'Default',
    icon: IconSun,
  },
  {
    theme: presetGpnDark,
    label: 'Dark',
    icon: IconMoon,
  },
];

const Examle = () => {
  const [value, setValue] = useState(themes[0].theme);
  const getThemeValueDefault = (theme) => theme.theme;
  const getThemeIconDefault = (theme) => theme.icon;

  return (
    <ThemeToggler
      size="m"
      themes={themes}
      value={value}
      setValue={setValue}
      getThemeValue={getThemeValueDefault}
      getThemeIcon={getThemeIconDefault}
    />
  );
};
```
