> **Прекращение поддержки**
>
> В следующем мажерном релизе компонент перестанет поддерживаться, рекомендуем уже сейчас переходит на новую библиотеку для создания таблиц [`@consta/table`](##LIBS.LIB/lib:table)

```tsx
import { Table } from '@consta/uikit/Table';
```

Таблица — компонент для отображения данных, с возможностью сортировки и фильтрации.

> **Другие варианты для создания таблиц**
>
> - [agGridAdapter](https://ag-grid-adapter.consta.design/) — библиотека ag-grid-react, стилизованная для Consta.
> - [rcTableAdapter](https://rc-table-adapter.consta.design/) — библиотека rc-table, стилизованная для Consta.
