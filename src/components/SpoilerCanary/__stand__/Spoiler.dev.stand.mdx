## Содержимое

### Тип

Компонент отображает контент в зависимости от флага `open`.

```tsx
const SpoilerExampleOpen = () => {
  return (
    <>
      <Spoiler />
      <Spoiler open />
    </>
  );
};
```

<SpoilerExampleOpen />

Есть два вида модификации компонента: с нижним подчеркиванием и иконкой. По умолчанию с нижним подчеркиванием.

```tsx
const SpoilerExampleMode = () => {
  return (
    <>
      <Spoiler />
      <Spoiler moreIcon={IconArrowDown} />
    </>
  );
};
```

<SpoilerExampleMode />

### Лейбл

Лейбл к компоненту можно указать через свойства `moreLabel` и `lessLabel`, для раскрытия и скрытия содержимого соответственно.

```tsx
const SpoilerExampleLabel = () => {
  const [isOpen, setIsOpen] = useFlag();

  return (
    <Spoiler
      open={isOpen}
      lessLabel="Скрыть"
      moreLabel="Показать"
      onClick={setIsOpen.toggle}
    />
  );
};
```

<SpoilerExampleLabel />

### Иконки

Для определения иконок используйте свойства `moreIcon` и `lessIcon`, для раскрытия и скрытия содержимого соответственно.

```tsx
const SpoilerExampleIcon = () => {
  const [isOpen, setIsOpen] = useFlag();

  return (
    <Spoiler
      open={isOpen}
      lessIcon={IconRemove}
      moreIcon={IconQuestion}
      onClick={setIsOpen.toggle}
    />
  );
};
```

<SpoilerExampleIcon />

## Размер

За размер отвечает свойство `size`. Варианты: `'l'`, `'m'`, `'s'`, `'xs'`, по умолчанию: `'m'`.

```tsx
export const SpoilerExampleSize = () => {
  return (
    <>
      <Spoiler size="l" />
      <Spoiler size="m" />
      <Spoiler size="s" />
      <Spoiler size="xs" />
    </>
  );
};
```

## Свойства

| Свойство               | Тип                         | По умолчанию        | Описание                     |
| ---------------------- | --------------------------- | ------------------- | ---------------------------- |
| [`open?`](#тип)        | `'more'`, `'less'`          | `'more'`            | Тип контента скрыт / раскрыт |
| [`lessLabel?`](#лейбл) | `string`                    | `'Показать меньше'` | Текст для раскрытия          |
| [`moreLabel?`](#лейбл) | `string`                    | `'Показать больше'` | Текст для скрытия            |
| [`lessIcon?`](#иконки) | `IconComponent`             | `'Показать больше'` | Иконка для раскрытия         |
| [`moreIcon?`](#иконки) | `IconComponent`             | `'Показать больше'` | Иконка для скрытия           |
| [`size?`](#размер)     | `'l'`, `'m'`, `'s'`, `'xs'` | `'m'`               | Размер компонента            |
| `as`                   | `React.ElementType`         | `div`               | HTML-тег для кнопки          |
| `className?`           | `string`                    | -                   | Дополнительные CSS-классы    |
