| Свойство     | Тип                       | По умолчанию | Описание                 |
| ------------ | ------------------------- | ------------ | ------------------------ |
| `className?` | `string`                  | -            | Дополнительный CSS-класс |

## Композиция

Композировать блоки, чтобы они визуально напоминали ваш компонент, можно несколькими способами:

1. По возможности старайтесь использовать те же классы, что и для лэйаута обычного компонента. Это позволит избежать дублирования и сделает вид компонента и его скелетона максимально похожими.
   <SkeletonExampleAvatar />

```tsx
<div className="main">
  <SkeletonCircle className="avatar" size={32} />
  <SkeletonText className="text" rows={2} fontSize="xs" lineHeight="s" />
</div>
```

```css
.main {
  display: flex;
  width: 150px;
}

.avatar {
  flex-shrink: 0;
}

.text {
  margin-left: var(--space-s);
}
```

2. Если используете компоненты для лэйаута (например, Grid), то кладите внутрь скелетоны, а после загрузки меняйте их на реальные компоненты.
   <SkeletonExampleImageAndText />

```tsx
<Grid gap="m" style={{ width: 200 }}>
  <SkeletonBrick height={200} />
  <SkeletonText rows={4} />
</Grid>
```

3. Если реиспользовать классы или компоненты лэйаута не удаётся, можно написать отдельные стили для обёртки скелетона.
