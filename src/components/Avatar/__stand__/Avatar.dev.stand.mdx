## Обзор

Компонент `Avatar` отображает аватар пользователя в виде изображения или инициалов.

Типичные сценарии использования:

- Отображение фотографии пользователя в профиле.
- Показ инициалов, если изображение отсутствует.
- Индикация пользователей в таблицах или карточках.

## Импорт

```tsx
import { Avatar } from '@consta/uikit/Avatar';
```

## Свойства

```tsx
type AvatarPropSize = 'xs' | 's' | 'm' | 'l';
type AvatarPropForm = 'default' | 'round' | 'brick';
```

| Свойство                            | Тип                               | По умолчанию | Описание                                    |
| ----------------------------------- | --------------------------------- | ------------ | ------------------------------------------- |
| [`url?`](#изображение-и-имя)        | `string`                          | -            | Ссылка на изображение аватара               |
| [`name?`](#изображение-и-имя)       | `string`                          | -            | Имя пользователя для формирования инициалов |
| [`size?`](#размер)                  | `'xs' \| 's' \| 'm' \| 'l'`       | `'m'`        | Размер аватара                              |
| [`form?`](#форма)                   | `'default' \| 'round' \| 'brick'` | `'round'`    | Форма аватара                               |
| [`monochrome?`](#монохромный-режим) | `boolean`                         | -            | Включает монохромный режим отображения      |
| `className?`                        | `string`                          | -            | Дополнительный CSS-класс                    |
| `as?`                               | `React.ElementType`               | -            | HTML-тег для рендера компонента             |
| `ref?`                              | `React.Ref<HTMLDivElement>`       | -            | Ссылка на корневой DOM-элемент              |

## Содержимое

### Изображение и имя

Свойство `url` задаёт ссылку на изображение аватара. Если изображение отсутствует, свойство `name` формирует инициалы пользователя. Цвет фона при использовании `name` выбирается автоматически. Если не указаны ни `url`, ни `name`, отображается пустой аватар.

<MdxTabs>

<AvatarExampleImage />

```tsx
import React from 'react';
import { Avatar } from '@consta/uikit/Avatar';

const AvatarExampleImage = () => {
  return (
    <>
      <Avatar url="/static/media/avatar.png" name="Вадим Матвеев" />
      <Avatar name="Вадим Матвеев" />
      <Avatar name="Куст Геннадий Альбертович" />
      <Avatar />
    </>
  );
};
```

</MdxTabs>

### Монохромный режим

Свойство `monochrome` включает монохромный режим, убирая цветное оформление аватара или изображения.

<AvatarExampleMonochrome />

```tsx
import React from 'react';

export const AvatarExampleMonochrome = () => (
  <>
    <Avatar name="Вадим Матвеев" monochrome />
    <Avatar url={avatar} name="Вадим Матвеев" monochrome />
  </>
);
```

## Внешний вид

### Размер

Свойство `size` задаёт размер аватара. Возможные значения: `xs`, `s`, `m` (по умолчанию), `l`.

<MdxTabs>

<AvatarExampleSize />

```tsx
import React from 'react';
import { Avatar } from '@consta/uikit/Avatar';

const AvatarExampleSize = () => {
  return (
    <>
      <Avatar size="xs" name="Вадим Матвеев" />
      <Avatar size="s" name="Вадим Матвеев" />
      <Avatar size="m" name="Вадим Матвеев" />
      <Avatar size="l" name="Вадим Матвеев" />
    </>
  );
};
```

</MdxTabs>

### Форма

Свойство `form` определяет форму аватара: `round` (по умолчанию), `default` (прямоугольная), `brick` (прямоугольная с острыми углами).

<MdxTabs>

<AvatarExampleForm />

```tsx
import React from 'react';
import { Avatar } from '@consta/uikit/Avatar';

const AvatarExampleForm = () => {
  return (
    <>
      <Avatar form="round" name="Вадим Матвеев" />
      <Avatar form="default" name="Вадим Матвеев" />
      <Avatar form="brick" name="Вадим Матвеев" />
    </>
  );
};
```

</MdxTabs>

## Пример использования

Пример показывает, как отобразить аватар с изображением.

```tsx
import React from 'react';
import { Avatar } from '@consta/uikit/Avatar';

const App = () => {
  return (
    <Avatar url="https://pbs.twimg.com/profile_images/1150453787603156992/DoiKLDMY_400x400.png" />
  );
};
```
