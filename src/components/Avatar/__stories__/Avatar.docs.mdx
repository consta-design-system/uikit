# Avatar

Аватар — графическое изображение пользователя, фотография или рисунок.

Он нужен, чтобы быстро узнавать пользователя. Например, в чате или в ветке комментариев.
А ещё — чтобы пользователь мог узнать сам себя. Например, проверить, что вошёл в
аккаунт под нужным логином.

## Аватар

Если у вас есть фотография пользователя, укажите ссылку на неё с помощью
`url`.

Если фотографии нет, аватар сформируется из имени пользователя (свойство `name`).
В этом случае цвет аватара выбирается автоматически, повлиять на него не получится.

Если вы не укажете ни то, ни другое, аватар всё равно появится.

```ts
<Avatar url="https://pbs.twimg.com/profile_images/1150453787603156992/DoiKLDMY_400x400.png" name="Вадим Матвеев" />
<Avatar name="Геннадий Куст" />
<Avatar name="Куст Геннадий Альбертович" />
<Avatar />
```

## Размер

За размер отвечает свойство `size`.

```ts
<Avatar size='xs' name='...'/>
<Avatar size='s' name='...'/>
<Avatar size='m' name='...'/>
```

<AvatarExampleSize />

## Форма

За форму отвечает свойство `form`. По умолчанию аватар **круглый**.

```ts
<Avatar form='round' name='...' />
<Avatar form='default' name='...' />
<Avatar form='brick' name='...' />
```

## Свойства

| Свойство          | Тип                            | По умолчанию | Описание                                                         |
| ----------------- | ------------------------------ | ------------ | ---------------------------------------------------------------- |
| [`url`](#аватар)  | `string`                       | -            | Ссылка на изображение (аватар пользователя)                      |
| [`name`](#аватар) | `string`                       | -            | Имя пользователя                                                 |
| [`size`](#размер) | `'xs', 's' , 'm'`              | 'm'          | Размер аватара                                                   |
| [`form`](#форма)  | `'default' , 'round', 'brick'` | `'round'`    | Форма аватара                                                    |
| `children`        | `React.ReactNode`              | -            | Дочерние компоненты                                              |
| `className`       | `string`                       | -            | Дополнительный CSS-класс                                         |
| `as`              | `React.ElementType`            | -            | HTML-тег, в виде которого этот компонент будет представлен в DOM |
| `ref`             | `React.Ref<HTMLDivElement>`    | -            | Ссылка на корневой DOM-элемент этого компонента                  |

## Пример

```ts
// src/App.ts
import React from 'react';
import { Avatar } from '@consta/uikit/Avatar';

const App = () => {
  return (
    <Avatar url="https://pbs.twimg.com/profile_images/1150453787603156992/DoiKLDMY_400x400.png" />
  );
};
```
