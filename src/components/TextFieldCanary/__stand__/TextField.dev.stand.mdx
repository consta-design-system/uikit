`TextField` – поле для ввода текста и чисел из одной или нескольких строк. Используется в формах.

## Импорт

Вы можете импортировать один общий компонент:

```tsx
import { TextField } from '@consta/uikit/TextFieldCanary';
```

Или каждый тип по отдельности:

```tsx
import {
  TextFieldTypeNumber,
  TextFieldTypePassword,
  TextFieldTypeText,
  TextFieldTypeTextArea,
  TextFieldTypeTextArray,
} from '@consta/uikit/TextFieldCanary';
```

## Тип поля

Свойство `type` определяет тип поля и тег, который будет для него использоваться:

- [`text`](#поле-в-одну-строку). Поле для ввода текста в одну строку.
- [`textarea`](#поле-в-несколько-строк). Текстовое поле из нескольких строк, можно регулировать [количество строк и колонок](#строки-и-колонки).
- [`number`](#поле-для-ввода-чисел). Поле для ввода чисел, значение можно менять с помощью кнопок-стрелочек.
- [`password`](#поле-для-ввода-пароля). Маскированное поле для ввода пароля.
- [`textarray`](#поле-для-ввода-массива-строк). Поле ввода массива строк.

Здесь описаны основные типы, но вы можете использовать любые `<input type="*" />`, например, `<input type="email">`.

### Шаг, набор значений и диапазон

Для поля ввода с `type=number` вы можете указать шаг, целое число, на которое можно менять значение в поле с помощью кнопок-стрелочек, или набор значений в виде конечных точек. За это отвечает свойство `step`.

Ограничить диапазон значений, которые меняются по стрелочкам, можно с помощью свойств `min` (минимальное значение) и `max` (максимальное значение).

Отобразить стрелочки можно с помощью свойства `incrementButtons`.

### Парсинг значения при вставке

Используйте, когда при вставке строки ее нужно сразу разбивать на подстроки и добавлять к `value`. Так, например, когда есть список электронных почт, который при вставке необходимо преобразовать в массив строк и задать `value`.

```json
// Список через пробел
user@company.ru user2@company.ru user3@company.ru user4@company.ru

// Список через запятую
user@company.ru, user2@company.ru, user3@company.ru, user4@company.ru

// Список через перенос строки
user@company.ru
user2@company.ru
user3@company.ru
user4@company.ru
```

<MdxTabs>

```tsx
export const TextFieldExampleTextArrayPreParsing = () => {
  const [value, setValue] = useState<string[] | null>(null);

  const inputRef = useRef<HTMLInputElement>(null);

  const handleInputClear = useCallback(() => {
    if (inputRef.current) {
      inputRef.current.value = '';
    }
  }, []);

  const handleChange = useCallback((value: string[] | null) => {
    setValue(value);
    handleInputClear();
  }, []);

  const pasteHandle = useCallback((e: React.ClipboardEvent) => {
    const data = e.clipboardData.getData('text/plain');
    if (!data) {
      return;
    }

    const array = [data.split(','), data.split(' '), data.split('\n')];

    let maxLength = 0;
    let validIndex = 0;

    for (let index = 0; index < array.length; index++) {
      if (array[index].length > maxLength) {
        maxLength = array[index].length;
        validIndex = index;
      }
    }

    const validData = array[validIndex]
      .map((item) => item.trim())
      .filter((item) => !!item);

    if (validData.length > 1) {
      e.preventDefault();
      handleInputClear();
      setValue((state) => [...(state || []), ...validData]);
    }
  }, []);

  return (
    <Example col={1}>
      <TextField
        value={value}
        onChange={handleChange}
        placeholder="Вставьте вышеуказанный текст"
        onPaste={pasteHandle}
        type="textarray"
        inputRef={inputRef}
        withClearButton
      />
    </Example>
  );
};
```

## Маскирование

Чтобы использовать маски вместе с компонентом `TextField`, импортируйте хук [useIMask](https://github.com/uNmAnNeR/imaskjs/tree/master/packages/react-imask#using-hook).
Как составить собственные маски, читайте в документации [imask](https://imask.js.org/guide.html).

## Как работает useIMask

Значение `TextField` или любого другого `<input/>` меняется через хук со связкой с полем через `ref`. Чтобы указать значение в момент первого рендера, используйте `defaultValue`. Чтобы изменить поле программно, используйте `setValue` из хука.

<MdxTabs>

```tsx
import { ReactMaskOpts, useIMask } from 'react-imask';
import React, { memo } from 'react';
import { IconRevert } from '@consta/icons/IconRevert';
import { Text } from '@consta/uikit/Text';
import { TextField } from '@consta/uikit/TextFieldCanary';
import { FieldGroup } from '@consta/uikit/FieldGroup';
import { FieldLabel } from '@consta/uikit/FieldLabel';
import { Button } from '@consta/uikit/Button';
import { FieldWrapper } from '@consta/uikit/FieldComponents';

const defaultValue = '9876543210';

export const TextFieldExampleMaskSetValue = () => {
  const { ref, setValue } = useIMask<HTMLInputElement, ReactMaskOpts>({
    mask: '+{7}(000)000-00-00',
  });

  return (
    <FieldWrapper label="Телефон">
      <FieldGroup>
        <TextField
          placeholder="+7(999)999-99-99"
          inputRef={ref}
          defaultValue={defaultValue}
          withClearButton
          onClear={() => setValue('')}
        />
        <Button
          iconLeft={IconRevert}
          onClick={() => setValue(defaultValue)}
          title="Сбросить"
        />
      </FieldGroup>
    </FieldWrapper>
  );
};
```

<TextFieldExampleMaskSetValue />

</MdxTabs>

  const [error, setError] = useState<string>('');
  const { ref, setValue: setInputValue } = useIMask<
    HTMLInputElement,
    ReactMaskOpts
  >({ mask });

  const onChange = useCallback((value: string[] | null) => {
    if (
      (value?.length && value[value.length - 1].length === mask.length) ||
      value?.length === 0
    ) {
      setValue(value);
      setError('');
      setInputValue('');
    } else {
      setError(`Введите серийный номер в формате "${mask}"`);
    }
  }, []);

  const onInputChange = useCallback(() => setError(''), []);

  const onClear = useCallback(() => {
    setValue(null);
    setInputValue('');
  }, []);

  const status = error ? 'alert' : undefined;

  return (
    <FieldWrapper label="Серийные номера" caption={error} status={status}>
      <TextField
        type="textarray"
        inputRef={ref}
        value={value}
        onChange={onChange}
        onInputChange={onInputChange}
        status={status}
        placeholder={mask}
        withClearButton
        onClear={onClear}
      />
    </FieldWrapper>
  );
};
