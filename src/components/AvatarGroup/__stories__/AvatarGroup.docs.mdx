## Аватары

Для того чтоб использовать компонент передадите список в поле `items`.

Для того чтобы указать имя укажите в элементе списка поле `name` или определите свойство
`getItemName`.

Для того чтобы указать изображение укажите в элементе списка поле `url` или определите свойство
`getItemUrl`.

За количество отображаемых аватаров отвечает свойство `visibleCount`, изначально имеет значение `4`.

```ts
<AvatarGroup items={avatarGroupItems}  />
<AvatarGroup items={avatarGroupItems} visibleCount={6} />
<AvatarGroup items={avatarGroupItems.slice(0, 3)} />
```

## Форма

За форму отвечает свойство `form`. По умолчанию аватар **круглый**.

```ts
<AvatarGroup form="round" items={avatarGroupItems} />
<AvatarGroup form="default" items={avatarGroupItems} />
<AvatarGroup form="brick" items={avatarGroupItems} />
```

## Свойства

```tsx

export type AvatarGroupDegaultItem = {

  name?: string;
};
```

| Свойство                    | Тип                                                                         | По умолчанию | Описание                                                                       |
| --------------------------- | --------------------------------------------------------------------------- | ------------ | ------------------------------------------------------------------------------ |
| [`items?`](#аватары)        | `ITEM[]`                                                                    | -            | Список элементов                                                               |
| [`visibleCount?`](#аватары) | `number`                                                                    | `4`          | количество отображаемых элементов                                              |
| [`size?`](#размер)          | `'xs', 's' , 'm', 'l'`                                                      | `'m'`        | Размер аватара                                                                 |
| [`form?`](#форма)           | `'default' , 'round', 'brick'`                                              | `'round'`    | Форма аватара                                                                  |
| `getItemUrl?`               | `(item: ITEM) => string`<code>&#124;</code>`undefined`                      | -            | Результат функции используется в качестве изображения.                         |
| `getItemName?`              | `(item: ITEM) => string`<code>&#124;</code>`undefined`                      | -            | Результат функции используется в качестве имени.                               |

## Пример

```ts
// src/App.ts
import React from 'react';
import { AvatarGroup } from '@consta/uikit/AvatarGroup';

const App = () => {
  const avatarGroupItems: AvatarGroupDegaultItem[] = [
    {
      label: 'Вадим Матвеев',
      image: '',
    },
    {
      label: 'Лидия Индриксон',
      image: '',
    },
    {
      label: 'Геннадий Морозов',
      image: '',
    },
  ];

  return (
    <AvatarGroup
      items={avatarGroupItems}
      size="m"
      getItemName={(item) => item.label}
      getItemUrl={(item) => item.image}
    />
  );
};
```
