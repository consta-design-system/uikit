Табы переключают вкладки на странице. Активный таб выделен цветом и помечен специальным маркером — горизонтальной или вертикальной линией.

В табах можно использовать иконки или бейджики. Бейджи рекомендуется использовать в модификации isMinified, размера S для всех размеров табов.

Всегда выравниваются по левому краю страницы. 
По ширине контейнера растягивается только последний таб 
в компоненте. Все остальные — прибиты к левому краю страницы 
или контентной области.

Не используйте для верхнеуровневой навигации по интерфейсу.
Используйте для второстепенной навигации по странице.
