# Breadcrumbs

Breadcrumbs - компонент для отображения пути к текущей странице

## Список свойств

```ts
type Page = {
  icon?: React.FC<IconProps>;
  link: string;
  label: string;
};

```

| Свойство              | Тип или варианты значения                                             | По умолчанию   | Описание                                           |
| --------------------- | --------------------------------------------------------------------- | -------------- | -------------------------------------------------- |
| pages                 | `Page[]`                                                              | -              | Список страниц                                     |
| size?                 | `'xs', s', 'm', 'l'`                                                  | 'm'            | Размер хлебных крошек                              |
| maxCount?             | `number`                                                              | -              | Максимальное количество отображаемых элементов     |
| onClick`              | `(page: Page, e: React.MouseEvent) => void`                           | -              | Хэндлер, который отработает по клику на брэкрамб   |
| onlyIconRoot?         | `boolean`                                                             | false          | Показывать только иконку первого брэдкрамба        |
| className?            | `string`                                                              | -              | Дополнительный класс                               |

## Пример использования

```ts

// src/App.ts
import React from 'react';
import { Breadcrumbs } from '@consta/uikit/Breadcrumbs';

const App = () => {

  const pages = {
  {
      icon: IconUser,
      label: 'Home',
      link: 'https://url.com',
    },
    {
      label: 'Page1',
      link: 'https://url.com/page-1',
    },
    {
      label: 'Page2',
      link: 'https://url.com/page-2',
    },
    {
      label: 'Page3',
      link: 'https://url.com/page-3',
    },
  }

  return (
    <Breadcrumbs
       className="myClassName"
       pages={pages}
       size={'m'}
       maxCount={3}
       onlyIconRoot={true}
       onClick={(page, e) => {
          e.preventDefault();
          console.log(page.link, e);
       }}
     />
  );
};


```
