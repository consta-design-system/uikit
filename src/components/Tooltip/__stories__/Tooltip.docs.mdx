Тултип — компонент для отображения всплывающего окна возле элемента или определенной точки на экране.

## Тема тултипа

Содержимое тултипа использует инвертированную тему: в светлой теме тултип будет темным, в темной — светлым.

## Размер

За размер тултипа отвечает свойство `size`. Варианты: `s`, `m`, `l`. В зависимости от выбранного размера изменятеся максимальная ширина тултипа.

## Статус

Иногда может быть полезно дополнительно выделить статус сообщения в Тултипе, например для сообщения об ошибки в валидации. Сделать это можно с помощью свойства `status` со значениями: `normal`, `success`, `warning` и `alert`.

## Позиционирование тултипа

Для позиционирования тултипа используется компонент [Popover](?path=/docs/components-popover--popover-positioned-by-coords-story) и его способы позиционирования.

### По элементу-якорю

<TooltipPositionedByAnchorStory />

### Пример

```ts
<Button label="Открыть тултип" type="button" ref={buttonRef} />
<Tooltip size="s" anchorRef={buttonRef}>
  Контент тултипа
</Tooltip>
```

### По координатам

<TooltipPositionedByCoordsStory />

### Пример

```ts
<Tooltip size="s" position={{ x: 100, y: 200 }}>
  Контент тултипа
</Tooltip>
```
