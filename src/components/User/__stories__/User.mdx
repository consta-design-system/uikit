## Аватарка

Изображение человека (обычно фотография). Подгружается по ссылке из `avatarUrl`.

Вы можете убрать имя и оставить только аватарку (например, если в интерфейсе нет места)
с помощью свойства `onlyAvatar`.

## Тексты

За имя человека отвечает `name`. Здесь может быть ник, логин, имя и фамилия.

Вы можете добавить что-то важное в `info` —  например, должность, местонахождение
или статус человека. Этот текст появится под именем.


## Статус

`status` показывает, как можно связаться с человеком.

##  Внешний вид

`view = 'ghost'` добавит компоненту обводку.

 ### Размер

У этого блока два размера — 's' и 'm'.

Чтобы растянуть блок на всю ширину, добавьте `width = full`.

### Подробная информация

Добавьте стрелочку `withArrow`, чтобы показать, что по клику на аватарку или весь блок
можно посмотреть подробную информацию о человеке.


## Свойства

<!-- props:start -->

| Свойство         | Тип и значения  | По умолчанию  | Описание                                     |
| ---------------- | ----------------| ------------- | -------------------------------------------- |
| [`avatarUrl`](#аватарка) | `string` | - | Ссылка на аватарку|
| [`onlyAvatar`](#аватарка) | `boolean` | - | Показывать только аватарку, без имени |
| [`name`](#тексты)| `string`| - | Имя человека (имя и фамилия) |
| [`info`](#тексты) | `string` | `-`  | Дополнительная информация о человеке |
| [`view`](#внешний-вид) | `'ghost', 'clear'` | `'clear'`  | Внешний вид компонента |
| [`withArrow`](#подробная-информация)| `boolean`  | - | Показывать стрелочку, чтобы раскрывать подробную информацию |
| [`status`](#статус)| `'available', 'remote', 'out'`| -  | Статус пользователя |
| [`size`](#размер) | `'s', 'm'`  |`'m'` | Размер |
| [`width`](#размер) | `'full', 'default'` | `'default'` | Ширина блока |
| `className` | `string` | -  | Дополнительный CSS-класс |

<!-- props:end -->

## Пример

```ts
// src/App.ts
import React from 'react';
import { User } from '@gpn-design/uikit/User';

const App = () => {
  return <User avatarUrl="https://pbs.twimg.com/profile_images/1150453787603156992/DoiKLDMY_400x400.png" name="Винни-Пух" info="И все-все-все" />;
};
```
