## Обзор

Компонент `User` предназначен для отображения информации о пользователе, включая аватарку, имя, дополнительную информацию и статус. Он поддерживает настройку внешнего вида, размеров и интерактивных элементов, таких как стрелка для раскрытия контекстного меню.

## Импорт

```tsx
import { User } from '@consta/uikit/User';
```

## Свойства

| Свойство                                     | Тип и значения                      | По умолчанию | Описание                                                   |
| -------------------------------------------- | ----------------------------------- | ------------ | ---------------------------------------------------------- |
| [`avatarUrl?`](#аватарка)                    | `string`                            | -            | Ссылка на изображение аватарки                             |
| [`onlyAvatar?`](#аватарка)                   | `boolean`                           | -            | Показывать только аватарку, без имени и информации         |
| [`name?`](#тексты)                           | `string`                            | -            | Имя пользователя (например, имя и фамилия или ник)         |
| [`info?`](#тексты)                           | `string`                            | -            | Дополнительная информация о пользователе                   |
| [`status?`](#статус)                         | `'available', 'remote', 'out'`      | -            | Статус пользователя                                        |
| [`size?`](#размер)                           | `'s', 'm', 'l'`                     | `'m'`        | Размер компонента                                          |
| [`width?`](#размер)                          | `'full', 'default'`                 | `'default'`  | Ширина компонента                                          |
| [`view?`](#обводка)                          | `'ghost', 'clear'`                  | `'clear'`    | Стиль обводки компонента                                   |
| [`withArrow?`](#подробная-информация)        | `boolean`                           | -            | Показывать стрелку для раскрытия дополнительной информации |
| [`iconRight?`](#подробная-информация)        | `IconComponent`                     | -            | Иконка в правой части компонента                           |
| [`onIconRightClick?`](#подробная-информация) | `(e: React.SyntheticEvent) => void` | -            | Обработчик клика по иконке справа                          |
| `className?`                                 | `string`                            | -            | Дополнительный CSS-класс                                   |
| `as?`                                        | `React.ElementType`                 | -            | HTML-тег для рендеринга компонента                         |
| `ref?`                                       | `React.RefObject`                   | -            | Ссылка на DOM-элемент компонента                           |

## Содержимое

### Аватарка

Свойство `avatarUrl` задаёт ссылку на изображение аватарки. Если изображение не указано, аватарка формируется автоматически на основе имени пользователя (`name`) с цветным фоном, который выбирается системой и не подлежит изменению.

Свойство `onlyAvatar` позволяет отображать только аватарку, скрывая имя и дополнительную информацию. Это полезно для компактных интерфейсов.

```tsx
<User avatarUrl="https://i.ibb.co/K2R8Lqb/Rectangle-1496.png" name="Роберт Пласт" info="Сейсмик" />
<User avatarUrl="https://i.ibb.co/K2R8Lqb/Rectangle-1496.png" name="Роберт Пласт" info="Сейсмик" onlyAvatar />
```

### Тексты

Свойство `name` задаёт имя пользователя, которое может включать ник, логин или полное имя с фамилией. Свойство `info` используется для отображения дополнительной информации, например, должности, местоположения или статуса.

```tsx
<User avatarUrl="https://i.ibb.co/K2R8Lqb/Rectangle-1496.png" name="Гюзель Скважина" info="В отпуске" />
<User avatarUrl="https://i.ibb.co/K2R8Lqb/Rectangle-1496.png" name="Люций" info="Эксперт" onlyAvatar />
```

## Внешний вид

### Размер

Свойство `size` определяет размер компонента и принимает значения `'s'`, `'m'` или `'l'`. По умолчанию используется `'m'`.

<UserExampleSize />

Свойство `width` управляет шириной компонента: `'default'` для стандартной ширины или `'full'` для растяжения на всю доступную ширину.

```tsx
<User name="Бон Фишер" info="Старший эксперт" width="full" />
```

<UserExampleSizeWidth />

### Обводка

Свойство `view` задаёт стиль обводки компонента:

- `'ghost'` — добавляет обводку.
- `'clear'` — убирает обводку (по умолчанию).

```tsx
<User name="Бон Фишер" info="Старший эксперт" view="ghost" />
```

<UserExampleView />

### Статус

Свойство `status` отображает статус пользователя:

- `'available'` — пользователь онлайн.
- `'remote'` — пользователь работает удалённо.
- `'out'` — пользователь недоступен.

```tsx
<User name="Камилла Орлова" info="Онлайн" status="available" />
<User name="Камилла Орлова" info="Из дома" status="remote" />
<User name="Камилла Орлова" info="Недоступна" status="out" />
```

## Поведение

### Подробная информация

Свойство `withArrow` добавляет стрелку, указывающую на возможность раскрытия дополнительной информации, например, контекстного меню с пунктами вроде **Мой профиль**, **Настройки профиля** или **Выйти**.

Свойство `iconRight` позволяет добавить иконку в правую часть компонента, а `onIconRightClick` задаёт обработчик клика по этой иконке. Эти свойства работают независимо от `withArrow`.

```tsx
<User name="Бон Фишер" info="Старший эксперт" withArrow />
```
