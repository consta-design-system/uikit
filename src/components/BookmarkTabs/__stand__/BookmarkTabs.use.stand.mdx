Компонент **BookmarkTabs** используется для быстрой навигации по страницам с различным контентом без необходимости перезагрузки страницы или открытия новой.

<MdxImage src={start_example} />

# Контекст использования

Компонент обычно используется в веб-сервисах для управления большим количеством информации.

## Верхнеуровневая навигация

Используется, когда пользователю нужно быстро переключаться между основными разделами или модулями.

**Пример**. Сервис для работы с таблицами.

<MdxImage src={table} />

**Пример**. Графический редактор.

<MdxImage src={graphics} />

## Навигация в сложном интерфейсе

Подойдет, когда требуется структурировать большое количество информации и функций в рамках одной страницы. Это помогает разбить интерфейс на логически связанные части, позволяя пользователю легко переходить между ними без необходимости загружать новые страницы.

**Пример**. Многооконные интерфейсы.

<MdxImage src={mdi} />

# Лучшие практики

## Категоризация

Позвольте пользователям организовывать закладки в категории или папки для лучшей структурированности.

## Простота в использовании

Убедитесь, что пользователю будет интуитивно понятно, как управлять вкладками (добавлять, удалять, редактировать).

## Иконки

[Иконки](##LIBS.LIB.STAND/lib:icons/stand:icons-icon-stable) могут помочь пользователю быстрее определить содержимое табов. Используйте иконки, когда они действительно помогают улучшить восприятие информации.

# Похожие компоненты

- [Tabs](##LIBS.LIB.STAND/lib:uikit/stand:components-tabs-stable) — это компонент для навигации по вкладкам второго уровня.
- [ChoiceGroup](##LIBS.LIB.STAND/lib:uikit/stand:components-choicegroup-stable) — группа кнопок. Чаще используется для навигации на конкретном участке интерфейса, а также в качестве элемента управления.
