# ProgressLine

```tsx
import { ProgressLine } from '@consta/uikit/ProgressLine';
```

Индикатор загрузки или выполнения задачи в виде линии.

## Тип

По умолчанию компонент отображается с бесконечной анимацией — 
чтобы показать, что процесс идёт.

Если нужно показать процент загрузки или выполнения задачи,
передайте соответствующее значение в свойстве `value`.
В этом случае анимация выключится.

```tsx
<ProgressLine />
```

<ProgressLineExampleMode />

```tsx
<ProgressLine value={30} />
```

<ProgressLineExampleModeValue />

## Размер

Свойство `size` отвечает за размер индикатора. Доступные размеры: `s` и `m`.

```tsx
<ProgressLine size="s" />
<ProgressLine size="m" />
```

<ProgressLineExampleSize />

## Импорт

```tsx
import { ProgressLine } from '@consta/uikit/ProgressLine';
```

## Свойства

```tsx
type ProgressLinePropMode = 'determinate' | 'indeterminate';
```

| Свойство           | Тип                         | По умолчанию | Описание                                  |
| ------------------ | --------------------------- | ------------ | ----------------------------------------- |
| [`value?`](#тип)   | `number`                    | -            | Процент загрузки или выполнения           |
| [`size?`](#размер) | `'s', 'm'`                  | `'m'`        | Размер индикатора                         |
| `className?`       | `string`                    | -            | Дополнительный CSS-класс                  |
| `ref?`             | `React.Ref<HTMLDivElement>` | -            | Ссылка на корневой DOM-элемент компонента |

## Пример

```tsx
import React, { useState } from 'react';
import { ProgressLine } from '@consta/uikit/ProgressLine';

const App = () => {
  return <ProgressLine size="s" value={20} />;
};
```
