## Обзор

Компонент `ProgressLine` используется для отображения прогресса выполнения задачи или процесса. Он может работать в трёх режимах:

Компонент поддерживает настройку размера и отображение шагов с помощью функции для получения подписей.

## Импорт

```tsx
import { ProgressLine } from '@consta/uikit/ProgressLine';
```

## Свойства

| Свойство                            | Тип                         | По умолчанию | Описание                                  |
| ----------------------------------- | --------------------------- | ------------ | ----------------------------------------- |
| [`value?`](#тип-отображения)        | `number`                    | -            | Процент загрузки или номер текущего шага  |
| [`steps?`](#тип-отображения)        | `ITEM[]`                    | -            | Массив шагов                              |
| [`getItemLabel?`](#тип-отображения) | `(item: ITEM) => string`    | -            | Функция для получения подписи шага        |
| [`size?`](#размер)                  | `'s' \| 'm'`                | `'m'`        | Размер индикатора                         |
| `className?`                        | `string`                    | -            | Дополнительный CSS-класс                  |
| `ref?`                              | `React.Ref<HTMLDivElement>` | -            | Ссылка на корневой DOM-элемент компонента |

## Содержимое

### Тип отображения

Свойства `value`, `steps` и `getItemLabel` определяют, как будет отображаться прогресс.

- **Бесконечная анимация**: Если не указаны `value` и `steps`, компонент отображается с бесконечной анимацией, сигнализирующей о выполнении процесса.

<MdxTabs>

<ProgressLineExampleMode />

```tsx
<ProgressLine />
```

</MdxTabs>

- **Процент выполнения**: Если передать свойство `value` (число от 0 до 100), компонент отобразит прогресс в процентах, а анимация отключится.

<MdxTabs>

<ProgressLineExampleModeValue />

```tsx
<ProgressLine value={30} />
```

</MdxTabs>

- **Пошаговый прогресс**: Если передать массив `steps` и функцию `getItemLabel`, компонент отобразит прогресс в виде шагов. Свойство `value` указывает номер текущего шага (начинается с 0). Функция `getItemLabel` возвращает подпись для каждого шага.

<MdxTabs>

<ProgressLineExampleModeSteps />

```tsx
<ProgressLine
  value={3}
  steps={['Первый', 'Второй', 'Третий']}
  getItemLabel={(item) => item}
/>
```

</MdxTabs>

## Внешний вид

### Размер

Свойство `size` определяет размер индикатора. Доступны два значения:

- `s` — маленький размер.
- `m` — средний размер (по умолчанию).

<MdxTabs>

<ProgressLineExampleSize />

```tsx
<>
  <ProgressLine size="s" />
  <ProgressLine size="m" />
</>
```

</MdxTabs>
