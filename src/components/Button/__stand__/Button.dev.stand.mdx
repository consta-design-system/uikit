## Обзор

Компонент `Button` — это универсальная кнопка для интерфейсов. Она поддерживает различные типы, размеры, формы и иконки, а также адаптивное поведение для удобного использования в разных сценариях.

## Импорт

```tsx
import { Button } from '@consta/uikit/Button';
```

## Содержимое

### Текст на кнопке

Свойство `label` определяет текст, отображаемый на кнопке. Оно задаёт основное содержимое кнопки и помогает пользователю понять её назначение.

<MdxTabs>

<ButtonExample />

```tsx
<Button label="Сохранить" />
```

</MdxTabs>

## Внешний вид

### Типы кнопок

Свойство `view` определяет внешний вид кнопки. В дизайн-системе Consta UI Kit доступны четыре типа кнопок: `primary`, `secondary`, `ghost` и `clear`. Каждый тип предназначен для определённых сценариев использования.

#### `primary`

Основной тип кнопки, используемый для ключевых действий. Рекомендуется для акцентирования самого важного действия на странице.

<MdxTabs>

<ButtonExampleViewPrimary />

```tsx
<Button label="Войти" />
<Button label="Продолжить" iconRight={IconForward} />
<Button label="Добавить" iconRight={IconAdd} onlyIcon />
```

</MdxTabs>

#### `secondary`

Кнопка второго уровня важности, менее акцентированная, чем `primary`. Подходит для действий, которые не требуют сильного выделения.

<MdxTabs>

<ButtonExampleViewSecondary />

```tsx
<Button label="Читать далее" view="secondary" />
<Button label="Комментировать" view="secondary" iconLeft={IconChat} />
<Button
  label="Редактировать"
  view="secondary"
  iconRight={IconEdit}
  onlyIcon
/>
```

</MdxTabs>

#### `ghost`

Второстепенная кнопка для альтернативных или менее важных действий, таких как «Отмена» или «Настройки».

<MdxTabs>

<ButtonExampleViewGhost />

```tsx
<Button label="Отмена" view="ghost" />
<Button label="Настройки" view="ghost" iconLeft={IconSettings} />
<Button label="Фильтр" view="ghost" iconLeft={IconFunnel} onlyIcon />
```

</MdxTabs>

#### `clear`

Кнопка с прозрачным фоном, подходящая для незначительных действий или списков кнопок-иконок.

<MdxTabs>

<ButtonExampleViewClear />

```tsx
<Button label="Развернуть" view="clear" />
<Button label="Скопировать" view="clear" iconLeft={IconCopy} />
<Button
  label="Добавить в Избранное"
  view="clear"
  iconLeft={IconBookmarkStroked}
  onlyIcon
/>
```

</MdxTabs>

### Размер

Свойство `size` задаёт высоту кнопки. Доступны четыре размера: `xs`, `s`, `m` (по умолчанию), `l`. Ширина кнопки зависит от содержимого и отступов.

<MdxTabs>

<ButtonExampleSizeBasic />

```tsx
<Button label="Размер XS" size="xs" />
<Button label="Размер S" size="s" />
<Button label="Размер M" size="m" />
<Button label="Размер L" size="l" />
```

</MdxTabs>

#### Ширина кнопки

Свойство `width` позволяет сделать кнопку на всю ширину контейнера (`full`) или оставить стандартную ширину (`default`).

<MdxTabs>

<ButtonExampleSizeFull />

```tsx
<Button label="Отправить" width="full" />
```

</MdxTabs>

### Кнопка с иконкой

Свойства `iconLeft` и `iconRight` добавляют иконки слева или справа от текста. Они принимают компоненты иконок из [библиотеки иконок Consta](##LIBS.LIB/lib:icons). Свойство `onlyIcon` позволяет отображать только иконку без текста.

<MdxTabs>

<ButtonExampleIconBasic />

```tsx
<Button label="Назад" iconLeft={IconBackward} />
<Button label="Вперёд" iconRight={IconForward} />
```

</MdxTabs>

<MdxTabs>

<ButtonExampleIconOnly />

```tsx
<Button label="Назад" view="ghost" iconLeft={IconHamburger} onlyIcon />
<Button label="Назад" view="ghost" iconLeft={IconCalendar} onlyIcon />
<Button label="Вперёд" view="ghost" iconRight={IconFilter} onlyIcon />
```

</MdxTabs>

<MdxInformer status="success">

В панели с большим количеством кнопок иконки могут создавать визуальный шум. Используйте их только для ключевых действий.

</MdxInformer>

#### Размер иконки

Свойство `iconSize` позволяет задать размер иконки (`xs`, `s`, `m`). По умолчанию размер иконки зависит от размера кнопки:

| Размер кнопки | Размер иконки |
| ------------- | ------------- |
| `xs`          | `xs`          |
| `s`           | `xs`          |
| `m`           | `s`           |
| `l`           | `m`           |

### Форма

Свойство `form` определяет форму кнопки: `default` (скруглённые края), `brick` (квадратная), `round` (круглая), а также гибридные варианты для комбинации с полями ввода.

<MdxTabs>

<ButtonExampleFormBasic />

```tsx
<Button label="Default" />
<Button label="Brick" form="brick" />
<Button label="Round" form="round" />
```

</MdxTabs>

#### Гибридные формы

Гибридные формы (`brickRound`, `roundBrick`, `brickDefault`, `defaultBrick`) используются для интеграции кнопки с полями ввода.

<MdxTabs>

<ButtonExampleFormHybrid />

```tsx
<div style={{ display: 'flex' }}>
  <TextField
    placeholder="Электронная почта"
    form="roundClear"
    style={{ width: '100%', maxWidth: '260px' }}
  />
  <Button label="Подписаться" form="brickRound" />
</div>
```

</MdxTabs>

### Индикатор загрузки

Свойство `loading` отображает индикатор загрузки вместо содержимого кнопки, сигнализируя о выполнении процесса.

<MdxTabs>

<ButtonExampleLoading />

```tsx
<Button label="Загрузка" loading />
```

</MdxTabs>

## Поведение

Свойство `disabled` блокирует кнопку, делая её неактивной. Используйте его, если действие недоступно, например, пока пользователь не заполнил обязательные поля формы.

<MdxTabs>

<ButtonExampleViewDisabled />

```tsx
<Button label="Войти" disabled />
<Button label="Продолжить" disabled iconRight={IconForward} />
<Button label="Добавить" disabled iconRight={IconAdd} onlyIcon />
```

</MdxTabs>

Свойство `onClick` задаёт обработчик события клика по кнопке.

## Адаптивность

Если кнопка не помещается в контейнер, текст обрезается, и в конце добавляется многоточие.

<MdxTabs>

<ButtonExampleAdaptive />

```tsx
<Button label="Этот текст не помещается" />
```

</MdxTabs>

## Пример использования

```tsx
import React from 'react';
import { Button } from '@consta/uikit/Button';

const App = () => {
  return <Button label="Сохранить" />;
};
```
