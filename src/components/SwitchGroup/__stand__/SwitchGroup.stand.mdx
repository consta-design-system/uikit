Группа переключателей, объединённых одним `name`. Пользователь может выбрать несколько вариантов из списка.

Если нужно выбрать только один вариант из списка, лучше использовать [RadioGroup](##LIBS.LIB.STAND/lib:uikit/stand:components-radiogroup-stable).

Если в интерфейсе есть место, лучше использовать [ChoiceGroup](##LIBS.LIB.STAND/lib:uikit/stand:components-choicegroup-stable) с двумя кнопками.
