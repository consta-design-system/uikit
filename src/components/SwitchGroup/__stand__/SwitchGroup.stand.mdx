Группа переключателей, объединённых одним `name`. Пользователь может выбрать несколько вариантов из списка.

Если нужно выбрать только один вариант из списка, лучше использовать [RadioGroup](##LIBS.STAND|stand:uikit-components-radiogroup-stable).

Если в интерфейсе есть место, лучше использовать [ChoiceGroup](##LIBS.STAND|stand:uikit-components-choicegroup-stable) с двумя кнопками.
