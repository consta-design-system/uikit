```tsx
import { Steps } from '@consta/uikit/StepsCanary';
```

Этот компонент позволяет показывать контент пользователю в нужной последовательности — во вкладках. И, если нужно, показать, какие шаги он прошел, какие пропустил и сколько осталось.

Это удобно, если пользователю нужно заполнить длинную форму, например, пройти тестирование.
