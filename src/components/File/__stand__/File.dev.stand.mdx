## Обзор

Компонент `File` используется для отображения иконки файла с указанием его расширения, размера и состояния загрузки. Поддерживает настройку внешнего вида и отображение прогресса загрузки.

## Импорт

```tsx
import { File } from '@consta/uikit/File';
```

## Свойства

| Свойство                                           | Тип                           | По умолчанию | Описание                                                    |
| -------------------------------------------------- | ----------------------------- | ------------ | ----------------------------------------------------------- |
| [`extension?`](#расширение-файла)                  | `string`                      | -            | Расширение файла, определяющее иконку                       |
| [`size?`](#размер-иконки)                          | `'m' \| 's'`                  | `'m'`        | Размер иконки                                               |
| [`loading?`](#индикаторы-загрузки)                 | `boolean`                     | -            | Показывает, что файл загружается                            |
| [`loadingWithProgressSpin?`](#индикаторы-загрузки) | `boolean`                     | -            | Показывает индикатор прогресса загрузки                     |
| [`loadingProgress?`](#индикаторы-загрузки)         | `number`                      | -            | Прогресс загрузки в процентах (0–100)                       |
| `className?`                                       | `string`                      | -            | Дополнительный CSS-класс                                    |
| `as?`                                              | `keyof JSX.IntrinsicElements` | -            | HTML-тег, в виде которого компонент будет представлен в DOM |
| `ref?`                                             | `React.Ref<HTMLDivElement>`   | -            | Ссылка на корневой DOM-элемент компонента                   |

## Содержимое

### Расширение файла

Свойство `extension` определяет иконку файла в зависимости от его расширения. Если указанное расширение не поддерживается, отображается иконка со знаком вопроса.

**Поддерживаемые расширения**:

- `bmp`, `csv`, `avi`, `doc`, `docx`, `gif`, `exe`, `jpg`, `jpeg`, `mp3`, `mp4`, `mov`, `pdf`, `ppt`, `pptx`, `png`, `rar`, `rtf`, `tiff`, `txt`, `wav`, `zip`, `gz`, `xls`, `xlsx`, `json`

## Внешний вид

### Размер иконки

Свойство `size` задаёт размер иконки файла. Доступные значения: `'m'` (средний, по умолчанию) и `'s'` (маленький).

<MdxTabs>

<FileExampleSize />

```tsx
<>
  <File size="m" />
  <File size="s" />
</>
```

</MdxTabs>

### Индикатор загрузки

Свойства `loading`, `loadingWithProgressSpin` и `loadingProgress` управляют отображением состояния загрузки файла:

- `loading`: включает индикатор загрузки.
- `loadingWithProgressSpin`: добавляет спиннер, показывающий прогресс.
- `loadingProgress`: задаёт процент загрузки (от 0 до 100).

<MdxTabs>

<FileExampleLoading />

```tsx
<>
  <File loading />
  <File loading loadingWithProgressSpin />
  <File loading loadingWithProgressSpin loadingProgress={30} />
  <File loading loadingWithProgressSpin loadingProgress={60} />
  <File loading loadingWithProgressSpin loadingProgress={90} />
</>
```

</MdxTabs>

```tsx
import React from 'react';
import { File } from '@consta/uikit/File';

const App = () => {
  return <File extension="doc" />;
};
```
