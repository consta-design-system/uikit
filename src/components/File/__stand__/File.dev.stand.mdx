## Иконка

Укажите расширение файла с помощью свойства `extension`, и пользователь увидит
нужную иконку. Если в дизайн-системе нет изображения для какого-то расширения, на иконке будет знак вопроса.

### Доступные расширения

- `bmp`
- `csv`
- `avi`
- `doc`
- `docx`
- `gif`
- `exe`
- `jpg`
- `jpeg`
- `mp3`
- `mp4`
- `mov`
- `pdf`
- `ppt`
- `pptx`
- `png`
- `rar`
- `rtf`
- `tiff`
- `txt`
- `wav`
- `zip`
- `gz`
- `xls`
- `xlsx`
- `json`

```ts
<File extension="doc" />
<File extension="docx" />
<File extension="jpg" />
<File extension="mov" />
<File extension="BlaBla" />
```

## Размер иконки

За размер иконки отвечает свойство `size`.

```ts
<File size="m" />
<File size="s" />
```

## Индикаторы загрузки

Свойство `loading` показывает, что файл загружается.

Если нужен индикатор загрузки, добавьте `loadingWithProgressSpin`.

Процент загрузки можно указать с помощью `loadingProgress`.

```ts
<File loading />
<File loading loadingWithProgressSpin />
<File loading loadingWithProgressSpin loadingProgress={30} />
<File loading loadingWithProgressSpin loadingProgress={60} />
<File loading loadingWithProgressSpin loadingProgress={90} />
```

## Свойства

| Свойство                                          | Тип                           | По умолчанию | Описание                                                         |
| ------------------------------------------------- | ----------------------------- | ------------ | ---------------------------------------------------------------- |
| [`extension`](#иконка)                            | `string`                      | -            | Иконка файла                                                     |
| [`size`](#размер-иконки)                          | `'m', 's'`                    | `'m'`        | Размер иконки                                                    |
| [`loading`](#индикаторы-загрузки)                 | `boolean`                     | -            | Показывает, что файл загружается                                 |
| [`loadingWithProgressSpin`](#индикаторы-загрузки) | `boolean`                     | -            | Показывает индикатор прогресса загрузки                          |
| [`loadingProgress`](#индикаторы-загрузки)         | `number`                      | -            | Прогресс загрузки в процентах                                    |
| `as`                                              | `keyof JSX.IntrinsicElements` | -            | HTML-тег, в виде которого этот компонент будет представлен в DOM |
| `className`                                       | `string`                      | -            | Дополнительный CSS-класс                                         |
| `ref`                                             | `React.Ref<HTMLDivElement>`   | -            | Ссылка на корневой DOM-элемент компонента                        |

## Пример использования

// src/App.ts
