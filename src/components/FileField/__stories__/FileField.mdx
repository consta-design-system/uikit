# FileField

Компонент для выбора файлов. Пользователь может выбрать файлы на своем компьютере, чтобы куда-то загрузить или отправить.

- [Пример с renderProp](#пример-с-renderprop)
- [Пример без renderProp](#пример-без-renderprop)
- [Список свойств](#свойства)
- [Пример использования](#пример)

## Пример с renderProp

<FileFieldExampleRenderProp />

```ts
import React from 'react';
import { FileField, FileFieldProps } from '@consta/uikit/FileField';
import { Button } from '@consta/uikit/Button';

const App = () => {
  return <FileField>{(props) => <Button {...props} label="Выбрать файлы" />}</FileField>;
};
```

## Свойства

| Свойство    | Тип                                          | По умолчанию | Описание                        |
| ----------- | -------------------------------------------- | ------------ | ------------------------------- |
| `id`        | `string`                                     | -            | Идентификатор                   |
| `onChange?` | `(e: DragEvent, React.ChangeEvent) => void;` | -            | Действие при выборе файла       |
| `inputRef?` | `React.Ref<HTMLInputElement>;`               | -            | Ссылка на DOM елемент `<input>` |
|             | и все атрибуты input                         |              |                                 |
