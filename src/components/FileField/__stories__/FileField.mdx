# FileField

Компонент для выбора фалов. Пользователь может выбрать файлы на своем компьютере чтобы куда-то загрузить или отправить.

- [Пример с renderProp](#пример-с-renderprop)
- [Пример без renderProp](#пример-без-renderprop)
- [Список свойств](#свойства)
- [Пример использования](#пример)

## Пример с renderProp

<FileFieldExampleRenderProp />

```ts
import React from 'react';
import { FileField, FileFieldProps } from '@gpn-design/uikit/FileField';
import { Button } from '@gpn-design/uikit/Button';

const App = () => {
  return <FileField>{(props) => <Button {...props} label="Выбрать файлы" />}</FileField>;
};
```

## Свойства

| Свойство  | Тип                                                                      | По умолчанию | Описание                |
| --------- | ------------------------------------------------------------------------ | ------------ | ----------------------- |
| `id`        | `string`                                                                 | -            | Идентификатор                      |
| `onChange`  | `(object: { e: React.ChangeEvent; value: FileList , null }) => void;`    | -            | Действие при выборе файла    |
| `value`     | `FileList , null`                                                        | -            | Выбранные файлы         |
| `children`  | `React.ComponentType<{ onClick: React.MouseEventHandler<HTMLElement> }>` | -            | Компонент представления |
| `multiple `| `boolean`                                                                | -            | Множественный выбор     |

import React from 'react';
import { FileField } from '@gpn-design/uikit/FileField';

  return <FileField>{(props) => <Button {...props} label="Выбрать файл" />}</FileField>;
