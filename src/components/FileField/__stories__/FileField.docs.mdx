# FileField

Компонент для выбора файлов. С его помощью пользователь может выбрать файлы на своем компьютере, чтобы куда-то загрузить или отправить.

Это компонент-обёртка: сам по себе он не существует, его можно использовать вместе с любым другим компонентом.

Добавьте кнопку, иконку, текст, изображение или другой компонент в контейнер `<FileField></FileField>`. Нажмите на этот компонент — откроется выбор файлов.

## Пример с Render Prop

Нажмите на кнопку, чтобы открыть выбор файлов.

<FileFieldExampleRenderProp />

```ts
import React from 'react';
import { FileField, FileFieldProps } from '@consta/uikit/FileField';
import { Button } from '@consta/uikit/Button';

const App = () => {
  return <FileField>{(props) => <Button {...props} label="Выбрать файлы" />}</FileField>;
};
```

## Пример без Render Prop

Нажмите на текст `Нажми меня` или на иконку `+`, чтобы открыть выбор файлов.

<FileFieldExampleWithoutRenderProp />
<FileFieldExampleWithoutRenderPropIcon />

```ts
import React from 'react';
import { FileField, FileFieldProps } from '@consta/uikit/FileField';
import { Text } from '@consta/uikit/Text';
import { IconAdd } from '@consta/uikit/IconAdd';

const App = () => {
  return (
    <FileField id="FileFieldWithText">
      <Text>Нажми меня</Text>
    </FileField>
    <FileField id="FileFieldWithIcon">
      <IconAdd view="alert" />
    </FileField>
  );
};
```

## Свойства

| Свойство    | Тип                                         | По умолчанию | Описание                                           |
| ----------- | ------------------------------------------- | ------------ | -------------------------------------------------- |
| `id`        | `string`                                    | -            | Идентификатор компонента                           |
| `onChange?` | `(e: DragEvent, React.ChangeEvent) => void` | -            | Действие при выборе файла                          |
| `inputRef?` | `React.Ref<HTMLInputElement>`               | -            | Ссылка на DOM-элемент `<input>` и все его атрибуты |
