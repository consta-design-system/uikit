## Обзор

Компонент `CheckboxGroup` позволяет создавать группу чекбоксов, которые могут быть выбраны пользователем. Поддерживает настройку содержимого, внешнего вида, расположения и поведения, включая возможность отключения отдельных чекбоксов или всей группы.

## Импорт

```tsx
import { CheckboxGroup } from '@consta/uikit/CheckboxGroup';
```

## Содержимое

Свойство `items` задаёт массив объектов, описывающих каждый чекбокс. Объект может содержать поля:

- `name` — название чекбокса (обязательное).
- `disabled` — отключает отдельный чекбокс (необязательное).
- `key` — уникальный ключ элемента (необязательное).
- `attributes` — дополнительные HTML-атрибуты для чекбокса.
- `ref` — ссылка на элемент чекбокса.

Свойство `getItemLabel` определяет, какое поле объекта использовать для отображения текста чекбокса. Свойство `name` задаёт имя `<input>` для обеспечения доступности.

<MdxTabs>

<CheckboxGroupExampleName />

```tsx
const items = [
  { name: 'Один' },
  { name: 'Два' },
  { name: 'Три' },
  { name: 'Четыре', disabled: true },
  { name: 'Пять' },
];

const [value, setValue] = React.useState(null);

<CheckboxGroup
  value={value}
  items={items}
  getItemLabel={(item) => item.name}
  getItemDisabled={(item) => item.disabled}
  onChange={setValue}
  name="CheckboxGroup"
/>;
```

</MdxTabs>

## Поведение

### Неактивный чекбокс

Для отключения отдельного чекбокса используйте свойство `getItemDisabled` или поле `disabled` в объекте `items`. Неактивный чекбокс нельзя выбрать.

<MdxTabs>

<CheckboxGroupExampleName />

```tsx
const items = [
  { name: 'Один' },
  { name: 'Два' },
  { name: 'Три' },
  { name: 'Четыре', disabled: true },
  { name: 'Пять' },
];

const [value, setValue] = React.useState(null);

<CheckboxGroup
  value={value}
  items={items}
  getItemLabel={(item) => item.name}
  getItemDisabled={(item) => item.disabled}
  onChange={setValue}
  name="CheckboxGroup"
/>;
```

</MdxTabs>

### Неактивная группа чекбоксов

Свойство `disabled` отключает всю группу чекбоксов, делая её неинтерактивной.
