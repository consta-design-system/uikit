Группа чекбоксов, объединённых одним `name`. Пользователь может выбрать несколько вариантов из списка.

Если пользователю нужно выбрать только один вариант из списка, лучше использовать [RadioGroup](##LIBS.LIB.STAND/lib:uikit/stand:components-radiogroup-stable).

Если нужен только один чекбокс, можно использовать [Checkbox](##LIBS.LIB.STAND/lib:uikit/stand:components-checkbox-stable).
