# CheckboxGroup

Группа чекбоксов, объединённых одним `name`. Пользователь может выбрать несколько вариантов из списка.

Если пользователю нужно выбрать только один вариант из списка, лучше использовать [RadioGroup](?path=/docs/components-radiogroup--playground).

Если нужен только один чекбокс, можно использовать [Checkbox](?path=/docs/components-checkbox--playground)

## Содержимое

Информация об отдельных чекбоксах описывается в массиве `items`. В него можно поместить название чекбокса и его неактивность. Например, вот так:

```tsx
const items: Item[] = [
  { name: 'Один' },
  { name: 'Два' },
  { name: 'Три' },
  { name: 'Четыре', disabled: true },
  { name: 'Пять' },
];
```

<CheckboxGroupExampleName />

```tsx
const [value, setValue] = React.useState<Item | null>(null);

return (
  <CheckboxGroup
    value={value}
    items={items}
    getLabel={(item) => item.name}
    getDisabled={(item) => item.disabled}
    onChange={({ value }) => setValue(value)}
    name={'CheckboxGroup'}
  />
```

### Название чекбокса

Надпись рядом с чекбоксом можно получить с помощью функции `getLabel`. Лучше писать название с заглавной буквы, точка не нужна.

#### Согласуйте с заголовком

Это просто лучше читается. Сравните:

<CheckboxGroupExampleHead />

Если не нравится, что пункты начинаются с предлогов, поменяйте заголовок.

<CheckboxExampleGroupHead3 />

#### Не используйте отрицания

Чтобы не путать пользователя. Когда выбран чекбокс, рядом с которым написано **Не хочу**,
приходится мысленно решать равенство:`не + не + хочу = хочу`.

<CheckboxGroupExampleNeg />

#### Не обобщайте

Если хотите дать пользователю возможность отметить все элементы в списке
или, наоборот, удалить все отметки, сделайте это другим способом.

<CheckboxGroupExampleGeneral />

### Неактивный чекбокс

Один или несколько чекбоксов можно сделать неактивными — если вы не хотите, чтобы их выбирали. За неактивность отвечает функция `getDisabled`.

## Расположение чекбоксов

Вы можете вывести чекбоксы в столбик или расположить на одной строчке с помощью `direction`. Варианты:

- `column` — в столбик, друг под другом (работает по умолчанию),
- `row` — на одной строке.

**Чекбоксы в столбик**

<CheckboxGroupExampleColumn />

```tsx
type Item = {
  name: string;
  disabled?: boolean;
};

const items: Item[] = [
  { name: 'Один' },
  { name: 'Два' },
  { name: 'Три' },
  { name: 'Четыре', disabled: true },
  { name: 'Пять' },
];

function CheckboxGroupExampleRow() {
  const [value, setValue] = React.useState<Item[] | null>(null);

  return (
    <CheckboxGroup
      value={value}
      items={items}
      getLabel={(item) => item.name}
      getDisabled={(item) => item.disabled}
      onChange={({ value }) => setValue(value)}
      direction="column"
    />
  );
}
```

## Внешний вид

Всего в дизайн-системе два вида групп чекбоксов: `primary` и `ghost`. За это отвечает свойство `view`.

<CheckboxGroupExampleView />

```tsx
type Item = {
  name: string;
  disabled?: boolean;
};

const items: Item[] = [
  { name: 'Один' },
  { name: 'Два' },
  { name: 'Три' },
  { name: 'Четыре', disabled: true },
  { name: 'Пять' },
];

function CheckboxGroupExampleView() {
  const [value, setValue] = React.useState<Item[] | null>(null);

  return (
    <>
      <p className={cnDocsExample('Caption')}>
        <b>primary</b>
      </p>
      <CheckboxGroup
        value={value}
        items={items}
        getLabel={(item) => item.name}
        getDisabled={(item) => item.disabled}
        onChange={({ value }) => setValue(value)}
        name="CheckboxGroup"
        direction="row"
        view="primary"
      />
      <p className={cnDocsExample('Caption')}>
        <b>ghost</b>
      </p>
      <CheckboxGroup
        value={value}
        items={items}
        getLabel={(item) => item.name}
        getDisabled={(item) => item.disabled}
        onChange={({ value }) => setValue(value)}
        name="CheckboxGroup"
        direction="row"
        view="ghost"
      />
    </>
  );
}
```

## Свойства

```
type OnChange = (props: { e: React.ChangeEvent<HTMLInputElement>; value: ITEM[] | null }) => void;
```

| Свойство                              | Тип                       | По умолчанию | Описание                                                                                                |
| ------------------------------------- | ------------------------- | ------------ | ------------------------------------------------------------------------------------------------------- |
| `value`                               | `Item[]`                  | —            | Значение                                                                                                |
| [`items`](#содержимое)                | `Item[]`                  | —            | Элементы                                                                                                |
| [`getLabel`](#название-чекбокса)      | `(item: Item) => string`  | —            | Функция для формирования текста для чекбоксов. Также используется в качестве уникального ключа элемента |
| [`getDisabled?`](#неактивный-чекбокс) | `(item: Item) => boolean` | —            | Функция для определения disabled состояния чекбокса                                                     |
| `onChange`                            | `OnChange`                | —            | Действие при выборе элемента                                                                            |
| `name`                                | `string`                  | —            | Имя `<input>`, указывается для доступности                                                              |
| [`direction?`](#направление)          | `'column', 'row'`         | `'column'`   | Направление чекбоксов                                                                                   |
| [`disabled?`](#состояние)             | `boolean`                 | `false`      | Делает группу чекбоксов неактивной                                                                      |
| [`size?`](#размер)                    | `'m', 'l'`                | `'m'`        | Размер группы чекбоксов                                                                                 |
| [`view?`](#внешний-вид)               | `'primary', 'ghost'`      | `'primary'`  | Внешний вид                                                                                             |
| `className?`                          | `string`                  | —            | Дополнительные CSS-классы                                                                               |
