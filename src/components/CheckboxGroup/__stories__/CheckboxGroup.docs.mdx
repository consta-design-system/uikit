# CheckboxGroup

Группа чекбоксов, объединённых одним `name`. Пользователь может выбрать несколько вариантов из списка.
Если пользователю необходимо выбрать только один вариант из списка, лучше использовать [RadioGroup](?path=/docs/components-radiogroup--playground).

## Направление

Группа чекбоксов может располагаться как по вертикали, так и по горизонтали. За направление отвечает свойство `direction`.

```tsx
<CheckboxGroup
  direction="column"
/>
<CheckboxGroup
  direction="row"
/>
```

## Состояние

`disabled` не дает пользователю сделать выбор.

```tsx
<CheckboxGroup disabled />
```

## Размер

За размер отвечает свойство `size`.

```tsx
<CheckboxGroup size="m" />
<CheckboxGroup size="l" />
```

## Внешний вид

Всего в дизайн-системе два вида групп чекбоксов: `primary` и `ghost`. За это отвечает свойство `view`.

```tsx
<CheckboxGroup view="primary" />
<CheckboxGroup view="ghost" />
```

## Пример

```ts
// src/App.ts
import React from 'react';
import { CheckboxGroup } from '@consta/uikit/CheckboxGroup';

type Item = {
  name: string;
  disabled?: boolean;
};

const items: Item[] = [
  { name: 'один' },
  { name: 'два' },
  { name: 'три' },
  { name: 'disabled', disabled: true },
  { name: 'пять' },
];

const App = () => {
  const [value, setValue] = React.useState<Item[] | null>(null);

  return (
    <CheckboxGroup
      value={value}
      items={items}
      getLabel={(item) => item.name}
      getDisabled={(item) => item.disabled}
      onChange={({ value }) => setValue(value)}
      name={'CheckboxGroup'}
    />
  );
};
```
