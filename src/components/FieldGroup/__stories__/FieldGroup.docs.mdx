# FieldGroup

```tsx
import { FieldGroup } from '@consta/uikit/FieldGroup';
```

Этот компонент объединяет элементы формы в одну строку, например, поле ввода и кнопку.

Компонент проставляет отступы между полями и форму для всей конструкции. Подходит для компонентов со свойством `form`: Select, Combobox, TextField, UserSelect, DatePicker, Button и похожих.

## Содержимое

Добавьте в контейнер FieldGroup компоненты в том порядке, в котором их нужно
объединить. Укажите форму и размер для всех полей — компонент проставит нужные
рамки и размеры для всех вложенных компонентов.

```tsx
import { Button } from '@consta/uikit/Button';
import { Select } from '@consta/uikit/Select';
import { TextField } from '@consta/uikit/TextField';
import { FieldGroup } from '@consta/uikit/FieldGroup';

<FieldGroup form="round" size="m">
  <TextField placeholder="TextField" />
  <Select
    placeholder="Select"
    items={items}
    onChange={() => {
      console.log('onChange');
    }}
  />
  <Button label="Button" />
</FieldGroup>;
```

## Форма

У компонента есть два набора форм:

- простые: `'default'`, `'brick'`, `'round'`
- и составные: `'roundBrick'`, `'brickDefault'`, `'defaultBrick'`, `'defaultClear'`, `'brickClear'`, `'roundClear'`, `'clearRound'`, `'clearDefault'`, `'clearBrick'`, `'clearClear'`.

##### Пример составной формы

```tsx
<FieldGroup form="clearBrick" size="m">
  <TextField placeholder="TextField" />
  <Select
    placeholder="Select"
    items={items}
    onChange={() => {
      console.log('onChange');
    }}
  />
  <Button label="Button" />
</FieldGroup>
```

## Размер

За размер группы полей отвечает свойство `size`. Варианты: `'xs'`, `'s'`, `'m'`, '`l'`.

```tsx
<FieldGroup size="xs">
  <TextField placeholder="xs" />
  <Select
    placeholder="Select"
    items={items}
    onChange={() => {
      console.log('onChange');
    }}
  />
  <Button label="Button" />
</FieldGroup>
```
