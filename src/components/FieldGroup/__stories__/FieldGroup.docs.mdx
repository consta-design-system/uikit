# FieldGroup

```tsx
import { FieldGroup } from '@consta/uikit/FieldGroup';
```

Компонент для обьеденения полей ввода в одну строку.

```tsx
<FieldGroup form="round" size="m">
  <TextField placeholder="TextField" />
  <Select
    placeholder="Select"
    items={items}
    onChange={() => {
      console.log('onChange');
    }}
  />
  <Button label="Button" />
</FieldGroup>
```
