## Импорт

```tsx
import { FieldGroup } from '@consta/uikit/FieldGroup';
```

## Свойства

| Свойство           | Тип                                                                                                                                                                                                                                                                                                                      | По умолчанию | Описание                                  |
| ------------------ | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ | ------------ | ----------------------------------------- |
| [`form?`](#форма)  | `'default'` &#124; `'brick'` &#124; `'round'` &#124; `'clear'` &#124; `'brickRound'` &#124; `'roundBrick'` &#124; `'brickDefault'` &#124; `'defaultBrick'` &#124; `'defaultClear'` &#124; `'brickClear'` &#124; `'roundClear'` &#124; `'clearRound'` &#124; `'clearDefault'` &#124; `'clearBrick'` &#124; `'clearClear'` | `'default'`  | Форма вложенных элементов                 |
| [`size?`](#размер) | `'xs'` &#124; `'s'` &#124; `'m'` &#124; `'l'`                                                                                                                                                                                                                                                                            | `'m'`        | Размер вложенных элементов                |
| `ref?`             | `React.Ref<HTMLDivElement>`                                                                                                                                                                                                                                                                                              | -            | Ссылка на корневой DOM-элемент компонента |
| `className?`       | `string`                                                                                                                                                                                                                                                                                                                 | -            | Дополнительный CSS-класс компонента       |

## Содержимое

Добавьте в контейнер `FieldGroup` компоненты в том порядке, в котором их нужно объединить. Укажите форму и размер для всех полей — компонент проставит нужные рамки и размеры для всех вложенных компонентов.

```tsx
import { Button } from '@consta/uikit/Button';
import { Select } from '@consta/uikit/Select';
import { TextField } from '@consta/uikit/TextField';
import { FieldGroup } from '@consta/uikit/FieldGroup';

<FieldGroup form="round" size="m">
  <TextField placeholder="TextField" />
  <Select
    placeholder="Select"
    items={items}
    onChange={() => {
      console.log('onChange');
    }}
  />
  <Button label="Button" />
</FieldGroup>;
```

## Внешний вид

### Форма

У компонента есть два набора форм:

- простые: `'default'`, `'brick'`, `'round'`, `'clear'`
- и составные: `'roundBrick'`, `'brickRound'`, `'brickDefault'`, `'defaultBrick'`, `'defaultClear'`, `'brickClear'`, `'roundClear'`, `'clearRound'`, `'clearDefault'`, `'clearBrick'`, `'clearClear'`.

Пример составной формы:

<FieldGroupExampleForm />

```tsx
<FieldGroup form="clearBrick" size="m">
  <TextField placeholder="TextField" />
  <Select
    placeholder="Select"
    items={items}

      console.log('onChange');
    }}
  />
  <Button label="Button" />
</FieldGroup>
```

### Размер

За размер группы полей отвечает свойство `size`. Варианты: `'xs'`, `'s'`, `'m'`, '`l'`.
