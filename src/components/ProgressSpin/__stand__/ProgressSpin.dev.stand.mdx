## Обзор

Компонент `ProgressSpin` — это индикатор загрузки, который отображает прогресс выполнения процесса или бесконечную анимацию ожидания. Используется для визуализации статуса выполнения задачи в интерфейсе.

## Импорт

```tsx
import { ProgressSpin } from '@consta/uikit/ProgressSpin';
```

## Свойства

| Свойство                            | Тип                                  | По умолчанию | Описание                                                                           |
| ----------------------------------- | ------------------------------------ | ------------ | ---------------------------------------------------------------------------------- |
| [`value?`](#процент-готовности)     | `number`                             | -            | Выполненная часть процесса в процентах (определяет заполненную область индикатора) |
| [`animation?`](#процент-готовности) | `boolean`                            | -            | Включает плавное изменение индикатора до величины, указанной в `value`             |
| [`size?`](#размер)                  | `'s' \| 'm' \| 'l' \| 'xl' \| '2xl'` | `'m'`        | Размер индикатора                                                                  |
| `className?`                        | `string`                             | -            | Дополнительный класс                                                               |
| `ref?`                              | `React.Ref<SVGSVGElement>`           | -            | Ссылка на корневой DOM-элемент компонента                                          |

## Содержимое

### Процент готовности

Свойство `value` позволяет указать, насколько процесс близок к завершению (в процентах). Если задать `value`, индикатор отобразит заполненную область, соответствующую указанному значению.

Для добавления плавной анимации изменения прогресса используйте свойство `animation`. При включении анимации индикатор будет плавно заполняться до значения, указанного в `value`.

Если свойство `value` не задано, индикатор будет бесконечно вращаться, показывая состояние ожидания.

## Внешний вид

### Размер

Свойство `size` определяет размер индикатора. Доступные значения: `s`, `m`, `l`, `xl`, `2xl`. По умолчанию используется размер `m`.

<MdxTabs>

<ProgressSpinExampleSize />

```tsx
<>
  <ProgressSpin value={70} size="2xl" />
  <ProgressSpin value={70} size="xl" />
  <ProgressSpin value={70} size="l" />
  <ProgressSpin value={70} size="m" />
  <ProgressSpin value={70} size="s" />
</>
```

## Пример использования

Ниже приведён пример использования компонента `ProgressSpin` с указанием прогресса:

```tsx
import React from 'react';
import { ProgressSpin } from '@consta/uikit/ProgressSpin';

const App = () => {
  return <ProgressSpin value={70} />;
};
```
