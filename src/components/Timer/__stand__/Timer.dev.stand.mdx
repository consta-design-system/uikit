## Обзор

Компонент `Timer` отображает таймер с прогресс-баром для коротких промежутков времени (рекомендуется до 10 секунд). Позволяет задавать длительность в секундах, размер, процент прогресса и анимацию заполнения.

## Импорт

```tsx
import { Timer } from '@consta/uikit/Timer';
```

## Свойства

| Свойство                      | Тип                                  | По умолчанию | Описание                                        |
| ----------------------------- | ------------------------------------ | ------------ | ----------------------------------------------- |
| [`seconds?`](#секунды)        | `number`                             | `5`          | Секунды                                         |
| [`progress?`](#прогресс-бар)  | `number`                             | `0`          | Процент заполнения прогресс-бара                |
| [`animation?`](#прогресс-бар) | `boolean`                            | -            | Включает плавное изменение прогресс-бара        |
| [`size?`](#размер)            | `'s' \| 'm' \| 'l' \| 'xl' \| '2xl'` | `'m'`        | Размер                                          |
| `className?`                  | `string`                             | -            | Дополнительный CSS-класс                        |
| `ref?`                        | `React.Ref<HTMLDivElement>`          | -            | Ссылка на корневой DOM-элемент этого компонента |

## Содержимое

### Секунды

Вы можете установить таймер на нужное количество секунд с помощью свойства `seconds`. Компонент предназначен для очень коротких промежутков времени, до 10 секунд, поэтому двузначные числа лучше не использовать. По умолчанию 5 секунд.

```tsx
<Timer size="m" seconds={3} progress={80} />
<Timer size="m" seconds={5} progress={80} />
<Timer size="m" seconds={7} progress={80} />
```

<TimerExampleSeconds />

### Прогресс-бар

Вы можете указать процент заполнения прогресс-бара (от 0 до 100) в свойстве `progress`. По умолчанию 0.

```tsx
<Timer size="m" seconds={5} />
<Timer size="m" seconds={5} progress={20} />
<Timer size="m" seconds={5} progress={40} />
<Timer size="m" seconds={5} progress={60} />
<Timer size="m" seconds={5} progress={80} />
```

<TimerExampleProgress />

Включите анимацию с помощью `animation`. Скорость анимации по умолчанию равна 1 секунде, ее можно изменить в CSS-переменной `--animation-duration`.

```css
.MyBlock.Timer-Progress.ProgressSpin {
  --animation-duration: 5s;
}
```

## Внешний вид

### Размер

За размер таймера отвечает свойство `size`. Возможные значения: `'s'`, `'m'`, `'l'`, `'xl'`, `'2xl'`. По умолчанию `'m'`.

```tsx
<Timer size="s" seconds={5} progress={80} />
<Timer size="m" seconds={5} progress={80} />
<Timer size="l" seconds={5} progress={80} />
<Timer size="xl" seconds={5} progress={80} />
<Timer size="2xl" seconds={5} progress={80} />
```

## Пример использования

```tsx
// src/App.ts
import React from 'react';
import { Timer } from '@consta/uikit/Timer';

const App = () => {
  return <Timer seconds={5} progress={80} />;
};
```
