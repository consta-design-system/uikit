## Обзор

Компонент `RadioGroup` предназначен для создания группы радиокнопок, позволяющих пользователю выбрать один вариант из нескольких. Подходит для форм, где требуется однозначный выбор, например, выбор размера, типа или категории. Компонент поддерживает настройку содержимого, внешнего вида и поведения.

## Импорт

```tsx
import { RadioGroup } from '@consta/uikit/RadioGroup';
```

## Содержимое

### Название радиокнопки

Название радиокнопки задаётся через свойство `label` в объекте элемента массива `items` или с помощью функции `getItemLabel`. Название должно начинаться с заглавной буквы и не содержать точку в конце.

Пример:

```tsx
const items = [{ name: 'Один' }, { name: 'Два' }, { name: 'Три' }];
```

### Неактивная радиокнопка

Чтобы сделать радиокнопку неактивной, используйте свойство `disabled` в объекте элемента массива `items` или функцию `getItemDisabled`. Неактивные радиокнопки нельзя выбрать.

<MdxTabs>

<RadioGroupExampleDisabledItem />

```tsx
type Item = {
  name: string;
  disabled?: boolean;
};

const items = [
  { name: 'Один' },
  { name: 'Два' },
  { name: 'Три' },
  { name: 'Четыре' },
  { name: 'Пять', disabled: true },
];

function RadioGroupExampleDisabledItem() {
  const [value, setValue] = React.useState<Item | null>(items[0]);

  return (
    <RadioGroup
      value={value}
      items={items}
      getItemLabel={(item) => item.name}
      getItemDisabled={(item) => item.disabled}
      onChange={({ value }) => setValue(value)}
    />
  );
}
```

</MdxTabs>

### Неактивная группа радиокнопок

Свойство `disabled` делает всю группу радиокнопок неактивной, отключая взаимодействие с ней. Значение по умолчанию: `false`.

<MdxTabs>

<RadioGroupExampleDisabled />

```tsx
const simpleItems = ['один', 'два', 'три', 'четыре', 'пять'];

function RadioGroupExampleDisabled() {
  const [value, setValue] = React.useState<string | null>(simpleItems[0]);

  return (
    <RadioGroup
      value={value}
      items={simpleItems}
      getItemLabel={(item) => item}
      onChange={({ value }) => setValue(value)}
      disabled
    />
  );
}
```

</MdxTabs>

## Внешний вид

### Размер

Свойство `size` определяет размер радиокнопок. Доступные значения: `'xs'`, `'s'`, `'m'`, `'l'`. Значение по умолчанию: `'m'`.

```tsx

function RadioGroupExampleSize() {
  const [value, setValue] = React.useState<string | null>(simpleItems[0]);

  return (
    <Example>
      <RadioGroup
        value={value}
        items={simpleItems}
        getItemLabel={(item) => item}
        onChange={({ value }) => setValue(value)}
        size="l"
      />
      <RadioGroup
        value={value}
        items={simpleItems}
        getItemLabel={(item) => item}
        onChange={({ value }) => setValue(value)}
        size="m"
      />
      <RadioGroup
        value={value}
        items={simpleItems}
        getItemLabel={(item) => item}
        onChange={({ value }) => setValue(value)}
        size="s"
      />
      <RadioGroup
        value={value}
        items={simpleItems}
        getItemLabel={(item) => item}
        onChange={({ value }) => setValue(value)}
        size="xs"
      />
    </Example>
  );
}
```

### Стиль

Свойство `view` задаёт внешний вид группы радиокнопок. Доступные значения: `'primary'` (по умолчанию) и `'ghost'`.

<MdxTabs>

<RadioGroupExampleView />

```tsx
const simpleItems = ['один', 'два', 'три', 'четыре', 'пять'];

function RadioGroupExampleView() {
  const [value, setValue] = React.useState<string | null>(simpleItems[0]);

  return (
    <Example>
      <RadioGroup
        value={value}
        items={simpleItems}
        getItemLabel={(item) => item}
        onChange={({ value }) => setValue(value)}
        direction="row"
        view="primary"
      />

        value={value}
        items={simpleItems}
        getItemLabel={(item) => item}
        onChange={({ value }) => setValue(value)}
        direction="row"
        view="ghost"
      />
    </Example>
  );
}
```

### Выравнивание

Свойство `align` определяет выравнивание текста относительно радиокнопок. Доступные значения: `'center'` (по умолчанию) и `'top'`.

<MdxTabs>

<RadioGroupExampleAlign />

```tsx

  'Это текст',
  'Это длинный текст',
  'Это очень длинный текст',
  'Это очень-очень длинный текст, которому хочется быть ещё длиннее',
  'Этот текст гораздо длиннее всех предыдущих текстов, что само по себе уже даже странно, хотя на свете бывает много гораздо более странных вещей',
];

export function RadioGroupExampleAlign() {
  const [value, setValue] = React.useState<string | null>(alignItems[0]);

  return (
    <>
      <RadioGroup
        value={value}
        items={alignItems}
        getItemLabel={(item) => item}
        onChange={({ value }) => setValue(value)}
        align="top"
      />
      <RadioGroup
        value={value}
        items={alignItems}
        getItemLabel={(item) => item}
        onChange={({ value }) => setValue(value)}
        align="center"
      />
    </>
  );
}
```

</MdxTabs>

### Расположение

Свойство `direction` задаёт направление расположения радиокнопок. Доступные значения: `'column'` (по умолчанию, вертикально) и `'row'` (горизонтально).

```tsx
return (
  <RadioGroup
    value={value}
    items={simpleItems}
    getItemLabel={(item) => item}
    onChange={({ value }) => setValue(value)}
    direction="row"
  />
);
```
