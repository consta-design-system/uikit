# Установка Consta

<MdxTabs labels = {['npm', 'yarn']} view = "clear">
<>

1. Установите [Node.js](https://nodejs.org/en).
2. Создайте новый проект:

```cli
npx create-react-app my-app
```

3. Перейдите в директорию проекта:

```cli
cd my-app
```

4. Установите пакет Consta:

```cli
npm install @consta/uikit
```

</>

1. Установите [Node.js](https://nodejs.org/en).

```cli
yarn create react-app my-app
```



# Импорт компонентов

1. Импортируйте в корневой файл:
   - библиотеку React,
   - компонент [Theme](##LIBS.LIB.STAND.TAB/lib:uikit/stand:components-theme-stable/tab:dev),
   - [пресет](##LIBS.LIB.STAND.TAB/lib:uikit/stand:components-theme-stable/tab:dev/hash:пресеты),
   - компоненты.
2. Вложите компоненты в тему.

Пример:

```jsx
import React from 'react';
import { Theme, presetGpnDefault } from '@consta/uikit/Theme';
import { Button } from '@consta/uikit/$компонент';

const App = () => (
  <Theme preset={presetGpnDefault}>
    <Button label="Кнопка" />
  </Theme>
);
```

# Что дальше

Создайте интерфейс по макетам и спецификации, которые передал дизайнер. Используйте компоненты из [библиотек Consta](https://github.com/consta-design-system) на Github. Если нужного компонента нет, [создайте новый](##LIBS.LIB.STAND/lib:portal/stand:contributers-componentguide) и [отправьте](##LIBS.LIB.STAND/lib:portal/stand:contributers-addcomponent) его нам — мы будем рады вашему участию.
