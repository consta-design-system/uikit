# useClickOutside

Хук ислпульзуется для получения событий клика вне элемента

| Свойство                 | Тип                                     | По умолчанию | Описание                                                                                                                      |
| ------------------------ | --------------------------------------- | ------------ | ----------------------------------------------------------------------------------------------------------------------------- |
| `isActive`               | `boolean`                               | -            | включение работы хука                                                                                                         |
| `ignoreClicksInsideRefs` | `ReadonlyArray<RefObject<HTMLElement>>` | -            | ссылки на элементы по которым `handler` не будет вызыватся, обычно это ссылка на элемент по вне корторого должен сработат хук |
| `handler`                | `(event: MouseEvent) => void;`          | -            | функция которая сработает при клике вне элемента                                                                              |
