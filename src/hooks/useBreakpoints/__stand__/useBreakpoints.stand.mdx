```tsx
import { useBreakpoints } from '@consta/uikit/useBreakpoints';
```

Хук `useBreakpoints` подписывается на изменение ширины экрана и возвращает набор контрольных точек (`breakpoints`), которые применяются в момент этого изменения.

Если вызвать `useBreakpoints` без аргументов, вернутся стандартные брейкпоинты:

```tsx
const breakpoints = useBreakpoints();
// type breakpoints = Record<"5xs" | "4xs" | "3xs" | "2xs" | "xs" | "s" | "m" | "l" | "xl" | "2xl" | "3xl" | "4xl" | "5xl", boolean>
```

Можно установить свои брейкпоинты:

```tsx
const breakpoints = useBreakpoints({ isDesktop: 1000 });
// type breakpoints = Record<"isDesktop", boolean>
```

## Свойства

```tsx
Map<POINTS extends string> = Record<POINTS, number>
```

| Свойство | Тип и значения | По умолчанию | Описание                                |
| -------- | -------------- | ------------ | --------------------------------------- |
| `map?`   | `Map`          | -            | Набор контрольных точек (`breakpoints`) |
