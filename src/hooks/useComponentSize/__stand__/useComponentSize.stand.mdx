```tsx
import { useComponentSize } from '@consta/uikit/useComponentSize';
```

Хук `useComponentSize` подписывается на изменение размеров передаваемого элемента и возвращает его ширину и высоту объекта.

Данный хук нужен для отслеживания размеров конкретного элмента DOM-дерева, если ваша задача состоит в адаптации контента испльзуйте хук [useComponentBreakpoints](##LIBS.LIB.STAND/lib:uikit/stand:hooks-useсomponentbreakpoints-stable), так как процесс ререндера происходит только при пересечении контрольной точки.

```tsx
import { useComponentSize } from '@consta/uikit/useComponentSize';

...

const containerRef = useRef<HTMLDivElement>(null);
const { width, height } = useComponentSize(containerRef);

return <div ref={containerRef}>...</div>;
```

## Свойства

| Аргумент | Тип и значения                                                        | По умолчанию | Описание          |
| -------- | --------------------------------------------------------------------- | ------------ | ----------------- |
| `ref`    | `React.RefObject<HTMLElement`<code>&#124;</code>`SVGGraphicsElement>` | -            | Ссылка на элемент |
