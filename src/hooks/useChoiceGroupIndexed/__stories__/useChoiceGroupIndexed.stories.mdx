# useChoiceGroupIndexed

```tsx
import { useChoiceGroupIndexed } from '@consta/uikit/useChoiceGroupIndexed';
```

Этот хук может понадобиться в компонентах, где пользователи могут выбрать несколько значений из списка.
Например, [CollapseGroup](/?path=/docs/components-collapsegroup--playgrounds).

Хук `useChoiceGroupIndexed` занимается всей логикой выбора: вы можете использовать его,
чтобы не обрабатывать эту логику самостоятельно.

> Для работы хука не нужен уникальный ключ элемента. Выбор происходит по порядковому номеру.

## Свойства

| Свойство           | Тип и значения                                                                                                                         | По умолчанию | Описание                                                                   |
| ------------------ | -------------------------------------------------------------------------------------------------------------------------------------- | ------------ | -------------------------------------------------------------------------- |
| `multiple`         | `boolean`                                                                                                                              | -            | Показывает, что может быть выбрано несколько значений                      |
| `value`            | `number[]` &#124; `number` &#124; `null`                                                                                               | -            | Текущее значение или несколько                                             |
| `callBack`         | `callBack: (props: { e: EVENT; value: number[]` &#124; `null;}) => void, (props: { e: EVENT; value: number` &#124; `null; }) => void;` | -            | Обработчик, который нужно вызывать                                         |
| `isNullableValue?` | `boolean`                                                                                                                              | -            | Показывает, что значение может быть нулевым. Подробности в примечании ниже |

**Примечание про `isNullableValue`:** некоторые компоненты позволяют удалить выбранные значения,
а некоторые — нет (например, [Radio](?path=/docs/components-radio--playground)).
Поэтому если установлено `isNullableValue`, свойство `value` может принимать разные значения.

- `isNullableValue` установлено, `multiple="false"` — значение `value` может быть `number` или `null`,
- `isNullableValue` установлено, `multiple="true"` — `value` может быть только `number`.

## Пример

Пример использования хука `useChoiceGroupIndexed` можно посмотреть в компоненете [CollapseGroup](https://github.com/consta-design-system/uikit/blob/master/src/components/CollapseGroup/useChoice.ts).
