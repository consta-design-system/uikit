```tsx
import { useRefs } from '@consta/uikit/useRefs';
```

Этот хук создаёт массив ссылок (`ref`).

## Свойства

| Параметры | Тип         | По умолчанию | Описание                                                         |
| --------- | ----------- | ------------ | ---------------------------------------------------------------- |
| `length`  | `number`    | -            | Длина массива — количество ссылок (`ref`), которые нужно создать |
| `deps`    | `unknown[]` | -            | Условия, при которых нужно обновить ссылки                       |

## Пример

```tsx
const refs = useRefs<HTMLDivElement>(4);

//  refs: RefObject<HTMLDivElement>[]
```
