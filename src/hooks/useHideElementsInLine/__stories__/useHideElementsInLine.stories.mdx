# useGlobalKeys

```tsx
import { useHideElementsInLine } from '@consta/uikit/useHideElementsInLine';
```

Этот хук может понадобиться в компонентах,

## Аргумены

| Аргумент   | Тип и значения | По умолчанию | Описание                                                     |
| ---------- | -------------- | ------------ | ------------------------------------------------------------ |
| `items`    | `ITEM[]`       | -            | Элементы которые буду скрываться                             |
| `busy?`    | `number`       | `0`          | Занятое просрансво в родительском элементе другими элеменами |
| `reverse?` | `number`       | -            | Скрытие элеменов с права на лево                             |

## Результат

| Свойство          | Тип и значения               | Описание                           |
| ----------------- | ---------------------------- | ---------------------------------- |
| `visibleItems`    | `ITEM[]`                     | Элемены которые оображаюся         |
| `hiddenItems`     | `ITEM[]`                     | Элемены которые нужно скрыть       |
| `itemsRefs`       | `RefObject<ELEMENT>[]`       | Ссылки на элементы                 |
| `wrapperRef`      | `RefObject<WRAPPER_ELEMENT>` | Ссылка на родительский элемент     |
| `moreRef`         | `RefObject<ELEMENT>`         | Ссылка на элемент 'еще'            |
| `itemsDimensions` | `number[]`                   | Размеры элеменов (ширина + оступы) |
| `wrapperWidth`    | `number`                     | Ширина родиеля                     |

## Пример

```css
.ExampleUseHideElementsInLine {
  &-Tag {
    margin-right: var(--space-xs);

    &_display {
      &_hidden {
        position: absolute;
        left: -300vw;

        visibility: hidden;
      }
      &_none {
        display: none;
      }
    }

    &:last-child {
      margin-right: 0;
    }
  }
}
```

```tsx
const cnExampleUseHideElementsInLine = cn('ExampleUseHideElementsInLine');

const items = [
  'первый',
  'вотрой',
  'третий',
  'чеверый',
  'пятый',
  'шестой',
  'седьмой',
  'восьмой',
  'девяый',
  'десятый',
];

export const ExampleUseHideElementsInLine = () => {
  const { itemsRefs, wrapperRef, hiddenItems, moreRef, visibleItems } = useHideElementsInLine(
    items,
  );

  return (
    <div ref={wrapperRef} className={cnExampleUseHideElementsInLine()}>
      {items.map((item, index) => {
        return (
          <Tag
            className={cnExampleUseHideElementsInLine('Tag', {
              display: !visibleItems[index] && 'hidden',
            })}
            key={index}
            label={item}
            ref={itemsRefs[index]}
            mode="info"
          />
        );
      })}
      <Tag
        className={cnExampleUseHideElementsInLine('Tag', {
          display: !hiddenItems.length && 'none',
        })}
        key="more"
        label={`+${hiddenItems.length}`}
        ref={moreRef}
        mode="info"
      />
    </div>
  );
};
```
