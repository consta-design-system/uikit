# MixSpace

С помощью миксина `MixSpace` можно добавить отступы от и внутри
любого элемента.

- [Список свойств](#список-свойств)
- [Пример использования](#пример-использования)

## Список свойств

| Свойство             | Тип или варианты значения | По умолчанию | Описание                               |
| -------------------- | ------------------------- | ------------ | -------------------------------------- |
| `padding?`           | `PropSpace`               | -            | Отступы внутри элемента                |
| `margin?`            | `PropSpace`               | -            | Отступы от элемента                    |
| `verticalPadding?`   | `PropSpace`               | -            | Вертикальные отступы внутри элемента   |
| `horizontalPadding?` | `PropSpace`               | -            | Горизонтальные отступы внутри элемента |
| `verticalMargin?`    | `PropSpace`               | -            | Вертикальные отступы от элемента       |
| `horizontalMargin?`  | `PropSpace`               | -            | Горизонтальные отступы от элемента     |
| `mTop?`              | `PropSpace`               | -            | Отступ сверху от элемента              |
| `mLeft?`             | `PropSpace`               | -            | Отступ слева от элемента               |
| `mRight?`            | `PropSpace`               | -            | Отступ справа от элемента              |
| `mBottom?`           | `PropSpace`               | -            | Отступ снизу от элемента               |
| `pTop?`              | `PropSpace`               | -            | Отступ сверху внутри элемента          |
| `pLeft?`             | `PropSpace`               | -            | Отступ слева внутри элемента           |
| `pRight?`            | `PropSpace`               | -            | Отступ справа внутри элемента          |
| `pBottom?`           | `PropSpace`               | -            | Отступ снизу внутри элемента           |

## Пример использования

```tsx
import React from 'react';

import { Text } from '@consta/uikit/Text';
import { cnMixCard } from '@consta/uikit/MixSpace';

export const MixSpaceExample = () => {
  return (
    <div
      className={cnMixSpace({
        padding: 'xl',
        horizontalPadding: '3xl',
        margin: 'm',
        mTop: '3xs',
      })}
    >
      <Text>Здесь может быть ваш текст</Text>
    </div>
  );
};
```
