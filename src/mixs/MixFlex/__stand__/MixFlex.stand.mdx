```tsx
import { cnMixFlex } from '@consta/uikit/MixFlex';
```

Миксин `MixFlex` делает из элемента, к которому он применяется - flex-контейнер.

## Список свойств

```tsx
import { Space } from '@consta/uikit/MixSpace';

export const mixFlexPropAlign = [
  'flex-start',
  'flex-end',
  'center',
  'baseline',
  'stretch',
] as const;
export type MixFlexPropAlign = typeof mixFlexPropAlign[number];

export const mixFlexPropJustify = [
  'flex-start',
  'flex-end',
  'space-between',
  'space-around',
  'space-evenly',
  'center',
] as const;
export type MixFlexPropJustify = typeof mixFlexPropJustify[number];

export const mixFlexPropFlex = ['flex', 'inline-flex'] as const;
export type MixFlexPropFlex = typeof mixFlexPropFlex[number];

export const mixFlexPropWrap = ['nowrap', 'wrap', 'wrap-reverse'] as const;
export type MixFlexPropWrap = typeof mixFlexPropWrap[number];

export const mixFlexPropDirection = [
  'row',
  'row-reverse',
  'column',
  'column-reverse',
] as const;
export type MixFlexPropDirection = typeof mixFlexPropDirection[number];
```

| Свойство     | Тип или варианты значения | По умолчанию | Описание                                                 |
| ------------ | ------------------------- | ------------ | -------------------------------------------------------- |
| `align?`     | `MixFlexPropAlign`        | -            | Определяет свойство `align-items`                        |
| `justify?`   | `MixFlexPropJustify`      | -            | Определяет свойство `justify-content`                    |
| `flex?`      | `MixFlexPropFlex`         | -            | Определяет свойство `display` в `flex` или `inline-flex` |
| `wrap?`      | `MixFlexPropWrap`         | -            | Определяет свойство `flex-wrap`                          |
| `direction?` | `MixFlexPropDirection`    | -            | Определяет свойство `flex-direction`                     |
| `gap?`       | `Space`                   | -            | Определяет свойство `gap`                                |
| `order?`     | `-1, 0, 1`                | -            | Определяет свойство `order`                              |

## Пример использования

```tsx
import React from 'react';

import { cnMixFlex } from '@consta/uikit/MixFlex';

export const MixFlexExample = () => {
  return (
    <div
      className={cnMixFlex({
        wrap: 'wrap',
        justify: 'flex-start',
        align: 'flex-end',
        gap: 'xs',
      })}
    >
      <div className={cnMixFlex({ align: 'center', justify: 'center' })}>1</div>
      <div className={cnMixFlex({ align: 'center', justify: 'center' })}>2</div>
      <div className={cnMixFlex({ align: 'center', justify: 'center' })}>3</div>
      <div
        className={cnMixFlex({ align: 'center', justify: 'center', order: -1 })}
      >
        4
      </div>
    </div>
  );
};
```
