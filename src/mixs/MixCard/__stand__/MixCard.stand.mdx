## Отступы

Отступы внутри компонента можно менять с помощью двух свойств:

- за отступы сверху и снизу отвечает `verticalSpace`,
- за отступы по бокам — `horizontalSpace`.

Варианты:
`'xs'`, `'s'`, `'m'`, `'l'`, `'xl'`, `'2xl'`, `'3xl'`, `'4xl'`, `'5xl'`.

По умолчанию — `'xs'`.

```tsx
export const MixCardExampleSpace = () => {
  return (
    <>
      <div
        className={cnMixCard({
          verticalSpace: 'xs',
          horizontalSpace: 'xs',
          form: 'round',
          shadow: true,
        })}
      >
        <Text>Отступы xs</Text>
      </div>
      <div
        className={cnMixCard({
          verticalSpace: '5xl',
          horizontalSpace: '5xl',
          form: 'round',
          shadow: true,
        })}
      >
        <Text>Отступы 5xl</Text>
      </div>
    </>
  );
};
```

## Статус

Свойство `status` окрашивает рамку компонента в цвета соответствующего статуса. Варианты:
`'alert'`, `'success'`, `'warning'`.

```tsx
export const MixCardExampleStates = () => {
  return (
    <div
      className={cnMixCard({
        verticalSpace: 'xl',
        horizontalSpace: 'xl',
        status: 'alert',
        form: 'round',
        shadow: true,
      })}
    >
      <Text>alert</Text>
    </div>
    <div
      className={cnMixCard({
        verticalSpace: 'xl',
        horizontalSpace: 'xl',
        status: 'warning',
        form: 'round',
        shadow: true,
      })}
    >
      <Text>warning</Text>
    </div>
    <div
      className={cnMixCard({
        verticalSpace: 'xl',
        horizontalSpace: 'xl',
        status: 'success',
        form: 'round',
        shadow: true,
      })}
    >
      <Text>success</Text>
    </div>
  );
};
```

## Форма

За форму компонента отвечает свойство `form`. Варианты:
`'round'`, `'square'`, по умолчанию `'round'`.

```tsx
export const MixCardExampleForm = () => {
  return (
    <div
      className={cnMixCard({
        verticalSpace: 'xl',
        horizontalSpace: 'xl',
        form: 'square',
        shadow: true,
      })}
    >
      <Text>square</Text>
    </div>
    <div
      className={cnMixCard({
        verticalSpace: 'xl',
        horizontalSpace: 'xl',
        form: 'round',
        shadow: true,
      })}
    >
      <Text>round</Text>
    </div>
  );
};
```

## Тень

По умолчанию у карточки есть тень, за тень компонента с миксином отвечает свойство `shadow`.

```tsx
export const MixCardExampleShadow = () => {
  return (
    <div
      className={cnMixCard({
        verticalSpace: 'xl',
        horizontalSpace: '3xl',
        status: 'success',
        form: 'round',
        shadow: true,
      })}
    >
      <Text>Карточка с тенью</Text>
    </div>
    <div
      className={cnMixCard({
        verticalSpace: 'xl',
        horizontalSpace: '3xl',
        status: 'success',
        form: 'round',
        shadow: false,
      })}
    >
      <Text>Карточка без тени</Text>
    </div>
  );
};

```

## Граница

По умолчанию у карточки отключены границы, за их отображение отвечает свойство `border`.

```tsx

    <>
      <div
        className={cnMixCard({

          horizontalSpace: '3xl',

## Список свойств

| Свойство                       | Тип или варианты значения                               | По умолчанию | Описание                              |
| ------------------------------ | ------------------------------------------------------- | ------------ | ------------------------------------- |
| [`verticalSpace?`](#отступы)   | `'xs', 's', 'm', 'l', 'xl', '2xl', '3xl', '4xl', '5xl'` | `'xs'`       | Вертикальные отступы                  |
| [`horizontalSpace?`](#отступы) | `'xs', 's', 'm', 'l', 'xl', '2xl', '3xl', '4xl', '5xl'` | `'xs'`       | Горизонтальные отступы                |
| [`status?`](#статус)           | `'alert', 'success', 'warning'`                         | -            | Статус, меняет цвет границ компонента |
| [`form?`](#форма)              | `'round'`, `'square'`                                   | `'round'`    | Форма                                 |
| [`shadow?`](#тень)             | `boolean`                                               | `'true'`     | Тень                                  |
| [`border?`](#границы)          | `boolean`                                               | `'false'`    | Граница                               |

        horizontalSpace: '3xl',

        form: 'round',
        shadow: true,
      })}
    >
      <Text>Здесь может быть ваш текст</Text>

  );
};
```
