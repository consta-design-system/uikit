# Файловая структура

```
builder/            # Сборщик
docs/               # Документация
src/
    components/     # Компоненты
    fileIcons/      # Иконки файлов
    hooks/          # Хуки
    icons/          # Иконки
    mixs/           # Миксины
    uiKit/          # Компоненты, которые используются для создания документации
```
