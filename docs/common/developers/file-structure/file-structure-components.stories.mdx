# Файловая структура компонентов

```
components/
    Block/                                  # Директория блока
        __tests__/                          # Директория тестов
            Block.test.tsx                  # Реализация теста блока Block
            helpers.test.tsx                # Реализация теста вспомогательных функций
        __mocks__/                          # Директория для моковых данных
            data.mock.tsx                   # Моковые данные для стори и/или тестов
        __stories__/                        # Директория для стори
            examples/                       # Директория для примеров
                BlockExampleName/           # Директория блока BlockExampleName
                    BlockExampleName.css    # Реализация блока BlockExampleName в технологии CSS
                    BlockExampleName.tsx    # Реализация блока BlockExampleName в технологии TS
            Block.mdx                       # Документация для Block
            Block.stories.css               # Cтили для стори Block
            Block.stories.tsx               # Реализация стори Block
        BlockName/                          # Директория блока, используемого в родительском блоке
            BlockName.css                   # Реализация блока BlockName в технологии CSS
            BlockName.tsx                   # Реализация блока BlockName в технологии TS
        Block.css                           # Реализация блока Block в технологии CSS
        Block.tsx                           # Реализация блока Block в технологии TS
```

### Дочерний компонент — не элемент, а блок

Все дочерние компоненты должны быть реализованы как **блоки** в БЭМ. Их название должно быть с префиксом, равным названию родительского блока, пример:

```
Button/
    ButtonIcon/
        ButtonIcon.tsx
        ButtonIcon.css
    Button.tsx
    Button.tsx
```

### Дочерний компонент не может быть родителем других компонентов

Неправильно:

```
List/
    ListItem/
        ListItemButton/
            ListItemButton.tsx
            ListItemButton.css
        ListItem.tsx
        ListItem.css
    List.tsx
    List.css
```

Правильно:

```
List/
    ListItem/
        ListItem.tsx
        ListItem.css
    ListItemButton/
        ListItemButton.tsx
        ListItemButton.css
    List.tsx
    List.css
```
