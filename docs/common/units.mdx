# Сокращения

Работе с численными данными всегда сопутствуют единицы измерения.  
Ниже приведена таблица сокращений единиц измерения, включая пунктуацию.  

Подписи единиц измерения встречаются в лейблах, контролах, в заголовках таблиц, в виджетах и пр. 

Важно сохранять консистентность подписей в элементах интерфейса во всех сервисах ГПН.

## Величины

Точку не ставят при сокращении слов, обозначающих единицы величин по ГОСТ 8.417.

| Сокращение | Значение | Комментарий |
|-|-|-|
| тыс. | тысяча | **Точка обязательна (исключение).** |
| млн | миллион | Без точки |
| млрд | миллиард | Без точки |
| трлн | триллион | Без точки |
| м | метр | Без точки. Так же метр с любой приставкой, например км. В том числе квадратные и кубические |
| кг | килограмм | Без точки. Так же грамм с любой приставкой, например мг |
| т | тонна | Без точки |
| г | год | Без точки |
| мес | месяц | Без точки |
| д | день | Без точки |
| сут | сутки | Без точки |
| час | час | Без точки |
| мин | минута | Без точки |
| с | секунда | Без точки |
| ₽ | рубль | Специальный символ для каждой валюты |

## Дни недели

В дизайн системе для дней недели используются **только двухбуквенные сокращения**

| Сокращение | Значение |
|-|-|
| Пн | понедельник |
| Вт | вторник |
| Ср | среда |
| Чт | четверг |
| Пт | пятница |
| Сб | суббота |
| Вс | воскресенье |

Хитрость:  
Часто в графиках работы пишут: 8:00—17:00 с пн по пт, 9:00—15:00 с сб по вс.  
На первый взгляд, куча нечитаемых знаков. Можно так: 8—17 в будни, 9—15 в выходные.
